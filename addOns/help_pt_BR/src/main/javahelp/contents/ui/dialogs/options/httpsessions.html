Tela de opções das sessões HTTP

This screen allows you to configure the HTTP Sessions Extension's options:

Tokens default

Estes são nomes dos cookies considerados default como tokens de sessão. Logo que são detectados como parâmetros em um Site, eles são automaticamente adicionados como tokens de sessão para aquele site, a não ser manualmente substituídos a comando do usuário.

Nos tokens de sessão maiúsculas e minúsculas são indiferentes; portanto os tokens default de sessão estarão automaticamente em minúsculas. Além disso, quaisquer modificações dos tokens de sessão padrão (default) são visíveis somente após a nova visita às páginas do Site.

Habilitar só para o Proxy

Se esta opção estiver habilitada, a Extensão só irá processar as mensagens enviadas e recebidas através do Proxy (por exemplo, de/para o navegador). As mensagens de outras Extensões são processadas somente se esta opção estiver desabilitada.
