Diálogo Acrescentar Alerta

This dialog allows you to manually add or change an Alert associated with a specific request.

Campos

A caixa de diálogo possui os seguintes campos:

Tipo

O tipo do alerta é um campo 'pull down' que permite a você selecionar um conjunto de tipos de problemas já definidos.
Você também pode inserir seu próprio texto ou alterar o texto de um dos itens selecionados.
Se você selecionar um dos tipos existentes, então os campos Descrição, Solução e Outras Informações serão preenchidos com texto associado ao item que você escolheu.

Risco

Um campo pull down que permite a você especificar o grau de risco:
    Informativo
    Baixo
    Médio
    Alto

Confiança

Um campo pull down que permite a você especificar seu grau de confiança sobre a validade das conclusões:
    Falso Positivopara problemas em potencial que mais tarde você não considerará exploráveis
    Baixopara problemas não confirmados
    Médiopara problemas com os quais você está um tanto confiante
    Altopara descobertas nas quais você tem quase certeza
    Confirmadoproblemas confirmados

Parâmetro

Um campo pull down que permite a você especificar a que parâmetro o problema está associado.
O campo é previamente preenchido com quaisquer parâmetros encontrados de URL ou formulário, mas você também pode inserir seu próprio nome de parâmetro.
A matriz de parâmetros (no componente de query da URL x-www-form-urlencoded no corpo da requisição) é identificada com o seu índice. Por exemplo, para uma requisição contendo escolhas[]=EscolhaA&escolhas[]=EscolhaB o primeiro parâmetro será identificado como escolhas[0] e o segundo como escolhas[1].

Descrição

Uma descrição geral do tipo de problema encontrado.
Isto é preenchido quando você seleciona um dos tipos predefinidos, mas você também pode alterá-lo conforme necessário.
Observe que as alterações feitas serão perdidas se você selecionar outro tipo.

Outras informações

Informações específicas para o problema que você encontrou.
Isto não é previamente preenchido.

Solução

Recomendações sobre como corrigir o problema.
Isto é preenchido quando você seleciona um dos tipos predefinidos, mas você também pode alterá-lo conforme necessário.
Observe que as alterações feitas serão perdidas se você selecionar outro tipo.

Referência

Uma ou mais URLs apontando para mais informações na internet sobre o tipo de alerta selecionado.
Isto é preenchido quando você seleciona um dos tipos predefinidos, mas você também pode alterá-lo conforme necessário.
Observe que as alterações feitas serão perdidas se você selecionar outro tipo.
