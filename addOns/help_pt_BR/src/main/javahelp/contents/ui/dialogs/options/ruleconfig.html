Tela de opções de configuração de regra

Esta tela permite que você configure o comportamento de regras específicas de varredura ativa e passiva.

Selecione qualquer uma das linhas para ver detalhes de configuração específica e alterar o valor associado. Podem ser redefinidas tanto regras individuais quanto todas as regras .

As regras incluídas incluem:

NomePadrãoDescrição
rules.common.sleep15O comprimento de tempo em segundos, usado para ataques de sincronismo.
rules.cookie.ignorelistUma lista separada por vírgulas de nomes de cookies. Os cookies incluídos nesta lista serão ignorados em varreduras relacionadas a problemas com cookies.
rules.csrf.ignorelistUma lista de identificadores separados por vírgulas. Todos os formulários com um nome ou ID que correspondam a um desses identificadores serão ignorados em varreduras de tokens de Anti-CSRF ausentes. Só use esse recurso para ignorar os formulários que você sabe que são seguros, por exemplo, os de busca.
rules.csrf.ignore.attnameThe name of an HTML attribute that can be used to indicate that a form does not need an anti-CSRF Token. If rules.csrf.ignore.attvalue is specified then this must also match the attribute's value. If found any related alerts will be raised at INFO level.
rules.csrf.ignore.attvalueThe value of an HTML attribute named by rules.csrf.ignore.attname that can be used to indicate that a form does not need an anti-CSRF Token. If found any related alerts will be raised at INFO level.
rules.domains.trustedA comma separated list of URL regex patterns. Any URLs that match the patterns will be considered trusted domains and the issues ignored.
