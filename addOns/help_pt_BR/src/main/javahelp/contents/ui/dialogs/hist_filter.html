Filtro do histórico

Esta caixa de diálogo permite restringir quais solicitações são exibidas na Aba de Histórico.

Campos

A caixa de diálogo possui os seguintes campos:

Métodos

Os métodos HTTP a serem filtrados.

Códigos

Os códigos de resposta HTTP a serem filtrados.

Tags

As Tags a serem filtradas.

Alertas

Os tipos de Alertas a serem filtrados.

Notas

Se a presença ou ausência de Notas deve ser filtrada.

Regex de URL inc

Uma lista de padrões regex 'incluir' (um por linha) - se especificada, apenas as URLs que correspondem a esses padrões serão incluídas.

Regex de URL exc

Uma lista de padrões regex 'excluir' (um por linha) - se especificada, quaisquer URLs que correspondem a esses padrões serão excluídas/ocultas.
