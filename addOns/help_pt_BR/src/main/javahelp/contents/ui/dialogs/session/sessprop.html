Regex de URL

Nos diálogos Excluir de * você pode digitar expressões regulares para definir URLs excluídas. Você pode escapar um único metacaractere com uma barra invertida, e também pode usar a sequência de escape \Q...\E. Todos os caracteres entre o \Q e o \E são interpretados como caracteres literais. Por exemplo, o \Q*\d+*\E coincide com o texto literal * \d+*. Esta sequência de escape é usada no ZAP quando você exclui URLs através de alguns menus de contexto.
Nota: Se sua URL contém um "\E", então você tem que dar os seguintes passos ao usar a sequência \Q...\E para escape:
  • Abra a seqüência de escape
  • Feche a seqüência de escape antes do "caractere" \E
  • Escape a barra invertida
  • Abra depois do "\E" outra seqüência de escape;
  • Feche a seqüência de escape como normalmente faria.

Exemplo: subdomain.example.com/path?a=\E&moredata=2 deve aparecer como \Qsubdomain.example.com/path?a=\E\\E\Q&moredata=2\E
