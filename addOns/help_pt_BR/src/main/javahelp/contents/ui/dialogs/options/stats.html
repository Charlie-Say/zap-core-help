Tela de opções de estatísticas

This screen allows you to configure the statistics that ZAP maintains.

Estatísticas em memória habilitadas

Por padrão o ZAP mantém as estatísticas na memória. Para desabilitar as estatísticas em memória desmarque esta caixa.

Servidor de Statsd habilitado

Ligue esta caixa para configurar o endereço de um servidor de Statsd para onde o ZAP enviará suas estatísticas.

Hostname do Statsd

O host onde o servidor de Statsd está rodando.

Porta de Statsd

A porta que o servidor de Statsd está escutando. Porta padrão: 8125.

Prefixo Statsd

O prefixo que o ZAP vai inserir nas estatísticas que envia para o servidor de Statsd.
