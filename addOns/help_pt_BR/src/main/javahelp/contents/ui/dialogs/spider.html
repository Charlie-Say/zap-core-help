Diálogo do spider

Esta caixa de diálogo lança o spider.

Alcance

A primeira aba permite a você selecionar ou alterar o ponto de partida.
Se o ponto de partida for em um ou mais Contextos então você conseguirá escolher um deles.
Se esse contexto tiver algum Usuário definido, então você conseguirá escolher um deles.
Se você selecionar um dos usuários então a varredura ativa será executada com aquele usuário, e o ZAP o (re) autenticará como o usuário sempre que necessário.

If you select 'Recurse' then all of the nodes underneath the one selected will also be used to seed the spider.

Se você selecionar apenas 'Subárvore Spider' o spider vai acessar somente os recursos que estão naquele ponto de partida (URI). Ao avaliar se um recurso é encontrado dentro da subárvore especificada, o spider considera apenas o esquema, host, porta e componentes de caminho da URI.

If you select 'Show Advanced Options' then the following tab will be shown which provide fine grain control over the spider process.

Clicar no botão 'Reset' retornará todas as opções a seus valores padrão.

Avançado

Os parâmetros nesta guia correspondem aos mesmos parâmetros da Tela de opções do spider.

    Aba de SitesMenu de clique direito 'Ataque / Spider... '
