Tela de Opções do Banco de Dados

Esta tela permite a você configurar as opções de banco de dados:

Compactar (na saída)

Permite a compactação do database quando o ZAP é encerrado; compactar o database garante uso mínimo de espaço em disco mas também vai fazer com que o encerramento do ZAP demore mais.

Log de recuperação

Controla se o log de recuperação do banco de dados está habilitado ou não.
Melhora o desempenho do banco de dados quando desativado, mas pode levar à perda de dados se o ZAP é encerrado bruscamente.
Nota: sessões em andamento não serão afetada, as alterações terão efeito em sessões abertas e novas.

Tamanho máximo do corpo da requisição

É o maior tamanho do corpo em bytes que o ZAP irá permitir.

Maior tamanho do corpo da Resposta

É o maior tamanho do corpo em bytes que o ZAP irá permitir.

Entrada de opções de persistência em nova sessão

Se estiver ligado o diálogo Persistir na Sessão vai aparecer quando uma nova sessão for criada.

Opção padrão

Opção padrão para novas sessões. Se a entrada definida acima estiver habilitada, então a opção default será selecionada.
Se o prompt não estiver habilitado, então este é a ação que será tomada (a opção padrão será usada).
