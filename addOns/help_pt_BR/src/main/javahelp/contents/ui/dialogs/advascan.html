Varredura ativa

Esta caixa de diálogo lança a varredura ativa.

Alcance

A primeira aba permite a você selecionar ou alterar o ponto de partida.
Se você tiver mais que uma política de varredura então você conseguirá selecionar uma para usar.
Se o ponto de partida for em um ou mais Contextos então você conseguirá escolher um deles.
Se esse contexto tiver algum Usuário definido, então você conseguirá escolher um deles.
Se você selecionar um dos usuários então a varredura ativa será executada com aquele usuário, e o ZAP o (re) autenticará como o usuário sempre que necessário.

If you select 'Recurse' then all of the nodes underneath the one selected will also be scanned.
Vetores de entrada personalizados são suportados apenas se esta opção não estiver selecionada.

If you select 'Show Advanced Options' then the following tabs will be shown which provide fine grain control over the active scanning process.

Clicar no botão 'Reset' retornará todas as opções a seus valores padrão.

Vetores de entrada

A aba de vetores de entrada permite que você substitua os vetores de entrada padrão que estão definidos nas Tela de Entrada de opções para Vetores de Varredura Ativa.
Clicar no botão 'Reset' retornará todas as opções a seus valores padrão.

Vetores personalizados

A aba Vetores Personalizados permite a você indicar locais específicos na solicitação para atacar.
Custom Vectors are only available if the 'Recurse' option on the first tab is not selected.
Para adicionar vetores de entrada personalizados destaque (highlight/ilumine) os caracteres que você deseja atacar na request e clique no botão 'Adicionar'.
Você pode adicionar quantos vetores de entrada personalizados quiser.
Para remover vetores de entrada personalizados realce (highlight/iluminar) os caracteres selecionados e clique no botão 'Remover'.
Marcando a caixa 'Desabilitar vetores de entrada não personalizados' desabilita todos os vetores de entrada, exceto aqueles que você definir manualmente nesta guia.

Tecnologia

A aba de Tecnologia permite que você especifique quais tipos de tecnologias irá procurar no alvo.
Desmarcando tecnologias que com certeza não estão presentes na aplicação-alvo pode acelerar a varredura, já que as regras destinadas àquela tecnologia podem ser ignoradas.

Política

A aba Política permite substituir qualquer das configurações especificadas na política de varredura selecionada.

Filter

The Filter tab allows you to specify filter criteria and active scan will filter requests based on these criteria's.
There are four filter criteria's available in the filter tab
  • HTTP method
  • Status code
  • Tags
  • URL pattern
HTTP method and Status code filter criteria are only applied to check if a request can be included in active scan or not,
where as Tags and URL pattern filter criteria can be applied to include or exclude a request.
