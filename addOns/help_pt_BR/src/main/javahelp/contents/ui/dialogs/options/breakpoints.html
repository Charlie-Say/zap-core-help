Tela de opções dos pontos de interrupção

Esta tela permite que você configure opções de breakpoint :

Confirmar a mensagem de soltar preso

Pede confirmação quando um request ou resposta são derrubados.

Só parar em mensagens do escopo

The 'global' breakpoint will only be triggered by requests that are in escopo.
Custom breakpoints will not be affected.

ZAP sempre no topo quando chega a um ponto de interrupção

Se estiver selecionado, significa que o ZAP fica no topo de todas as janelas sempre que se chega a um ponto de interrupção.

Show buttons to select the requests you don't want ZAP to break on

This controls whether the buttons to select the file types you don't want ZAP to break on and the scope filter appear in the break panel.

Javascript URL regex

This sets the regex used to predict if the content in the response will be a Javascript file.

CSS and Fonts URL regex

This sets the regex used to predict if the content in the response will be a CSS or a Font file.

Multimedia URL regex

This sets the regex used to predict if the content in the response will be a multimedia file (images, videos or sounds).

Break buttons mode

This controls whether there is a single combined break button on the Barra de ferramentas superior or separate ones for Requests and Responses.
