Tela de opções de idioma/linguagem

A tela de idioma permite que você configure:

Seleção de linguagem

A linguagem que o ZAP usa para diálogos.
Um conjunto de idiomas vem embutido no ZAP, porém mais idiomas podem estar disponíveis após o último release.
Verifique a página de download do ZAP para o mais recente pacote de linguagem disponível.

Utiliza o local de sistema para formatar.

Caso a formatação de números, datas e horários seja feita de acordo com o local do sistema, ao invés do local da linguagem selecionada.

Importador de Pacote de Linguagem

Além da seleção da língua que é usada para exibição pelo ZAP, a ferramenta de importação permite importar novos pacotes de idiomas ou atualizações de arquivo de linguagem. Um pacote de arquivo de linguagem sempre terá a extensão zaplang. (Por exemplo, language-pack-en_GB.zaplang)
Após a importação, você imediatamente será capaz selecionar a linguagem recém-importada do menu de seleção suspenso (drop down) na parte superior da tela.
Note que você precisará reiniciar o ZAP para finalmente aplicar o novo idioma.
