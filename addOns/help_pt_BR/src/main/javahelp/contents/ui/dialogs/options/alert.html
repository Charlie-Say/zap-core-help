Tela de opções de alertas

Esta tela permite que você configure as opções de alertas:

Mesclar alertas relacionados no relatório

Se selecionado então questões relacionadas serão mescladas em todos os relatórios gerados.
Isto irá reduzir significativamente o tamanho do relatório já que informações duplicadas serão removidas.

Instâncias de alerta máximo no relatório

O número máximo de instâncias alertas para incluir em um relatório.
Um valor de zero é tratado como ilimitado.

Arquivo de substituição de alertas.

O nome completo de um arquivo de propriedades que especifica qualquer Substituições de alerta que você deseja aplicar.
