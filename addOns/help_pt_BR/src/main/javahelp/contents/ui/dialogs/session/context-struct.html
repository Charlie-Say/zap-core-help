Tela de estrutura do contexto de sessão

Esta é uma das Telas de Contexto de Sessão que permite que você gerencie o entendimento do ZAP da estrutura do aplicativo.

Separadores de par de valor de chave de URL

Os caracteres que separam pares de chaves e valores em URLs, por padrão '&'.

Separadores de valor de chave de URL

Os caracteres que separam as chaves e valores em URLs, por padrão '='.

Separadores de pares de valores chaves em POST

Os caracteres que separam pares de chaves e valores nos dados do POST, por padrão '&'.

Separadores de valor de chave de POST

Os caracteres que separam as chaves e valores em dados de POST, por padrão '='.

Modificadores estruturais

Os Modificadores estruturais que se aplicam a este contexto, que podem ser:
  • Data Driven Content which identify URL paths that represent data
  • Structural Parameters which identify parameters that represent application structure instead of user data
