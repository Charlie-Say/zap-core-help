Menu Visualização

Este menu manipula as opções de exibição.

Habilitar Imagens no Histórico

Se selecionado, as imagens serão exibidas na Aba de histórico.

Mostrar Aba

Isso permite a você mostrar qualquer uma das abas disponíveis, incluindo aquelas que você ocultou.

Mostrar Todas as Abas

Isso permite a você mostrar todas as abas.

Esconder Abas Não Fixadas

Isso permite a você esconder todas as abas não fixadas.

Fixar Todas Abas Visíveis

Isso permite a você fixar todas as abas visíveis.

Destravar Todas as Abas

Isso permite a você destravar todas as abas,

Note that add-ons can add additional menu items.
