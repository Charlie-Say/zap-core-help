Menu de nível superior

Por padrão, os seguintes itens de menu de nível superior estarão presentes:
     Arquivo Trata da sessão atual
     Editar This handles finding strings in specific tabs, searching for strings across all requests and responses, changing ZAP's mode, and managing forced user mode
     Exibir Isto lida com as opções de exibição
     Analisar Configura as políticas de varredura
     Relatórios Este manipula os relatórios
     Ferramentas Isto lida com as ferramentas e opções gerais
     Import menu This gives access to menus that allow to import data into ZAP.
     Online menu This gives access to a set of online resources
     Help menu This gives access to the 'about' dialog and this help file

Note que complementos pode criar itens de menu adicionais.
