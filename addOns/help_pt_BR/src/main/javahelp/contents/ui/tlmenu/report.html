Menu de relatórios

Este menu gerencia os relatórios.

Gerar relatório em HTML...

Isso gera um novo relatório HTML contendo os alertas gerados.

Generate relatório em XML...

Isso gera um novo relatório XML contendo os alertas gerados.

Gerar Relatório Markdown...

Isso gera um novo relatório Markdown contendo os alertas gerados.

Exportar Mensagens para o Arquivo...

Isso permite que você salve as solicitações e respostas para um arquivo de texto.
Selecione as mensagens a serem salvas na aba histórico - use a tecla shift para selecionar várias mensagens.

Exportar Respostas para o Arquivo...

Isso permite que você salve uma resposta específica em um arquivo.
Selecione a mensagem relevante na guia histórico - note que respostas binárias (como imagens) podem ser salvos, bem como respostas de testes.

Exportar todas as URLs para o Arquivo...

Isso permite que você salve todas as URLs acessadas em arquivo texto ou HTML.
Isto pode ser usado, entre outras coisas, para comparar as URLs disponíveis para usuários com diferentes funções ou permissões no mesmo sistema.

Exportar as URLs Selecionadas para o Arquivo...

Baseando-se nas URL selecionadas (incluindo seleções múltiplas) na árvore de sites, todas as URL e URL filhas das selecionadas.

Exportar URLs para Contexto

Todas as URL na árvore de sites que estiverem no contexto selecionado, serão exportadas. Essa funcionalidade também está disponível no menu (botão direito), quando usado em um nó de contexto dentro do painel árvore de sites.

Comparar com outra Sessão...

Isso solicita uma sessão ZAP que você salvou anteriormente.
Ele apresenta a você um arquivo de saída no qual estão gravadas todas as URLs acessadas pela sessão atual e pela sessão que você escolheu para comparar.
O arquivo conterá uma tabela listando as URLs e as respostas HTTP para as URLs nas 2 sessões.
Botões JavaScript permitem que você exiba todas as URLs, ou apenas aquelas acessados na primeira sessão, ou então na segunda sessão e também aquelas acessados pelas duas sessões.
Isto é particularmente útil para comparar 2 sessões que acessam o mesmo aplicativo com usuários diferentes. Você conseguirá ver quais URLs são visíveis para os usuários e poderá tentar acessar todas as URLs quando estiver logado como cada um dos usuários.

Note that add-ons can add additional menu items.
