Desktop UI Overview

The Desktop UI is made up of:
    Menu superior
    Barra de Ferramentas Superior
    Três janelas que contêm várias abas, detalhadas abaixo
    Rodapé
Cada uma das três janelas tem um conjunto de uma ou mais abas.
Por padrão, somente as guias essenciais são mostradas quando ZAP é iniciado.
As abas restantes aparecem quando são usadas (por exemplo, para Spider e Varredura Ativa) ou quando você decide exibí-las usando a guia especial, na extrema direita de cada janela com o ícone verde '+'. Esta aba especial desaparece quando não existem mais abas escondidas.
As abas podem ser fechadas clicando-se no ícone 'x' que é mostrado quando a aba é selecionada.
As abas podem também ser 'fixadas', usando o ícone 'alfinete', que também aparece quando a aba é selecionada - abas fixadas aparecerão também quando o ZAP for reiniciado.

A janela da 'árvore'

Isto é exibido no lado esquerdo da tela e permite que você navegue por todas as URLs visitadas.
     Aba de SitesIsso mostra todas as URLs visitadas em uma estrutura de árvore

A janela de trabalho

Isto é exibido no lado superior direito e é onde as solicitações e respostas são exibidas. Você também pode fazer alterações aqui quando um breakpoint é encontrado.
     Aba de RequestIsto mostra os dados do que seu navegador envia para o aplicativo
     Aba de RespostasIsto mostra os dados que do aplicativo envia de volta ao seu navegador
     Aba de BreakIsso permite que você manipule os dados

A janela de 'informações'

Isso é exibido na parte inferior e é onde a informação útil é exibida.
     Aba de históricoIsto mostra as solicitações na ordem que elas foram feitas
     Aba de BuscaIsso permite que você pesquise todos os requests e respostas
     Breakpoints tabThis shows the Breakpoints set
     Aba de alertasIsto mostra os alertas levantados sobre a aplicação
     Aba de varredura ativaIsto mostra as varreduras ativas
     Spider tabIsto mostra as URLs ainda não visitadas
     Aba de parâmetrosIsto mostra um resumo dos parâmetros que um site usa
     Aba da saídaIsto mostra várias mensagens informativas

Todas as abas, menos as essenciais, podem ser escondidas clicando sobre 'x' no título da aba.
Você pode reexibir as abas que ocultou através do Menu exibir.
Abas ocultas também podem aparecer quando 'ativadas' por outra ação.
