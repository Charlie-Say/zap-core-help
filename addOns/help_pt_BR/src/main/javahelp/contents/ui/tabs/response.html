Aba de Resposta

A aba de Resposta mostra-lhe os dados enviados ao seu navegador para a solicitação que esteja destacada na Aba de Sites ou na Aba de histórico .

Menus pull-down permitem que você selecione diferentes Visualizações para o cabeçalho de resposta e o corpo.

Note that images are only shown in the Aba de histórico Se a flag 'Ativar imagem no histórico ' estiver selecionada no Menu Exibir

Menu do botão direito

Clicar com o botão direito em um nó faz aparecer um menu que lhe permitirá:

Procurar...

Isso irá abrir o diálogo de Localizar.

Copy

This will copy the selected string to the clipboard.
