Aba de Request

A aba Requisição mostra-lhe os dados enviados pelo seu navegador para a solicitação que esteja destacada na Aba de Sites ou na Aba de histórico .
O painel superior mostra o cabeçalho de solicitação e o painel inferior mostra todos os dados enviados também, por exemplo na sequência de um pedido POST.

Menus pull-down permitem que você selecione diferentes Visualizações do cabeçalho de request e do corpo.

Menu do botão direito

Clicar com o botão direito em um nó faz aparecer um menu que lhe permitirá:

Procurar...

Isso irá abrir o diálogo Localizar.

Copy

This will copy the selected string to the clipboard.
