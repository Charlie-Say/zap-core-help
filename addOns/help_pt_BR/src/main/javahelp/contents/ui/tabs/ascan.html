Aba de Varredura Ativa

A aba de Varredura Ativa permite a você executar uma Varredura Ativa (Active Scan) .

Botão 'Gerenciador da Política de Varredura...' abre o Gerenciador de políticas de varredura que permite configurar as políticas de varredura.

O botão 'Nova Varredura' abre o diálogo de Varredura Ativa, que permite a você especificar exatamente o que deve ser escaneado.

A barra de ferramentas contém um conjunto de botões que permitem iniciar, parar, pausar e continuar a varredura selecionada.
Uma barra de progresso mostra o quanto progrediu a varredura do site selecionado.
O valor 'Varreduras em andamento' mostra quantas varreduras estão ativas - passar o cursor sobre esse valor irá mostrar num pop-uo uma lista dos sites que estão sendo examinados.
O botão 'Mostrar Detalhes do progresso da Varredura' abre o Diálogo de progresso da Varredura que lhe permite ver detalhes sobre quais regras estão rodando, ignorar regras individuais e ver um gráfico das respostas.

Menu do botão direito

Clicar com o botão direito em um nó faz aparecer um menu que lhe permitirá:

Excluir de

Este menu possui os seguintes submenus:

Proxy

Isto irá excluir do proxy os nós selecionados. Eles ainda passarão pelo ZAP, mas não serão exibidos em qualquer uma das guias.
Isto pode ser usado para ignorar URLs que você sabe que não são relevantes para o sistema que você está testando.
Os nós podem ser incluídos novamente através das Propriedades da Sessão .

Scanner

Isto irá impedir que os nós selecionados entrem na varredura ativa.
Os nós podem ser incluídos novamente através das Propriedades da Sessão .

Spider

Isto irá impedir que os nós selecionados sejam percorridos pelo spider.
Os nós podem ser incluídos novamente através das Propriedades da Sessão .

Open/Resend with Request Editor...

Isso irá abrir o Manual Request Editor dialog que lhe permite reenviar o request depois de fazer quaisquer alterações que você deseje.

Novo Alerta...

Isso irá abrir o Diálogo Acrescentar Alerta que permite a você registrar manualmente um novo alerta para um request.

Mostrar na Aba de Sites

Isto irá mostrar a mensagem selecionada na aba de Sites.

Ver no Navegador

Isto irá abrir a URL do nó selecionado em seu navegador padrão.
