Release 1.3.1

Este release trouxe as seguintes mudanças:

Consertos de bugs:

Problema 97: Diretório de trabalho para aplicações

Problema 98: Não é possível cancelar a verificação de atualizações

Problema 100: Aba de Alertas tem 2 pop ups de botão direito para 'Reenviar...'

Problema 104: Está exibindo aviso quando não existe nenhum certificado de CA raiz

Problema 110: Está escolhendo o idioma do usuário por default, ao rodar pela primeira vez

Problema 112: Conseguir um modo de aplicar os pacotes de idioma no Mac OS

Problema 121: Javascript do relatório de comparação de sessão está quebrado no Firefox 3

Problema 123: Não se consegue selecionar site https na janela de varredura ativa se houver um http equivalente

Problema 127: o recurso de CSRF token não funciona no fuzz/difusor - acrescentada explicação de ajuda adicional como correção parcial

Problema 140: as opções de visualização não deviam incluir as opções avançadas nem as de gerenciamento de janelas

Bug de SSL impedia as conexões SSL de serem estabelecidas em qualquer navegador

Problemas conhecidos:

Mac OS X: SSL dinâmico e Google Chrome

O SSL dinâmico não está funcionando quando se usa o Google Chrome. Isto é devido a um problema conhecido e não resolvido com o Google Chrome e o chaveiro X do Mac OS. Quando importa o certificado raiz da autoridade OWASP ZAP e solicita um site SSL, aparece uma mensagem de erro indicando "Certificado inválido".
