Release 1.3.0

Este release trouxe as seguintes mudanças:

Alterações significativas:

Fuzzing/Difusão

Strings em uma resposta agora podem ser difusas para tentar encontrar vulnerabilidades.
Anti CRSF tokens podem ser detectados e regenerados automaticamente durante a difusão.
Esta funcionalidade é baseada no código do projeto OWASP JBroFuzz.

Certificados SSL Dinâmicos

O suporte para conexões SSL foi melhorado e simplificado.
Usuários agora podem criar seu próprio certificado raiz e distribuí-los em seus clientes HTTP.
Leia Opção: Certificados SSL Dinâmicos para obter mais detalhes.

Modos daemon e API

Iniciando o ZAP com a opção de linha de comando "-daemon" fará com que ele seja executado em segundo plano no modo 'sem cabeça', significando que nenhuma interface de usuário é exibida.
Uma API inicial foi implementada em XML, JSON e HTML.
Se o ZAP estiver sendo executado como um daemon, então a API é ativada automaticamente; caso contrário, a API deve ser habilitada através da tela Opções de API.
A API pode ser navegada abrindo-se http://zap/ em seu navegador quando ele estiver usando o ZAP como proxy.

Integração Beanshell

O BeanShell é uma shell de Java que pode ser usada para executar scripts BeanShell.
Esses scripts são uma forma simplificada de Java que usam muitos elementos de sintaxe Java, mas em um formato mais simples de script.
Todo o código Java é também válido como código BeanShell.
A integração do BeanShell no OWASP ZAP permite que você escreva scripts usando as funções ZAP e data sets.
Isto pode ser um recurso muito poderoso para a análise de aplicações web.

Internacionalização completa

Todas as strings de exibição (display) agora são totalmente internacionalizadas.

Localização

Suporte disponível para os seguintes idiomas:
    Inglês
    Português do Brasil
    Chinês
    Francês
    Alemão
    Grego
    Indonésio
    Japonês
    Polonês
    Espanhol

Pequenas mudanças:

Visualização hexa

As abas Requisição e, Resposta agora oferecem uma opção de 'Hex View' que irá exibir os corpos de solicitação e resposta em formato hexadecimal.

Resultados de Busca

A Aba de Busca agora exibe todas as ocorrências de uma string localizada em vez de apenas a primeira instância em cada solicitação ou resposta.

Excluir URLs

URLs podem ser explicitamente excluídas da varredura ativa, spider e do proxy.

Suporte de cópia

A maioria dos painéis agora proporciona uma opção de 'Copiar' no 'botão direito' , incluindo os painéis de Port Scan e Brute Force.

Suporte de desfazer/refazer (undo/redo)

Todos os campos de entrada agora suportam ações de desfazer/refazer usando os aceleradores padrão de Undo/Redo dos sistemas operacionais:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Suporte de proxy para varredura de porta e opção de timeout

A varredura de Porta (o port scanner) agora pode usar o proxy de saída (se configurado) e o tempo limite/timeout em milissegundos também pode ser definido.

Botões de pausa de solicitação e resposta

Agora existem dois botões 'Criar Initerrupção' para permitir a configuração de interrupções em todas as solicitações e respostas independentemente.

Botões para expansão das abas de Sites e Informação

Agora existem dois botões que permitem a você alternar entre ter as abas de Sites e de Informações expandidas.

Break tab icon changes colour when breakpoint hit

Enquanto a aba de interrupção não está em uso seu ícone é uma cruz cinza:  .
Quando se chega a um breakpoint o ícone da aba é alterado para uma cruz vermelha:  .

Timeout ajustável

A tela de Opções: conexão permite que você defina o tempo limite em segundos para tornar mais fácil o teste de aplicativos lentos.

Atualizações de biblioteca

A maioria das bibliotecas usadas pelo ZAP foram atualizadas para as versões mais recentes.

Um novo ícone :)

Agradecimentos ao Simon Egli e a todos os outros por apresentarem sugestões legais de ícones.

Problemas conhecidos:

Mac OS X: SSL dinâmico e Google Chrome

O SSL dinâmico atual não está funcionando quando se usa o Google Chrome. Isto é devido a um problema conhecido e não resolvido com o Google Chrome e o Mac OS X Keychain. Quando importa o certificado raiz da autoridade OWASP ZAP e solicita um site SSL, aparece uma mensagem de erro indicando "Certificado inválido".
