Release 1.3.4

Este release trouxe as seguintes mudanças:

Pequenas mudanças:

Problema 146: Regex inverso na busca e realce em fuzz quando coincidindo

Problema 202: Opção para desativar a recursividade do brute force

Problema 215: Permitir que arquivos personalizados de brute force sejam adicionados facilmente

Acrescentadao também recurso de definir o arquivo padrão de brute force.

Problema 217: Invocar apps - Adicionado suporte para cookies e postagem de parâmetros de dados

Problema 218: Permitir que usuários adicionem facilmente seus próprios arquivos de difusor

Acrescentada também a opção de anexar a saída a uma nota relacionada com entrada relevante.

Consertos de bugs:

Problema 56: Desativar POST reqs no Spider

Problema 186: Opções de conexão - Prompt para credenciais de proxy ao iniciar / validação de endereço preenchida

Problema 188: falha na atualização do ZAP em Linux e Windows

Problema 191: Exceção quando a URL contém caracteres de escape

Problema 196: Mau funcionamento quando várias caixas de diálogo da mesma opção ficam abertas simultaneamente.

Problema 199: Vulnerabilidades com textos truncados

Problema 204: Pesquisa nos headers só localiza regex em requests

Problema 206: Exceção na aba "Alertas" ao escolher uma opção de pop-up

Problema 214: Nenhuma mensagem de alerta ao salvar o relatório em um local somente de leitura

Problema 216: Exceção quando uma URI não tem o componente de caminho

Problema 219: Quebrar e ignorar urls por default incluem GET/POST

Problema 220: Mensagem incorreta: senha (armazenada em texto não criptografado)
