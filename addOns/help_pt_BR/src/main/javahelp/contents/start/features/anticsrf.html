Tokens anti CSRF

Tokens Anti CSRF são parâmetros (pseudo) aleatórios usados para proteção contra ataques de Cross Site Request Forgery (CSRF).
No entanto eles também tornam mais difícil o trabalho em testes de penetração, especialmente se os tokens são regenerados sempre que um formulário é solicitado.

O ZAP detecta tokens anti CSRF pelos nomes de atributo - a lista de nomes de atributos considerados tokens anti CSRF é configurada usando-se a Tela de opções Anti CSRF.
Quando o ZAP detecta esses tokens, grava o valor deles e qual URL gerou o token.
Recursos como varredura ativa têm opções que fazem o ZAP automaticamente regenerar os tokens se necessário.
