Modificadores estruturais

Modificadores estruturais são controles que mudam o modo pelo qual o ZAP representa a estrutura do aplicativo.
A Sites Tree is ZAP's representation of the application.
Se não for uma boa representação da estrutura, o ZAP não será capaz de atacar o aplicativo eficazmente.
Atualmente existem 2 tipos de modificadores estruturais:

  • Conteúdo orientado a Dados, que identifica caminhos de URL representando dados
  • Parâmetros estruturais, que identificam parâmetros representando a estrutura do aplicativo em vez de dados do usuário
Modificadores estruturais só podem ser aplicados a Contextos.
