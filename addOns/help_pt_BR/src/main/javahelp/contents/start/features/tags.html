Tags

Uma tag é curto pedaço de texto que você deseja associar a um pedido/request.
Um pedido pode ter várias tags associadas
Tags podem ser adicionadas e alteradas usando-se a caixa de diálogo Gerenciar Tags.
Elas também podem ser automaticamente adicionadas através da varredura passiva - isto permite que você automaticamente marque páginas que correspondem uma expressão regular particular.

Tags são mostradas na aba de histórico e podem ser filtradas no Filtro de histórico.
