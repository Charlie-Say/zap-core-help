Manipulator-in-the-middle Proxy

ZAP is a Manipulator-in-the-middle Proxy. Isso permite que você veja todas as solicitações feitas em um aplicativo da web e todas as respostas que você receber dele.
Entre outras coisas, isso permite que você veja chamadas AJAX que podem senão ser vistas de outra maneira.

You can also set breakpoints which allow you to change the requests and responses on the fly.
