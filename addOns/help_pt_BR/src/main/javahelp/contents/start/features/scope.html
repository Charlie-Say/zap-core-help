Escopo

O escopo é o conjunto de URLs que você está testando e é definido pelos contextos especificados.

Por padrão, nada está no escopo.

O escopo potencialmente altera:

  • O que pode fazer, quando você está no modo protegido
  • O que é mostrado na aba histórico
  • Protegido - usuário só pode realizar ações perigosas (potencialmente) em URLs no escopo
  • Padrão - como nas versões anteriores, você pode fazer qualquer coisa
  • ATAQUE - novos nós que estão no escopo são ativamente escaneados tão logo eles são descobertos
É recomendável que você defina um novo Contexto para cada aplicativo web que compõe o sistema que você está testando e configurá-los no escopo ao testar cada um.
