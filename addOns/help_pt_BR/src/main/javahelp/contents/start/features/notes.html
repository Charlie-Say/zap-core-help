Notas

Uma nota é qualquer texto que você deseja associar uma solicitação/request.
Por exemplo, você pode usar notas para gravar detalhes de testes extras que você precisa para realizar um request.
Notas são adicionadas e alteradas usando-se a caixa de diálogo Adicionar nota.

Notas são marcadas na guia histórico com este ícone.
