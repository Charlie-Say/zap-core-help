Estatísticas

ZAP mantém estatísticas que podem ajudá-lo a entender o que realmente está acontecendo, ao interagir com aplicativos grandes.

As estatísticas estão disponíveis através da API e também podem ser enviadas para um servidor de Statsd quando configurado através da tela de opções de estatísticas.

The full set of statistics maintained by the ZAP core and add-ons are maintained on the ZAP website: Internal Statistics.
