Tag

Tag adalah potongan pendek teks yang anda ingin mengasosiasikan dengan permintaan.
Permintaan dapat memiliki beberapa kategori yang berhubungan dengan itu.
Kategori dapat ditambahkan dan diubah menggunakan Mengelola Kategori dialog.
Mereka juga dapat ditambahkan secara otomatis melalui Passive scanning - hal ini memungkinkan anda untuk secara otomatis halaman tag yang sesuai dengan ekspresi reguler tertentu.

Kategori yang ditampilkan dalam Sejarah tab dan dapat disaring menggunakan Sejarah Filter dialog.
