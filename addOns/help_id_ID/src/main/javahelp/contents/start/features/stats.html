Statistik

ZAP mempertahankan statistik yang bisa membantu anda untuk memahami apa yang sebenarnya terjadi ketika berinteraksi dengan aplikasi besar.

Statistik tersedia melalui API dan juga dapat dikirim ke server Statsd ketika dikonfigurasi melalui Layar Statistik Pilihan.

Situs berdasarkan statistik

Statistik pemeliharaan pada setiap situs meliputi:
  • kode respon, misalnya:
    • 200 kode statistik
    • 302 kode statistik
  • waktu balasan dalam ms (menggunakan skala logaritmik), misal nya:
    • 1 waktu balasan statistik
    • waktu ke 2 balasan statistik
    • waktu ke 4 balasan statistik
    • waktu ke 8 balasan statistik
    • waktu ke 16 balasan statistik
  • jenis muatan, misalnya:
    • statistik.Jeniskonten.text/css
    • statistik.Jeniskonten.text/html;charset =utf-8
  • label, misal:
    • statistik.tag.Password
    • statistik.tag.Tersembunyi
  • anticsrf token yang dihasilkan:
    • statistik.acsrf.anticsrf
  • info otentikasi:
    • statistik.auth.sukses (jumlah otentikasi keberhasilan)
    • statistik.auth.kegagalan (jumlah kegagalan otentikasi)
    • stats.auth.state.assumedin (number of messages between successful polls that are assumed to be logged in)
    • stats.auth.state.loggedin (number of messages that appear to be logged in)
    • stats.auth.state.loggedout (number of messages that appear to be logged out)
    • stats.auth.state.noindicator (number of messages where no logged in or out indicators have been set)
    • stats.auth.state.unknown (number of messages which don't contain either logged in or out indicators)
