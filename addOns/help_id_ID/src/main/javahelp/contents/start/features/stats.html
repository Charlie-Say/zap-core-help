Statistik

ZAP mempertahankan statistik yang bisa membantu anda untuk memahami apa yang sebenarnya terjadi ketika berinteraksi dengan aplikasi besar.

Statistik tersedia melalui API dan juga dapat dikirim ke server Statsd ketika dikonfigurasi melalui Layar Statistik Pilihan.

The full set of statistics maintained by the ZAP core and add-ons are maintained on the ZAP website: Internal Statistics.
