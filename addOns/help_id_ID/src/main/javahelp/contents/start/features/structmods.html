Merubah struktural

Structural Modifier adalah kontrol yang mengubah bagaimana ZAP mewakili struktur aplikasi.
Itu Sites Tree is ZAP's representation of the application.
Jika bukan representasi struktur yang bagus maka ZAP tidak akan bisa menyerang aplikasi secara efektif.
Saat ini ada 2 jenis Structural Modifiers:

  • Data driven konten yang mengidentifikasi jalur url yang mewakili data
  • Parameter struktural yang mengidentifikasi parameter yang mewakili struktur aplikasi dan bukan data pengguna
Pengubah Struktur hanya dapat diterapkan Konteks.
