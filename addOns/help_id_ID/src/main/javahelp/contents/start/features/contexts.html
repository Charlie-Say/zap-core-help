Konteks

Konteks adalah cara untuk menghubungkan satu set URL secara bersamaan
Anda bisa menentukan konteks yang anda sukai, namun diharapkan bahwa konteks akan sesuai dengan aplikasi web.
Sebaiknya anda menentukan konteks baru untuk setiap aplikasi web yang membentuk sistem yang anda uji, dan atur dalam lingkup ketika anda mengujinya satu persatu.

Contexts are defined as a set of regular expressions (regexs) which are applied to all of the URLs in the Sites Tree.
Note: The regular expressions must match the whole URL.

You can configure contexts via:

  • Right click menu items in the in the Sites or History tab
  • The Session Contexts dialogs.
