Catatan

Catatan adalah setiap teks yang ingin anda kaitkan dengan sebuah permintaan.
Sebagai contoh anda bisa menggunakan catatan untuk merekam rincian tes tambahan yang perlu dilakukan pada sebuah permintaan.
Catatan ditambahkan dan diubah menggunakan Tambahkan dialog Catatan.

Catatan ditandai pada History tab dengan ikon.
