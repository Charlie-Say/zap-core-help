API

ZAP menyediakan sebuah Application Programming Interface (API) yang mengizinkan anda untuk berinteraksi dengan ZAP pemrograman.

API tersedia dalam format JSON, HTML dan XML.
UI web sederhana yang memungkinkan anda menjelajahi dan menggunakan API yang tersedia melalui URL http://zap/ Ketika anda mem-proxy melalui ZAP, atau melalui host dan port ZAP adalah mendengarkan pada, misalnya http://localhost:8080/.

Secara bawaan hanya mesin ZAP yang berjalan bisa untuk mengakses API. Anda dapat mengizinkan mesin lain, yang dapat menggunakan ZAP sebagai proxy, akses ke API. Konfigurasi API menggunakan Options API screen.

API menyediakan akses untuk sebagian besar fitur inti ZAP seperti active scanner dan spider.
Versi selanjutnya dari ZAP akan meningkatkan fungsionalitas yang tersedia melalui API.
