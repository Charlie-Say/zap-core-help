Manipulator-in-the-middle Proxy

ZAP is a Manipulator-in-the-middle Proxy. Ini memungkinkan anda untuk melihat seluruh permintaan yang anda buat pada aplikasi web dan seluruh respon yang anda terima darinya.
Antara lain hal ini memungkinkan anda untuk melihat panggilan AJAX yang mungkin tidak jelas.

You can also set breakpoints which allow you to change the requests and responses on the fly.
