Rilis 1.3.0

Berikut perubahan yang dibuat dalam rilis ini:

Perubahan yang signifikan:

Rebranding OWASP

ZAP sudah diterima sebagai proyek OWASP.
Its homepage is now: https://owasp.org/www-project-zap/

Kasar

Kemampuan untuk mengacak file dan direktori berdasarkan kode dari proyek DirBuster OWASP.
Tab Kasar yang baru menunjukkan file dan direktori yang ditemukan.

Port Scan

Kemampuan untuk memindai port situs.
Tab Port Scan baru menunjukkan port yang ditemukan.

Scan Aktif situs

Yang baru Scan Aktif situs menunjukkan permintaan dan respon sebagai hasil dari memindai situs aktif.

Url laba-laba

Itu Url laba-laba sekarang memungkinkan anda untuk melanjutkan menggunakan ZAP saat menjelajah situs.
Anda juga bisa berhenti sebentar dan lanjut menjelajah.

Dukungan kartu pintar

Dukungan kartu pintar telah ditambahkan bersama proyek Andiparos.
Perangkat kartu pintar berikut sudah diuji pada Windows:
    MenjagaAladdin eToken
    Aladdin eToken Pro
    OmnikeyOmnikey 3121
    CardMan 6121
    GemaltoReflex 20 V2
    Swiss Stick
Perangkat smartcad berikut ini dilaporkan bekerja:
    OmnikeyCardMan 4040

Menu tempuhan

Yang baru Tab situs klik kanan 'Attack' menu memungkinkan anda untuk mulai bergai scan.

Internasionalisasi

Semua tab utama dan sekarang memiliki item menu yang sudah diinternasionalkan.

Lokalisasi

Dukungan untuk bahasan berikut yang dibangun ke dalam versi ini:
    InggrisMengatur ulang bahasa
    Brazil portugiss
    Jerman
    Polandia
    Spanyol

Arah Tulisan Bahasa

Pada awal memulai anda akan diminta untuk memilih bahasa untuk digunakan.
Bahasa baru secara otomatis terdeteksi oleh kehadiran file dengan nama bentuk Pesan_<locale>.properties dalam direktori ZAP.

Perubahan kecil:

Menonaktifkan plugin 'file default'

Plugin yang mendeteksi file default secara efektif dibuat berlebihan oleh pemindai kasar yang baru.
Ini karena itu sudah dinonaktifkan.

Ringkasan pemindaian

Hitungan jumlah dan jenis pemindaian saat ini ditampilkan sekarang footer.
