Rilis 1.0.0

Berikut perubahan yang dibuat dalam rilis ini:

Perubahan yang signifikan:

Layar bantuan

Layar bantuan yang telah ditambahkan untuk semua fungsi sekarang disediakan oleh ZAP.
Bantuan sensitif konteks juga telah diimplementasikan sehingga ketika Anda menekan tombol F1 maka layar bantuan yang terkait dengan tab yang telah Anda pilih yang akan ditampilkan.

Breakpoints

Implementasi dari Breakpoints telah diubah untuk menjadi lebih dekat dengan yang disediakan oleh perangkat lunak Lingkungan Pengembangan Terpadu.
The breakpoint controls have been moved to the new Bilah Alat Aras Puncak yang baru.
Breakpoints can now also be set by right clicking nodes in the tab Situs dan Sejarah.

Menu laporan

Menu Laporan telah berubah signifikan dan sekarang meliputi butir-butir menu berikut:
    Buat Laporan...Yang sekarang menghasilkan laporan saat diminta
    Ekspor Pesan ke Berkas...Pindah dari Menu Berkas
    Ekspor Respon ke Berkas...Pindah dari Menu Berkas
    Ekspor Semua URL ke Berkas...Fungsionalitas baru
    Bandingkan dengan sesi lain...Fungsionalitas baru

Pencarian sejarah yang lebih baik

Tab pencarian yang baru memungkinkan Anda untuk mencari ekspresi reguler pada semua URL, permintaan, dan tanggapan.
Tombol selanjutnya dan sebelumnya memungkinkan Anda melihat istilah pencarian yang ditemukan di tab Permintaan dan Tanggapan.

Penyaring riwayat

Bilah Alat Penyaring Riwayat sudah ditambahkan yang memungkinkan anda untuk membatasi permintaan yang ditampilkan.
Permintaan dapat difilter berdasarkan metode HTTP, kode hasil HTTP, dan apapun tag, peringatan, apa catatan yang telah diterapkan.

Catatan

Anda sekarang bisa mengasosiasikannya catatan dengan permintaan apapun.

Beberapa Tag

Sekarang Anda dapat mengaitkan beberapa tag dengan permintaan apapun.

Lebih banyak kontrol atas peringatan

Itu Menambahkan dialog lampiran memungkinkan anda menambahkan atau mengubah peringatan secara manual.

Encoder / decoder baru

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Pemindai pasif

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Perubahan kecil:

Tingkat atas Toolbar

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Ringkasan fotter peringatan

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Tampilan dan nuansa baru

A new look and feel 'Nimbus' is used.

Penanganan password proxy

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Kirim ulang dari tab Sites

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Direktori pengguna dipertahankan antara sesi

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
