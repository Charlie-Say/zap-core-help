Rilis 1.3.0

Berikut perubahan yang dibuat dalam rilis ini:

Perubahan yang signifikan:

Masalah 133: Tambahkan Sintaks penyorotan untuk Respon Panel

Panel HTML sekarang mendukung penyorotan sintaks yang dapat dialihkan.

Isu 153: integrasi fuzzdb

Fuzzer termasuk fuzzdb (https://github.com/fuzzdb-project/fuzzdb) file fuzzing.
Perhatian bahwa beberapa file fuzzdb yang telah ditinggalkan karena menyebabkan anti virus umum untuk bendera mereka sebagai yang mengandung virus.
Anda dapat menggantikan mereka (dan meningkatkan fuzzdb) dengan mengunduh versi terbaru dari fuzzdb dan memperluas di perpustakaan 'fuzzers'.

Masalah 212: Parameter analisis

Tab param baru menampilkan ringkasan dari semua parameter situs yang digunakan.

Masalah 228: Pemindai XSS ditingkatkan

Pemindai aktif persimpangan scripting sudah ditulis ulang dari nol untuk menemukan lebih banyak potensiaal masalah XSS dan laporan sedikit salah positif.

Masalah 244: Port Pengintai mengecek pasif

Berikut pemeriksaan yang telah porting dari Pengamat (terima kasih kepada Chris Weber untuk oking ini):
    Cek.Pasv.CrossDomain.ScriptReference.cspemeriksaan untuk cross domain javascript file inclusion.
    Cek.Pasv.Header.CacheControl.csmemeriksa HTTP cache-control header pada halaman SSL.
    Memeriksa.Pasv.Header.ContentTypeMissing.cscek bahwa Content-Type header HTTP ini tidak hilang.
    Memeriksa.Pasv.Header.FrameOptions.csmemeriksa X-FRAME-OPTION tidak hilang atau diatur dengan tidak aman.
    Memeriksa.Pasv.Header.leXssProtection.csmemeriksa yang X-XSS-Perlindungan tidak telah ditetapkan untuk menonaktifkan IE XSS perlindungan.
    Check.Pasv.Header.MimeSniff.cspemeriksaan X-CONTENT-TYPE-OPTIONS telah ditetapkan.
    Cek.Pasv.InformationDisclosure.DatabaseErrors.cspemeriksaan untuk database pesan kesalahan.
    Check.Pasv.InformationDisclosure.DebugErrors.cspemeriksaan untuk debugging pesan kesalahan.
    Cek.Pasv.InformationDisclosure.InUrl.csmemeriksa untuk informasi penyingkapan dalam parameter URL.
    Check.Pasc.InformationDisclosure.ReferrerLeak.csmemeriksa HTTP pengarah header untuk informasi penyingkapan.

Masalah 253: Ekstensi plugable

Penuh ekstensi sekarang dapat terhubung ke ZAP secara dinamis dengan akses penuh ke semua ZAPs fitur.
