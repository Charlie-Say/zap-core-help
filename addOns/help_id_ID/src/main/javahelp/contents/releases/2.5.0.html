Rilis 2.5.0

Berikut perubahan yang dibuat dalam rilis ini:

Perubahan API ZAP:

Lihat otorisasi / getAuthorizationDetectionMethod

Sekarang kembali data yang dilibatkan dalam sebuah objek disebut otorisasi Metode Deteksi Misalnya: 
{"authorizationDetectionMethod": {"statusCode": "-1"..."headerRegex": ""}}
melainkan: 
{"statusCode": "-1"..."headerRegex": ""}

Lihat konteks / konteks + technologyList

Keduanya sekarang mengembalikan data yang berulas dalam sebuah objek yang disebut "konteks" Sebagai contoh: 
{"konteks": {"id": "1",..., "inScope": "benar", "loggedOutPattern": ""}}
melainkan: 
{"konteks": {"id": "1",..., "inScope": "benar", "loggedOutPattern": ""}}

Laba-laba aksi / scan + scanAsUser

Sekarang keduanya mendukung parameter 'subtreeOnly' opsional baru yang membatasi laba-laba ke subtree tertentu. Parameter 'url' sekarang juga opsional, selama disertakannya parameter 'konteks' yang valid

Komponen 'statistik' baru

Komponen 'Statistik' API baru menyediakan akses ke statistik sekarang pemeliharaan oleh ZAP.

Perhatikan bahwa beberapa komponen yang ada juga akan memiliki operasi baru, lihat API Web UI untuk lebih jelasnya.
