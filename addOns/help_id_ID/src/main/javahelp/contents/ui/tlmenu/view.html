Menu Tampilan

Menu ini menangani tampilan pilihan.

Aktifkan Gambar dalam Histori

Jika dipilih maka gambar akan ditampilkan di Tab Riwayat.

Tampilkan Tab

Hal ini memungkinkan anda untuk menampilkan semua tab yang tersedia, termasuk orang-orang yang anda telah tersembunyi.

Tampilkan Semua Tab

Hal ini memungkinkan anda untuk menampilkan semua tab.

Menyembunyikan Tab Yang Dipasangi Pin

Hal ini memungkinkan anda untuk menyembunyikan semua tab yang tidak disematkan.

Pin Semua Terlihat Tab

Hal ini memungkinkan anda untuk pin semua terlihat tab.

Lepaskan Semua Tab

Hal ini memungkinkan anda lepaskan semua tab.

Note that add-ons can add additional menu items.
