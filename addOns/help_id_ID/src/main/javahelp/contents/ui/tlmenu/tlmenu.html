Menu Tingkat Teratas

Secara lalai item menu tingkat teratas berikut akan hadir:
     Berkas menu Ini sesi menangani saat ini
     Menu koreksi This handles finding strings in specific tabs, searching for strings across all requests and responses, changing ZAP's mode, and managing forced user mode
     Lihat menu Ini menangani sebuah tampilan pilihan
     Menu analisis Ini menangani Kebijakan Pemindaian
     Menu laporan Ini menangani sebuah laporan
     Menu alat Ini menangani sebuah alat dan pilihan umum
     Import menu This gives access to menus that allow to import data into ZAP.
     Online menu This gives access to a set of online resources
     Help menu This gives access to the 'about' dialog and this help file

Perhatikan itu add-ons bisa menambahkan item menu tambahan.
