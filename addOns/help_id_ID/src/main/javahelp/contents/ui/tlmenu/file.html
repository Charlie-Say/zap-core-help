Menu file

Menu ini menangani sesi saat ini. Secara default menu berikut barang-barang yang akan hadir:

Sesi Baru

Ini menciptakan sesi baru.
Jika kamu belum menyimpan sesi kamu saat ini maka peringatan akan ditampilkan.
Memulai sesi baru tanpa menyimpan sesi saat ini akan kehilangan semua data dalam sesi saat ini.

Membuka Sesi...

Ini akan membuka sesi yang telah disimpan sebelumnya.
Pembukaan sesi tanpa menyimpan sesi saat ini akan kehilangan semua data dalam sesi saat ini.

Bertahan Sesi...

Ini terus berlanjut sesi saat ini.
Sementara sesi selalu disimpan dalam database pada disk, maka akan hilang saat ZAP dihentikan kecuali jika telah bertahan.
Kamu hanya butuh untuk bertahan sekali - setelah itu semua perubahan akan disimpan.

Sesi Snapshot Sebagai...

Ini menyimpan cuplikan dari sesi yang telah berlangsung lama.
Ini akan menyarankan nama berkas yang sama seperti yang bertahan sesi dengan tanggal-waktu string yang ditambahkan ke dalamnya, dan memungkinkan pengguna untuk mengatur apapun nama yang mereka pilih.

Properti Sesi...

Ini menampilkan Properti Sesi dialog.
Ini memungkinkan Anda untuk mengatur sesi nama dan deskripsi.

Impor Konteks...

Mengizinkan untuk mengimpor sebuah Konteks.

Ekspor Konteks...

Mengizinkan untuk mengekspor Konteks.

Muat berkas Add-on...

Beban lokal Tambahan-on berkas.
Add-ons yang biasanya dipasang melalui Mengelola Add-ons dialog, tapi pilihan ini dapat berguna jika anda telah mengunduh add-on secara manual atau pengujian salah satu yang telah anda kembangkan sendiri.
Add-ons tetap dipasang sampai anda secara manual menghapus mereka.

Keluar dan Menghapus Sesi...

Ini akan keluar dari ZAP dan menghapus sesi, bahkan jika kamu sebelumnya pernah melakukannya.
Sesi ini akan tidak dapat diakses lagi ketika anda me-restart ZAP, meskipun setiap foto yang anda ambil akan tetap tersedia.
Dialog peringatan akan ditampilkan untuk memastikan anda benar-benar dimaksudkan untuk memilih opsi ini.

Keluar

Ini akan keluar ZAP.
Jika anda belum menyimpan sesi saat ini kemudian anda akan diberikan pilihan untuk melakukannya.

Note that add-ons can add additional menu items.
