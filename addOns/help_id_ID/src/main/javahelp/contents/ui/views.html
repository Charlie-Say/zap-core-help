Tampilan

ZAP menyediakan satu set tampilan yang dapat dipindah-pindahkan yang memungkinkan anda menampilkan permintaan dan tanggapan dengan berbagai cara yang berbeda.
Berikut tampilan yang disertakan secara default:
    TeksDalam dalam format teks
    HexSebuah tabel yang menunjukkan representasi hex dari semua karakter di header dan body.
Untuk layar yang dapat disunting ini memungkinkan anda menambahkan karakter kontrol melalui kode hex mereka.
Anda tidak bisa menambahkan atau menghapus karakter dalam tampilan ini - untuk melakukannya maka beralihlah ke salah satu tampilan lain dan kemudian kembali melanjutkan tampilan hex.
    GambarGambar - hanya tersedia untuk tubuh yang berisi gambar
    TabelSebuah tabel dengan satu baris per atribut dan nilai.
Untuk layar yang bisa disunting semua nilai secara otomatis akan dikodekan URL ketika diserahkan.
    Tabel (adv)Tabel dengan satu baris per atribut serta nilai yang ditambahkan fungsi tambahan.
Untuk nilai layar yang bisa disunting tidak akan secara otomatis dikodekan oleh URL saat dikirimkan, tetapi ini bisa dilakukan dengan 'manual' melalui fungsi 'URLEncode'.
    Besar PermintaanPlaceholder view digunakan untuk mencegah UI mencoba untuk beban yang sangat besar permintaan tubuh.
Anda akan perlu untuk beralih ke tampilan yang berbeda untuk menampilkan isi sebenarnya.
ambang batas untuk tampilan dapat diubah melalui Opsi Tampilan layar
    Besar ResponPlaceholder view digunakan untuk mencegah UI mencoba untuk beban yang sangat besar respon tubuh.
Anda akan perlu untuk beralih ke tampilan yang berbeda untuk menampilkan isi sebenarnya.
ambang batas untuk tampilan dapat diubah melalui Opsi Tampilan layar
