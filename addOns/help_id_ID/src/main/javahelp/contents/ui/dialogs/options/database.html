Pilihan layar basis data

Layar ini memungkin anda untuk mengkonfigurasi pilihan basis data:

Kompak (pada keluar)

Memungkinkan untuk kompak database ketika ZAP ditutup, pemadatan database memastikan minimal pemakaian ruang disk tapi hal ini juga akan memakan waktu lebih lama untuk keluar dari ZAP.

Log Pemulihan

Mengontrol apakah log pemulihan basis data diaktifkan atau tidak.
Meningkatkan kinerja database saat tidak diaktifkan namun dapat menyebabkan hilangnya data jika ZAP keluar secara tiba-tiba.
Catatan: sesi saat ini tidak akan terpengaruh, perubahan akan berpengaruh pada sesi baru dan sesi terbuka.

Ukuran tubuh maksimum permintaan

Ukuran tubuhnya permintaan terbesar dalam byte yang ZAP akan memungkinkan.

Ukuran tubuh respon yang maksimum

Ukuran tubuh respon terbesar dalam byte yang ZAP akan memungkinkan.

Meminta pilihan ketukunan untuk sesi baru

Jika diperiksa kemudian Bertahan sesi dialog akan ditampilkan saat sesi baru telah dibuat.

Pilihan bawaan

Pilihan bawaan untuk sesi baru - jika prompt di atas diaktifkan maka opsi default akan dipilih.
Jika prompt tidak diaktifkan maka inilah tindakan yang akan dilakukan.
