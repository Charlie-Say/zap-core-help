Sesi Konteks Struktur layar

Ini adalah salah satu dari Acara Pertunjukan Filem yang memungkinkan kamu untuk mengelola pemahaman ZAP tentang struktur aplikasi.

URL pasangan nilai Kunci pemisah

Karakter yang memisahkan pasangan kunci dan nilai dalam URL, secara umum '&'.

URL nilai Kunci pemisah

Karakter yang memisahkan kuncinya dan nilai pada URL, secara default '='.

POSTING pasangan nilai Kunci pemisah

Karakter yang memisahkan pasangan kunci dan nilai-nilai dalam POSTING data, secara default '&'.

POSTING nilai Kunci pemisah

Karakter yang terpisah kunci dan nilai-nilai dalam POSTING data, secara default '='.

Struktural Pengubah

Itu Struktural Pengubah yang berlaku untuk konteks ini, yang mungkin:
  • Data Driven Content which identify URL paths that represent data
  • Structural Parameters which identify parameters that represent application structure instead of user data
