Memindai kemajuan dialog

Ini menunjukkan sebuah status dari anda aktif scan.

Tab kemajuan

Ini menunjukkan scan aturan yang berjalan untuk masing-masing host yang discan, serta rincian lainnya seperti waktu berlalu mereka telah berjalan dan jumlah permintaan yang dibuat per aturan.
Hal ini juga memungkinkan anda untuk melewati aturan yang saat ini dijalankan dengan mengklik tombol 'lewati pemindaian aktif saat ini' di kolom status.

Tab diagram tanggapan

Ini menunjukkan jumlah tanggapan perdetik yang diterima oleh ZAP saat melakukan pemindaian aktif terhadap suatu situs.
Tanggapan yang memetakan per 'HTTP kode status respon kelompok':
  • 1xx informasi
  • 2xx sukses
  • 3xx pengalihan
  • 4xx kesalahan klien
  • 5xx kesalahan server
Anda dapat memperbesar grafik dengan memilih area menggunakan mouse anda.
Anda juga bisa klik kanan grafik untuk lebih banyak pilihan.
Garis vertikal menunjukkan kira-kira ketika masing-masing scan aturan dimulai (hal ini sangat sulit untuk menjadi tepat).
Bagan hanya akan diperbarui saat dibuka, jika anda menutup dan kemudian membuka kembali bagan maka data sebelumnya akan hilang.
Secara default waktu maksimum grafik akan mencakup lebih 10 menit, anda dapat mengubah bahwa melalui Pilihan layar pindai aktif.
