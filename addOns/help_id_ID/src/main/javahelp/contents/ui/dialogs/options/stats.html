Opsi layar Statistik

This screen allows you to configure the statistics that ZAP maintains.

Aktifkan Dalam Memori Statistik

Cara default ZAP menyimpan statistik dalam memori. Untuk menonaktifkan statistik memori hapus centang pada kotak ini.

Server Statistik diaktifkan

Periksa kotak ini untuk mengubah alamat server stadsd yang akan dikirim oleh ZAP kedalam statistik.

Stadsd Hostname

Server host Statsd sedang berjalan.

Port Statsd

Port yang ditangani oleh server Statsd, default 8125.

Prefix Statsd

Prefix ZAP akan menambahkan statistik yang dikirimkan ke server Statsd.
