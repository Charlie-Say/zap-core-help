Memindai kebijakan dialog manager

Hal ini memungkinkan anda untuk mengelola kebijakan pemindaian menentukan sebuah aturan yang dijalankan ketika melakukan aktif scan.
Anda bisa memiliki sebanyak mungkin kebijakan pemindaian yang anda suka dan memilih salah satunya untuk dijalankan ketika anda melakukan pemindaian aktif.

Tambah

Tombol add akan meluncurkan Memindai kebijakan dialog semua pengaturan default.

Memodifikasi

Memodifikasi tombol peluncuran Memindai kebijakan dialog semua yang dipilih dipindai oleh kebijakan.

Hilangkan

Tombol Remove akan menghapus kebijakan pindaian terpilih jika anda memberi konfirmasi pada dialog peringatan.

Impor

Tombol Impor memungkinkan anda untuk mengimpor kebijakan pemindai yang dibuat oleh contoh lain dari ZAP.

Ekspor

Tombol Ekspor memungkinkan anda untuk mengekspor kebijakan pemindai sehingga anda bisa mengimpornya ke contoh ZAP yang lain, yang bisa saja ada pada mesin lain.

Diakses via

    Menu analisis tingkat atas'Memindai kebijakan manager ...' menu item
    Tingkat atas Toolbar'Memindai kebijakan tombol manager ...'
    Tab pemindai aktif'Memindai kebijakan tombol manager ...'

Melihat juga

     Peninjauan UIuntuk sebuah peninjauan antara pengguna
     Dialoguntuk rincian dialog atau popups
