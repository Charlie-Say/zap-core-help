Pilihan layar sesi HTTP

This screen allows you to configure the HTTP Sessions Extension's options:

Tanda-tanda default

Ini adalah nama dari Cookie yang dianggap secara default menjadi sesi token. Segera setelah mereka terdeteksi sebagai parameter pada situs, mereka akan ditambahkan secara otomatis sebagai sesi token sesi untuk situs tersebut, kecuali jika ditimpa secara manual dengan perintah pengguna.

Sesi token yang digunakan merupakan huruf tidak peka (case-insensitive), oleh sebab itu sesi bawaan token akan otomatis menjadi huruf kecil. Juga, setiap modifikasi token sesi default dapat terlihat setelah kunjungan kembali pada halaman Situs.

Aktifkan sebagai Proxy saja

Jika pilihan ini diaktifkan, ekstensi hanya akan memproses pesan yang dikirim dan diterima melalui Proxy (misal dari/ke peramban). Pesan dari Ekstensi lain hanya akan diproses jika opsi ini dinonaktifkan.
