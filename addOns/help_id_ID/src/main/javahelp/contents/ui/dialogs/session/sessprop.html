URL regexes

Dalam Tidak termasuk * dialog, anda dapat memasukkan ekspresi reguler untuk mendefinisikan Url dikecualikan. Sementara anda dapat membuka satu meta-karakter dengan garis miring, anda juga dapat menggunakan \Q...\E escape sequence. Semua karakter antara \Q dan \E ditafsirkan sebagai karakter yang persis. Contoh: \Q*\d+*\E persis dengan teks *\d+*. Urutan keluar ini digunakan di dalam ZAP saat mengecualikan URL melalui beberapa konteks menu.
Catatan: Jika URL anda berisi "\E", maka anda harus melakukan langkah-langkah berikut saat menggunakan \Q...\E urutan keluar:
  • Buka escape sequence
  • Tutup urutan keluar sebelum "karakter" \E
  • Melarikan diri backslash
  • Buka setelah urutan keluar lainnya "\E";
  • Menutup escape sequence seperti biasanya.

Contoh: subdomain.example.com/path?a=\E&moredata=2 seharusnya muncul sebagai Contoh: subdomain.example.com/path?a=\\E\Q&moredata=2\E
