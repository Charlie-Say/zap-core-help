Layar pilihan bahasa

Layar bahasa memungkinkan Anda untuk mengkonfigurasi:

Pilihan bahasa

Bahasa ZAP menggunakan tampilan.
Satu set bahasa dibangun ke dalam ZAP, namun lebih banyak bahasa mungkin tersedia setelah rilis terakhir.
Periksa halaman download ZAP untuk paket bahasa terbaru yang tersedia.

Gunakan sistem lokal untuk format

Jika format nomor, tanggal, dan waktu dilakukan dengan sistem lokal bukan lokal bahasa yang dipilih.

Impor paket bahasa

Selain pemilihan bahasa yang digunakan untuk tampilan ZAP's, impor alat memungkinkan Anda untuk mengimpor paket bahasa baru atau update file bahasa. Paket file bahasa akan selalu memiliki ekstensi zaplang. (E.g. language-pack-en_GB.zaplang)
Setelah mengimpor Anda segera akan dapat memilih bahasa baru diimpor dari bahasa pilihan menu drop-down di bagian atas layar.
Perhatikan bahwa Anda akan perlu restart ZAP untuk akhirnya menerapkan bahasa baru.
