Dialog laba laba

Dialog ini meluncurkan laba laba.

Cakupan

Tab pertama memungkinkan anda untuk memilih atau mengubah titik awal.
Jika titik awal adalah satu atau lebih Konteks maka Anda akan dapat memilih salah satunya.
Jika konteks itu ada Pengguna yang ditetapkan maka Anda akan dapat memilih salah satunya.
Jika anda memilih salah satu dari pengguna maka spider akan dilakukan sebagai pengguna, dengan ZAP (re)otentikasi sebagai user tersebut bila diperlukan.

If you select 'Recurse' then all of the nodes underneath the one selected will also be used to seed the spider.

Jika anda memilih 'Spider Subtree Only' laba-laba hanya akan mengakses sumberdaya yang berada dibawah titik awal (URL). Ketika mengevaluasi jika sumber daya ini ditemukan dalam waktu yang telah ditentukan sub, laba-laba hanya mempertimbangkan skema, host, port, dan path komponen dari URI.

If you select 'Show Advanced Options' then the following tab will be shown which provide fine grain control over the spider process.

Mengklik tombol 'reset' akan mengatur ulang semua opsi ke nilai defaultnya.

Maju

Parameter pada tab ini sesuai dengan parameter yang sama di Pilihan layar spider.
