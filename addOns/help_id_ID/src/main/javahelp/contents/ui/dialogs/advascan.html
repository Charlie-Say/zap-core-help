Dialog Pemindaian Aktif

Dialog ini meluncurkan pemindai aktif.

Cakupan

Tab pertama memungkinkan Anda untuk memilih atau mengubah titik awal.
Jika Anda punya lebih dari satu kebijakan pemindaian maka Anda akan dapat memilih mana yang akan digunakan.
Jika titik awal adalah satu atau lebih Konteks maka Anda akan dapat memilih salah satunya.
Jika konteks tersebut memiliki Pengguna yang ditetapkan maka Anda akan dapat memilih salah satunya.
Jika Anda memilih salah satu pengguna maka pindai aktif akan dijalankan sebagai pengguna tersebut, dengan ZAP (re)otentikasi sebagai pengguna tersebut kapanpun diperlukan.

If you select 'Recurse' then all of the nodes underneath the one selected will also be scanned.
Vektor masukan tersuai hanya didukung jika opsi ini tidak dipilih.

If you select 'Show Advanced Options' then the following tabs will be shown which provide fine grain control over the active scanning process.

Mengeklik tombol 'Reset' akan mengatur ulang semua opsi ke nilai bawaannya.

Vektor Masukan

Tab Vektor Masukan memungkinkan Anda untuk membatalkan vektor masukan bawaan yang ditetapkan dalam Layar Pilihan Pindai Aktif Vektor Masukan.
Mengeklik tombol 'Reset' akan mengatur ulang vektor masukan ke opsi bawaannya.

Vektor Suai

Tab Vektor Suai memungkinkan Anda menentukan lokasi tertentu dalam permintaan untuk menyerang.
Custom Vectors are only available if the 'Recurse' option on the first tab is not selected.
Untuk menambahkan vektor masukan sorot karakter yang ingin Anda serang dalam permintaan dan klik tombol 'Tambah'.
Anda dapat menambahkan vektor masukan suai sebanyak yang Anda inginkan.
Untuk menghapus vektor masukan suai sorot karakter mana saja yang dipilih dan klik tombol 'Hapus'.
Memeriksa kotak 'Menonaktifkan vektor input non suai' akan menonaktifkan semua vektor masukan kecuali yang Anda tentukan secara manual pada tab ini.

Teknologi

Tab Teknologi memungkinkan Anda untuk menentukan jenis teknologi untuk dipindai.
Membatalkan pilihan teknologi yang Anda tahu tidak ada dalam aplikasi tujuan dapat mempercepat pemindaian karena aturan-aturan yang menargetkan teknologi tersebut dapat melewatkan tes-tes itu.

Kebijakan

Tab Kebijakan memungkinkan Anda untuk membatalkan pengaturan apapun yang ditetapkan dalam kebijakan pemindaian.

Filter

The Filter tab allows you to specify filter criteria and active scan will filter requests based on these criteria's.
There are four filter criteria's available in the filter tab
  • HTTP method
  • Status code
  • Tags
  • URL pattern
HTTP method and Status code filter criteria are only applied to check if a request can be included in active scan or not,
where as Tags and URL pattern filter criteria can be applied to include or exclude a request.
