Pasang ulang ke Bawaan Pabrik

Tombol 'Pasang ulang ke Bawaan Pabrik' memasang ulang semua opsi ke nilai yang ditetapkan ketika pertama kali anda menjalankan ZAP. Ini mencakup reeseting host dan port yang ZAP nyalakan serta kunci API.
Salah satu item konfigurasi yang tidak akan berubah adalah ZAP Root CA Sertifikat. Jika anda ingin mengubah ini, maka anda dapat melakukannya melalui Sertifikat SSL Dinamis layar.

Diakses melalui

     Tingkat atas Alat menu'Pilihan...' item menu
