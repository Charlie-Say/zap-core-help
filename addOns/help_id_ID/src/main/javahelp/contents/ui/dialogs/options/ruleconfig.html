Pilihan Konfigurasi Aturan layar

Layar ini memungkinkan kamu untuk mengkonfigurasi perilaku aturan pemindaian aktif dan pasif yang spesifik.

Pilih salah satu baris yang tercantum untuk melihat rincian konfigurasi spesifik dan untuk mengubah nilai yang terkait. Baik individu dan semua peraturan bisa diatur ulang.

Aturan yang dibangun meliputi:

NamaStandarDeskripsi
aturan.umum.tidur15Panjang waktu dalam detik yang digunakan untuk waktu serangan.
rules.cookie.ignorelistDaftar dipisahkan koma dari nama cookie. Kue yang termasuk dalam daftar ini akan diabaikan ketika pemindaian untuk cookie isu-isu terkait.
rules.csrf.ignorelistDaftar pengenal yang dipisahkan koma. Formulir mana pun dengan nama atau ID yang cocok dengan salah satu pengidentifikasi ini akan diabaikan saat memindai bukti Anti-CSRF yang hilang. Gunakan saja fitur ini untuk mengabaikan formulir yang kamu tahu aman, misalnya formulir pencarian.
rules.csrf.ignore.attnameThe name of an HTML attribute that can be used to indicate that a form does not need an anti-CSRF Token. If rules.csrf.ignore.attvalue is specified then this must also match the attribute's value. If found any related alerts will be raised at INFO level.
rules.csrf.ignore.attvalueThe value of an HTML attribute named by rules.csrf.ignore.attname that can be used to indicate that a form does not need an anti-CSRF Token. If found any related alerts will be raised at INFO level.
rules.domains.trustedA comma separated list of URL regex patterns. Any URLs that match the patterns will be considered trusted domains and the issues ignored.
