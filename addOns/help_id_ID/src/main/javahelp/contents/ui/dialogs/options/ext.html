Pilihan Ekstensi layar

This screen allows you to enable/disable extensions that are available to ZAP, either core or added by add-ons.

Mengaktifkan/Menonaktifkan Ekstensi

Ekstensi yang diaktifkan akan dimuat oleh ZAP, sehingga menambah fungsionalitas yang disediakan oleh ekstensi tersebut. Ekstensi yang dinonaktifkan tidak akan dimuat.

Inti Ekstensi

Inti ekstensi adalah ekstensi bahwa jika dinonaktifkan akan mengganggu ZAP fungsi minimal. Ekstensi ditandai sebagai inti tidak dapat dinonaktifkan.

Catatan: Ekstensi inti dapat dinonaktifkan melalui file konfigurasi, meskipun tidak disarankan.

Ekstensi' Dependensi

Ekstensi mungkin bergantung pada ekstensi lain untuk bekerja. Bila salah satunya ekstensi (dependensi) dinonaktifkan maka akan membuat ZAP otomatis nonaktifkan semua ekstensi yang bergantung padanya. Jadi mungkin itu terjadi ketika menonaktifkan satu ekstensi ekstensi lain (yang bergantung padanya) akan juga dinonaktifkan. Ekstensi yang dinonaktifkan secara otomatis tidak akan diizinkan untuk diaktifkan sampai semua dependensi diaktifkan.

Catatan: Anda perlu memulai ulang ZAP agar perubahan diterapkan.
