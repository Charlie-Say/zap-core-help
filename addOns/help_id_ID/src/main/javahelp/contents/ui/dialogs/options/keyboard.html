Pilihan Keyboard layar

Layar ini memungkinkan Anda mengonfigurasi pintasan keyboard untuk semua menu ZAP.

Mengklik salah satu baris tabel akan memunculkan kotak dialog yang memungkinkan Anda mengubah cara pintas keyboard.

'Action' Cheatsheet

Mengklik tombol ini akan menghasilkan halaman HTML yang menunjukkan semua shortcut keyboard yang diurutkan berdasarkan 'action'.
Tombol ini hanya diaktifkan jika API diaktifkan dan sistem operasi Anda mendukung pembukaan URL di browser.

'Kunci' Cheatsheet

Mengklik tombol ini akan menghasilkan halaman HTML yang menunjukkan semua shortcut keyboard yang diurutkan berdasarkan 'key'.
Tombol ini hanya diaktifkan jika API diaktifkan dan sistem operasi Anda mendukung pembukaan URL di browser.

Setel ulang

Mengklik tombol ini akan ulang semua pintasan ke default mereka.
