Tab pecahan

Istirahat tab memungkinkan anda untuk mengubah permintaan atau tanggapan ketika telah tertangkap oleh ZAP melalui breakpoint.
Hal ini memungkinkan anda untuk mengubah unsur-unsur yang anda biasanya tidak akan dapat mengubah melalui browser anda, termasuk:
     Header
     Bidang tersembunyi
     Bidang cacat
     Bidang-bidang yang menggunakan javascript untuk menyaring karakter ilegal
Fungsi ini adalah kunci untuk secara efektif pena pengujian aplikasi anda.

2 panel hanya akan berisi apa saja jika ZAP telah menangkap permintaan atau tanggapan.
Anda dapat mengubah apa pun dalam 2 panel dan kemudian meneruskan permintaan atau tanggapan yang menggunakan tombol-tombol pada Tingkat Atas Toolbar.

Pull downs memungkinkan anda untuk memilih yang berbeda Pandangan untuk permintaan atau tanggapan header dan body.

While the Break tab is not in use its icon is a grey cross:    .

If you have selected "Show buttons to select the requests you don't want ZAP to break on" in the options menu you will be able to select if you just want to break on requests that are in scope and you will be able to select the file extensions you don't want to break on:
       Javascript Files
       CSS and Fonts Files
       Multimedia Files
       Only in scope

  Update Content Length

By default ZAP will automatically update (or add) the HTTP(S) Content-Length header to match the size of the data submitted. If this option is unselected then ZAP will not update the header allowing you to specify any value you like.
This button will only be shown for HTTP(S) messages.
