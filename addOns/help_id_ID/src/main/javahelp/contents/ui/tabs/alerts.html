Tab peringatan

The Alerts tab show the Alerts that have been raised in this session. Peringatan yang ditampilkan di sebuah pohon di risiko order di panel kiri, dan masing-masing node dari pohon menunjukkan jumlah dari tanda di bawah ini.

Memilih peringatan dengan sekali klik akan ditampilkan di panel sebelah kanan. Mengklik ganda sebuah peringatan akan menampilkan Tambahkan peringatan dialog yang akan memungkinkan anda untuk mengubah rincian peringatan.

Menyaring alert

Tombol filter, segera ditampilkan di atas pohon, yang memungkinkan anda untuk membatasi peringatan yang ditampilkan:

 /    Tampilkan hanya URL pada Scope / Tampilkan semua URL

Memungkinkan untuk hanya menampilkan peringatan dari URL yang ada di cakupan.

 /    Tautkan / Batalkan tautan dengan seleksi Situs

Memungkinkan untuk menampilkan hanya dari peringatan Tab situs simpul pohon terpilih.

Catatan: Pilihlah satu penyaring yang akan menonaktifkan yang lainnya karena ekslusif satu sama lain.

  Sunting peringatan

Akan menampilkan Tambah peringatan dialog yang akan memungkinkan anda untuk mengubah rincian peringatan, untuk peringatan yang dipilih.

Catatan: Jika beberapa tanda yang dipilih pada pohon, yang paling baru saja dipilih item adalah diedit.

  Delete All Alerts

Allows to delete all alerts.

Tekan pilihan kanan

Right clicking on a node will bring up a menu which will allow you to:

Open/Resend with Request Editor...

This will bring up the Manual Request Editor dialog which allows you to resend the request associated with the alert after making any changes to it that you want to.
