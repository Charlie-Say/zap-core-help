Tanggapan penyebut

Tanggapan penyebut menampilkan data yang dikirim ke browser anda untuk meminta bahwa anda telah disorot dalam baik Situs penyebut atau Sejarah tab .

Tarik kebawah memungkinkan anda untuk memilih berbagai Menampilkan untuk menanggapi yang terjatuh dan badan.

Note that images are only shown in the Sejarah tab jika 'Mengaktifkan Gambar dalam Sejarah' bendera di Menampilkan pilihan yang dipilih.

Tekan pilihan kanan

Tekan kanan pada sinpul akan memunculkan pilihan yang akan memungkinkan anda untuk:

Cari...

Ini akan memunculkan Temukan percakapan.

Copy

This will copy the selected string to the clipboard.
