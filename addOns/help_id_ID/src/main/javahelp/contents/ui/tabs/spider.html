Label Laba-laba

The Spider tab shows you the set of unique URIs found by the Spider during the scans.

'Baru Scan' meluncurkan tombol laba-Laba dialog yang memungkinkan anda untuk menentukan dengan tepat apa yang harus dipindai.
laba-Laba dapat dijalankan pada beberapa Situs secara paralel dan hasil untuk setiap scan yang ditampilkan dengan memilih scan melalui 'Kemajuan' pull-down.

Toolbar menampilkan informasi tentang pemindaian dan memungkinkan untuk melakukan pengontrolan. Ini menyediakan satu set tombol yang memungkinkan untuk:

  • Istirahat (and resume) pindai Laba-laba yang dipilih;
  • Berhenti pindai Laba-laba yang dipilih;
  • Bersih selesai scan;
  • Open the Spider Options screen.
Progress bar menunjukkan seberapa jauh pemindaian laba-laba yang dipilih telah berkembang. Hal ini juga menunjukkan jumlah pemindaian laba-laba yang aktif dan jumlah URL yang ditemukan untuk pemindaian yang dipilih.

Untuk setiap URI yang ditemukan anda bisa melihat:

  • Diproses - Apakah URI diproses oleh laba-Laba atau dilewati dari mengambil karena aturan (misalnya itu keluar dari ruang lingkup)
  • Metode - metode HTTP, misalnya GET atau POST, di mana sumber daya harus dapat diakses
  • URI - sumber daya yang ditemukan
  • Bendera - setiap informasi tentang URI (misalnya jika itu adalah sebuah benih atau mengapa itu tidak diproses)

For each spider message, shown under the Messages tab, you can see details of the request sent and response received. The Diproses column, indicates whether:

  • Successfully - the response was successfully received and parsed
  • Empty Message - the response was not parsed because it was empty
  • I/O Error - an input/output error occurred while fetching the response
  • Max Children - the response was not parsed because the corresponding parent Sites node already has more child nodes than the maximum allowed
  • Max Depth - the response was not parsed because it passed the maximum depth allowed
  • Max Size - the response was not parsed because its size is not under the maximum allowed
  • Not Text - the response was not parsed because it's not text, for example, an image
  • Spider Stopped - the response was not fetched or parsed because the spider was already stopped
