Tab permintaan

Permintaan tab menampilkan data yang dikirimkan oleh browser anda untuk permintaan yang telah anda soroti di salah satu Tab situs atau Riwayat tab .
Panel atas menunjukkan header permintaan dan panel bawah menunjukkan data apapun juga dikirim, misalnya sebagai hasil dari permintaan POST.

Tarik kebawah memungkinkan anda untuk memilih berbagai Tampilan untuk permintaan header dan tubuh.

Tekan pilihan kanan

Klik kanan pada node akan memunculkan menu yang akan memungkinkan anda untuk:

Cari...

Ini akan memunculkan Temukan percakapan.

Copy

This will copy the selected string to the clipboard.
