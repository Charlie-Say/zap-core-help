Versione 1.0.0

I seguenti cambiamenti sono stati apportati a questa versione:

Cambiamenti significativi:

Schermate d'aiuto

Le schermate d'aiuto sono state aggiunte per tutte le funzionalità offerte da ZAP.
Anche l'aiuto in base al contesto è stato implementato, in modo che quando premi il tasto F1 verrà visualizzata la schermata d'aiuto associata alla scheda che hai selezionato.

Breakpoints

L'implementazione dei Breakpoints è stata cambiata per essere più ravvicinata a quella fornita dai software integrati degli ambienti di sviluppo.
The breakpoint controls have been moved to the new Top Level Toolbar.
Breakpoints can now also be set by right clicking nodes in the Siti e Cronologia.

Menu delle segnalazioni

Il Menu delle segnalazioni ha subito dei cambiamenti significativi e ora include i seguenti elementi:
    Genera segnalazione...Che ora genera segnalazioni su richiesta
    Esporta messaggi in un File...Spostato da Menu File
    Esporta Risposta su File...Spostato da Menu File
    Esporta tutti gli URL su File...Nuova funzionalità
    Confronta con un altra sessione...Nuova funzionalità

Migliorata la ricerca nella cronologia

La nuova Casella di ricerca ti permette di cercare espressioni regolari in tutti gli URL, richieste e risposte.
I tasti seguente e precedente ti permettono di visualizzare i termini cercati in Richiesta e Caselle di risposta.

Filtro per la cronologia

Il toolbar del filtro per la cronologia è stato aggiunto per permetterti di limitare le richieste che vengono visualizzate.
Le richieste possono essere filtrate basandosi sui metodi HTTP, sui codici risultato HTTTP, e su ogni tag, avvisi, oppure annotazioni che sono state applicate.

Annotazioni

Ora puoi associale le annotazioni con qualsiasi richiesta.

Tag multipli

Ora puoi associare molteplici tag con qualsiasi richiesta.

Più controllo sugli avvisi

La funzione Aggiungi Avviso ti permette do aggiungere o cambiare manualmente un avviso.

Nuovo codificatore/decodificatore

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Scansione passiva

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Cambiamenti minori:

Il Toolbar Top Level

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Riepilogo avvisi a piè di pagina

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Nuovo aspetto

A new look and feel 'Nimbus' is used.

Gestione delle password Proxy

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Rispedisci dalla scheda dei siti

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Directory utente mantenuta tra le sessioni

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
