Keluaran 1.3.0

Perubahan berikut telah dilakukan dalam keluaran ini:

Perubahan signifikan:

Ujian Kabur

Rentetan dalam suatu maklum balas sekarang boleh dikaburkan untuk cuba mencarikan kelemahan.
Token anti CRSF boleh dikesan dan dijanakan semula secara automatik semasa ujian kabur.
Fungsi ini adalah berasaskan kod daripada projek OWASP JBroFuzz.

Sijil SSL Dinamik

Sokongan bagi sambungan SSL telah ditambah baikkan dan diringkaskan.
Pengguna sekarang boleh mencipta sijil root mereka sendiri dan mengedarkan ini ke dalam klien HTTP mereka.
Sila lihat Pilihan: Sijil SSL Dinamik untuk maklumat selanjutnya.

Mod Daemon dan API

Memulakan ZAP dengan pilihan baris perintah "-daemon" akan menyebabkannya untuk berjalan di latar belakang dalam mod 'tanpa kepala', yang bermaksud tiada UI yang akan dipaparkan.
Suatu API yang awal telah dilaksanakan dalam XML, JSON dan HTML.
Jika ZAP berjalan sebagai suatu daemon kemudiannya API akan diaktifkan secara automatik, kalau tidak API mesti diaktifkan melalui skrin Pilihan API.
API boleh dilayari dengan membukakan http://zap/ di pelayar anda semasa menggunakan proksi melalui ZAP.

Integrasi Beanshell

The BeanShell merupakan suatu Java shell interaktif yang boleh digunakan untuk melaksanakan skrip BeanShell.
Skrip ini merupakan suatu bentuk ringkas bagi Java yang menggunakan banyak unsur daripada sintaks Java, tetapi dalam format penskripan yang lebih mudah.
Semua kod Java juga merupakan kod BeanShell yang sah.
Integrasi BeanShell dalam OWASP ZAP membolehkan anda untuk menulis skrip menggunakan fungsi ZAP dan set data.
Ini boleh menjadi suatu ciri yang sangat berkuasa untuk menganalisis aplikasi web.

Pengantarabangsaan Penuh

Semua rentetan paparan telah sekarang diantarabangsakan sepenuhnya.

Persetempatan

Perubahan kecil:

Paparan Heks

The Tab Permintaan, Maklum Balas sekarang menyediakan suatu pilihan 'Paparan Heks' yang akan memaparkan tubuh badan permintaan dan maklum balas dalam format heks.

Hasil Carian

The Tab Carian sekarang memaparkan semua unsur sesuatu rentetan yang ditemui dan bukannya hanya unsur pertama dalam setiap permintaan atau maklum balas.

URL Dikecualikan

URL boleh dikecualikan secara eksplisit daripada pengimbas aktif, lelabah dan daripada proksi.

Sokongan menyalin

Kebanyakan panel sekarang menyediakan suatu pilihan menu 'klik kanan' 'Menyalin', termasuklah panel Port Scan dan Brute Force.

Sokongan Buat Asal/Buat Semula

Semua ruangan input sekarang menyokong tindakan Buat Asal/Buat Semula menggunakan pemecut lalai Buat Asal/Buat Semula sistem operasi:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Sokongan proksi pengimbas port dan pilihan masa tamat

Pengimbas Port sekarang boleh menggunakan proksi keluar (jika dikonfigurasikan) dan masa tamat dalam milisaat juga boleh ditetapkan.

Bebutang putus permintaan dan maklum balas

Sekarang terdapat 2 bebutang 'Menetapkan Putus' untuk membenarkan putus ditetapkan pada semua permintaan dan semua maklum balas secara bebas.

Bebutang memperluaskan tab Laman dan Maklumat

Sekarang terdapat 2 bebutang yang membolehkan anda untuk menukar di antara tab Laman dan Maklumat yang diperluaskan.

Break tab icon changes colour when breakpoint hit

Jika tab Putus tidak digunakan ikonnya ialah suatu pangkah kelabu:    .
Apabila satu breakpoint ditekan ikon tab akan bertukar kepada suatu pangkah merah:    .

Masa tamat yang boleh dilaras

Skrin Pilihan: Sambungan membolehkan anda untuk menetapkan masa tamat dalam saat untuk menjadikannya lebih senang untuk menguji aplikasi yang lambat.

Kemas kini perpustakaan

Kebanyakan perpustakaan yang digunakan oleh ZAP telah dikemas kini ke versi yang terkini.

Isu-isu yang diketahui:

Mac OS C: SSL Dinamik dan Google Chrome

Pada masa ini SSL dinamik tidak berfungsi apabila menggunakan Google Chrome. Ini kerana suatu isu dikenalikan yang tidak dapat diselesaikan dengan rantai kunci Google Chrome dan Mac OS X. Semasa mengimportkan Root CA OWASP ZAP ke dalam rantai kunci dan memintakan suatu laman web SSL, suatu mesej ralat "Sijil yang tidak sah" akan ditunjukkan.
