Keluaran 1.0.0

Perubahan berikut telah dibuat dalam keluaran ini:

Perubahan ketara:

Skrin bantuan

Membantu skrin telah ditambah untuk semua fungsi yang kini disediakan oleh ZAP.
Bantuan Context sensitif juga telah dilaksanakan supaya apabila anda menekan kekunci F1 kemudian skrin bantuan berkaitan dengan tab yang dipilih akan dipaparkan.

Breakpoints

Pelaksanaan Breakpoints telah berubah untuk menjadi lebih dekat kepada yang disediakan oleh perisian bersepadu pembangunan persekitaran.
The breakpoint controls have been moved to the new Toolbar peringkat tertinggi.
Breakpoints can now also be set by right clicking nodes in the Laman web dan Tab sejarah.

Menu Laporan

Dalam Menu Laporan telah ketara berubah dan kini termasuk perkara menu berikut:
    Menjana laporan...Yang kini menjana laporan mengikut permintaan
    Eksport mesej kepada fail...Berpindah dari Menu fail
    Eksport sebagai tindak balas kepada fail...Berpindah dari Menu fail
    Eksport mesej kepada fail...Fungsi baru
    Bandingkan dengan satu lagi sesi...Fungsi baru

Mencari sejarah yang lebih baik

Baru Cari membolehkan anda untuk mencari ungkapan-ungkapan yang biasa dalam semua URL, permintaan dan respons.
Butang depan dan sebelumnya membolehkan anda melihat istilah carian ditemui di dalam Permintaan dan Tab jawapan.

Sejarah untuk

Bar alat penapis sejarah telah ditambah yang membolehkan anda untuk menyekat permintaan yang dipaparkan.
Permintaan boleh ditapis berdasarkan kaedah HTTP, kod hasil HTTP dan mana-mana label, amaran, atau nota yang telah digunakan.

Nota

Anda sekarang boleh kaitkan nota dengan apa-apa permintaan.

Pelbagai Leds

Anda sekarang boleh kaitkan label dengan apa-apa permintaan.

Lebih banyak kawalan ke atas peringatan

Dalam Menambah dialog amaran membolehkan anda menambah atau mengubah secara manual amaran.

Baru pengekod BinHex

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Pengimbasan pasif

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Perubahan kecil:

Toolbar peringkat tertinggi

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Ringkasan footer berjaga-jaga

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Baru melihat dan merasa

A new look and feel 'Nimbus' is used.

Pengendalian kata laluan proksi

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Hantar dari tab laman

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Direktori pengguna dikekalkan di antara sesi

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
