Manipulator-in-the-middle Proxy

ZAP is a Manipulator-in-the-middle Proxy. Permite ver todas las peticiones que realice en una aplicación web y todas las respuestas que recibe de él.
Entre otras cosas esto le permite ver las llamadas AJAX que no otra cosa pueden ser obvias.

You can also set breakpoints which allow you to change the requests and responses on the fly.
