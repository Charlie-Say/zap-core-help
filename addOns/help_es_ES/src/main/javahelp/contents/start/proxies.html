Configurando proxies

You will need to configure your browser to use ZAP as a proxy.
By default, ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options > Network > Local Servers/Proxies screen. Las instrucciones para las últimas versiones de los navegadores más utilizados son:

Chrome (en Windows)

    Pulse el botón ( arriba a la derecha ) 'Personalizar y controlar Google Chrome'
    Seleccione el elemento de menú 'Opciones...'
    Pulse el botón 'Cambiar la configuración del proxy'
    Pulse el botón 'Configuración de LAN'
    Siga la sección 'Configuración LAN de Windows'
Note: To proxy localhost (and related addresses) with newer Chrome versions (>= 72) the command line argument --proxy-bypass-list=<-loopback> must be provided.

Firefox (en Windows)

    Seleccione el menú 'Herramientas'
    Seleccione el elemento de menú 'Opciones...'
    Select the 'General' panel
    Scroll to 'Network Settings' section
    Pulse el botón 'Configuración...'
    Seleccione la opción 'Configuración manual del proxy'
    Ingrese 'HTTP Proxy:' field the 'Address' you configured in the options screen
    Ingrese en la 'Port' field to the right of the 'HTTP Proxy' field the 'Port' you configured in the options screen. Ensure 'SSL Proxy' is also configured, either by selecting 'Use this proxy server for all protocols' or by setting the corresponding values.
    Pulse el botón 'OK'
    Pulse el botón 'OK'
Note: To proxy localhost (and related addresses) with newer Firefox versions (>= 67) the preference network.proxy.allow_hijacking_localhost (accessible through the about:config page) must be set to true.

Firefox (en Linux)

    SeleccioneMenú de 'Editar'
    SeleccioneObjeto del menu de 'Preferencias'
    Sigue lasinstructions above from the 'General' panel

Firefox (en OS X)

    Seleccione Menú 'Firefox'
    Seleccione el elemento de menú "Preferencias..."
    Follow the instructions above from the 'General' panel

Internet Explorer 7 y 8

    Seleccione Menú de 'Herramientas'
    Seleccione el elemento de menú "Opciones de Internet"
    Seleccione la pestaña "Conexiones"
    Pulse el botón "Configuración de LAN"
    Siga la sección "Configuración LAN de Windows"

Opera

    Seleccione la "Menú" en la esquina superior izquierda
    Seleccione Elemento de menú "Configuración"
    Seleccione Elemento de menú "preferencias"
    Seleccione el Pestaña "avanzado"
    Seleccione el Elemento "red" en la lista de la izquierda
    Presione el Botón de "Servidores proxy"
    Seleccione el 'HTTP' checkbox
    Ingrese en el field to the right of 'HTTP' the 'Address' you configured in the options screen
    Ingrese en el HTTP 'Port' field the 'Port' you configured in the options screen
    Press the Proxy servers 'OK' button
    Press the Preferences 'OK' button

Safari (on OS X)

    Pulse 'Safari settings' button in the top right hand corner
    Seleccione el elemento de menú 'Preferencias'
    Seleccione la pestaña 'Avanzado'
    Pulse el botón 'Cambiar configuraciones' a continuación de la etiqueta 'Proxies'
    Follow the 'OS X System Proxy settings' section below

Safari (on Windows)

    Press the 'Display a menu of general Safari settings' button in the top right hand corner
    Select the elemento de menú 'preferencias'
    Seleccione la pestaña 'avanzado'
    Pulse botón de 'cambiar configuraciónes' al lado de la etiqueta 'proxis'
    Pulse Botón "Configuración LAN"
    Sigua la sesión "configuración de Windows" más abajo

Configuración de Proxy del sistema OS X

    Seleccione 'Proxy web (HTTP)
    Ingrese 'Address:' field the 'Address' you configured in the options screen
    Ingrese 'Port' field the 'Port' you configured in the options screen
    Pulse Botón 'OK' de la configuración de servidores proxy
    Pulse Botón 'Aplicar' en opciones de red

Configuración LAN de Windows

    Seleccione la casilla de 'Usar un servidor Proxy para su servidor LAN'
    Ingrese en el 'Address:' field the 'Address' you configured in the options screen
    Enter in the 'Port' field the 'Port' you configured in the options screen
    Presione el Botón 'OK' de la configuración de Red de Área Local (LAN)
    Presione el Botón 'OK' de las Opciones de Internet
