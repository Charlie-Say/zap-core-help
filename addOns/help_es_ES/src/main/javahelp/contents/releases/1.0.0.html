Release 1.0.0

Se hicieron los siguientes cambios en esta versión:

Cambios significativos:

Pantallas de ayuda

Ayuda se han agregado pantallas para todas las funciones proporcionadas por ZAP.
Ayuda sensible al contexto también se ha implementado para que al presionar la tecla F1 se muestre la pantalla de ayuda de la tab seleccionada.

Breakpoints

La implementacion de Breakpoints se ha cambiado para estar más cerca a aquella proporcionada por entornos integrados de desarrollo de software.
The breakpoint controls have been moved to the new La barra de herramientas de nivel superior.
Breakpoints can now also be set by right clicking nodes in the Sitios y en la Pestaña Historia.

Menú de reporte

Se mostrará la Report Menu ha cambiado significativamente y ahora incluye los siguientes elementos del menú:
    Generar informe XML...Que genera informes en el demanda
    Exportar mensajes a fichero...Desde el menú Archivo
    Exportar respuesta a fichero...Desde el menú Archivo
    Exportar Todas las URLs a un fichero...Nueva funcionalidad
    Comparar con otra Sesión...Nueva funcionalidad

Búsqueda de la mejor historia

El nuevo Buscar en pestañas le permite buscar expresiones regulares en todas las URLs, las solicitudes y respuestas.
Botones anteriores y siguiente te permiten ver los términos de búsqueda encontrado en la Petición y en la Response tabs.

Filtro de historial

La barra de herramientas filtro de historia ha añadido que le permite restringir las solicitudes de que se muestran.
Las solicitudes pueden ser filtradas en base a los métodos HTTP, los códigos de resultado HTTP y cualquier etiquetas, alertas, o notas se han aplicado.

Notas

You can now associate notas con cualquier petición.

Múltiples etiquetas

Ahora puede asociar múltiples etiquetas con cualquier petición.

Más control sobre las alertas

Se mostrará la Añadir Alerta, permite manualmente agregar o cambiar una alerta.

Nuevo codificador/decodificador

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Escaneo pasivo

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Cambios menores:

La barra de herramientas de nivel superior

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Resúmenes de pie alerta

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Ver y sentir

A new look and feel 'Nimbus' is used.

Manejo de contraseña de proxy

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Enviar la ficha de sitios

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Directorio de usuario entre sesiones

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
