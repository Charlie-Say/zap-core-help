Menú Top Level Menu

Por defecto, los siguientes elementos de menú top level estarán presentes:
     File menu Esto maneja la sesión actual.
     Edit menu This handles finding strings in specific tabs, searching for strings across all requests and responses, changing ZAP's mode, and managing forced user mode
     View menu Esto maneja las opciones de visualización.
     Analyse menu Esto maneja las políticas de escaneo.
     Report menu Esto maneja los reportes.
     Tools menu Esto maneja las herramientas y las opciones generales.
     Import menu This gives access to menus that allow to import data into ZAP.
     Online menu This gives access to a set of online resources
     Help menu This gives access to the 'about' dialog and this help file

Note que los complementos puede agregar elementos de menú adicionales.
