Menú de Reportes

Este menú handles los informes.

Generar Reporte HTML

Esto genera un informe HTML que contiene las alertas levantadas.

Generar Reporte XML

Esto genera un nuevo informe XML que contiene las alertas levantadas.

Generar Reporte Markdown

Esto genera un nuevo informe Markdown que contiene las alertas levantadas.

Exportar mensajes a un archivo

Esto le permite guardar las solicitudes y respuestas en un archivo de texto.
Seleccione los mensajes de guardar en la ficha History. Utilice la tecla Mayús para seleccionar varios mensajes.

Exportar respuesta al archivo

Esto le permite guardar una respuesta específica en un archivo.
Seleccionar el mensaje relevante en la pestaña de History- tenga en cuenta que las respuestas binarias (como imágenes) pueden guardarse como prueba respuestas.

Exportar todas las direcciones URL a un archivo

Esto le permite guardar todas las URLs de acceso a un texto o archivo HTML.
Esto puede usarse, entre otras cosas, comparar las URLs disponibles para usuarios con diferentes roles o permisos en el mismo sistema.

Exportar las direcciones URL Seleccionadas a un archivo

Según la selección (incluida la selección múltiple) en el Sites tree se exportan todas las URL y las URL secundarias de los nodos seleccionados.

Exportar URL para el contexto

Se exportan todas las URLs en el Sites tree dentro del contexto seleccionado. Esta funcionalidad también está disponible desde el menú contextual cuando se utiliza en un nodo de contexto en el panel del Sites tree.

Comparar con otra sesión

Esto le solicita una sesión de ZAP que haya guardado previamente
A continuación, le solicita un archivo de salida en el que se escriben todas las URLs a las que se accedió por la sesión actual y la sesión que ha seleccionado para compararla.
El archivo contendrá una tabla que enumera las URLs y las respuestas HTTP para las URLs en las 2 sesiones
Los botones de JavaScript le permiten mostrar todas las URLs, solo las que se accedieron en la primera sesión, la segunda sesión y las accedidas por ambas sesiones.
Esto es particularmente útil para comparar 2 sesiones que acceden a la misma aplicación usando diferentes usuarios. Podrá ver qué URLs son visibles para los usuarios y podrá intentar acceder a todas las URLs cuando inicie sesión como cualquiera de los usuarios.

Note that add-ons can add additional menu items.
