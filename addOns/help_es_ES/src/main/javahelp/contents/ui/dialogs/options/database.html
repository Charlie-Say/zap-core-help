Opciones de la Pantalla de Base de Datos

Esta pantalla te permite configurar las opciones de la base de datos:

Compactar (al salir)

Te permite compactar la base de datos cuando ZAP se esté cerrando; compactar la base de datos te asegura un uso mínimo de espacio en disco, pero también ZAP tardará más en cerrar.

Registro de recuperación

Controla si el registro de recuperación de la base de datos se encuentra habilitado o no.
Mejora el rendimiento de la base de datos si se deshabilita, pero se pueden perder datos si ZAP se cierra de manera inesperada.
Nota: los cambios no afectarán a la sesión actual, estos se aplicarán en las sesiones nuevas.

Tamaño Máximo de Solicitud para Body

El tamaño de body para la solicitud máxima en bytes que permitirá ZAP.

Tamaño Máximo de Respuesta para Body

El tamaño máximo de la respuesta para el campo body en bytes que permitirá ZAP.

Preguntar por opciones de persistencia en nueva sesión

Sí, está marcado, el Cuadro de Diálogo de Sesión Persistente se mostrará al crear una sesión nueva.

Opciónes por defecto

La opción predeterminada para nuevas sesiones: si el cuadro anterior está marcado, se seleccionarán las opciones predeterminadas.
Si no está marcado, esta será la acción usada.
