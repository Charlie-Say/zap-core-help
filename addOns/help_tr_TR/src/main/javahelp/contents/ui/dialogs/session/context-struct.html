Oturum İçerik Yapısı Ekranı

Bu şunlardan biridir Oturum İçerik Ekranları ZAP'ın uygulama yapı bilgisini yönetmenizi sağlayan

URL anahtar değeri çifti ayırıcıları

URL'lerin içindeki çiftli anahtar ve değerleri ayıran karakterler, standart olarak '&' dur.

URL Anahtar değeri ayırıcıları

URL'lerin içindeki çiftli anahtar ve değerleri ayıran karakterler, standart olarak '=' dur.

POST Anahtar değer çifti ayırıcıları

POST veri içindeki çiftli anahtar ve değerleri ayıran karakterler, standart olarak '&' dur.

POST Anahtar değer ayırıcıları

POST veri içindeki çiftli anahtar ve değerleri ayıran karakterler, standart olarak '=' dur.

Yapısal Düzenleyiciler

, Yapısal Düzenleyiciler bu içeriğe uygulanacak olan şu olabilir:
  • Data Driven Content which identify URL paths that represent data
  • Structural Parameters which identify parameters that represent application structure instead of user data
