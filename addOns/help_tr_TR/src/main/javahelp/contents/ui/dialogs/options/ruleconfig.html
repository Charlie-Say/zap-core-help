Yapılandırma Kural Seçenekleri ekranı

Bu ekran belirli bir aktif ve pasif tarama kurallarının davranışlarını yapılandırmanıza olanak sağlar.

Herhangi bir belirlenen yapılandırma ayrıntılarını görmek ve ilişkili değeri değiştirmek için listelenen satırlardan birini seçin. Bireysel ve tüm kuralları sıfırlayabilirsiniz.

Yerleşik kurallar şunları içerir:

İsimVarsayılanAçıklama
rules.common.sleep15Zamanlama saldırıları için kullanılan zamanın saniye cinsinden süresi.
rules.cookie.ignorelistÇerez isimlerinin virgülle ayrılmış bir listesi. Çerezlerle ilgili sorunlarda tarama yapılırken bu listeye dahil olan çerezler dikkate alınmayacak.
rules.csrf.ignorelistTanımlayıcıların virgülle ayrılmış listesi. Kayıp Anti-CSRF belirteçleri taranırken, bu tanımlayıcılardan biriyle eşleşen isim veya kimliğe sahip herhangi bir FORM yoksayılır. Güvenli olduğunu bildiğiniz FORM'ları yok saymak için yalnızca bu özelliği kullanın, örnek olarak arama formları.
rules.csrf.ignore.attnameThe name of an HTML attribute that can be used to indicate that a form does not need an anti-CSRF Token. If rules.csrf.ignore.attvalue is specified then this must also match the attribute's value. If found any related alerts will be raised at INFO level.
rules.csrf.ignore.attvalueThe value of an HTML attribute named by rules.csrf.ignore.attname that can be used to indicate that a form does not need an anti-CSRF Token. If found any related alerts will be raised at INFO level.
rules.domains.trustedA comma separated list of URL regex patterns. Any URLs that match the patterns will be considered trusted domains and the issues ignored.
