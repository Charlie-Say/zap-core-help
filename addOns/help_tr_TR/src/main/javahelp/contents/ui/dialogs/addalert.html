Uyarı İletişimi Ekle

This dialog allows you to manually add or change an Alert associated with a specific request.

Alanlar

Diyalogda aşağıdaki alanlar bulunur:

Tür

Uyarı türü, önceden gönderilmiş bir sorun türünden birini seçmenize izin veren bir alandır.
Ayrıca metininizi girebilir veya seçtiğiniz öğelerden birinin metnini değiştirip seçebilirsiniz.
Açıklamadan sonraki mevcut biçimlerin birini seçerseniz, çözüm ve diğer bilgi alanları seçtiğiniz madde ile ilişkili metin ile doldurulur.

Risk

Risk almanın ne kadar ciddi olduğunu belirtmenizi sağlayan aşağı çekme alanı:
    Bilgilendirici
    Düşük
    Orta
    Yüksek

Güven

Bulgunun geçerliliğine ne kadar emin olduğunuzu belirlemenize olanak sağlayan bir çekme alanı:
    Hatalı Pozitifdaha sonra keşfedeceğiniz olası sorunlar istismar edilemez olduğu için
    Düşükdoğrulanmamış hatalar için
    Ortabiraz emin olduğunuz konular için
    Yüksekson derece güvendiğiniz bulgular için
    Onaylıdoğrulanmış hatalar için

Parametre

Sorunun hangi parametreyle ilişkili olduğunu belirlemenize olanak sağlayan bir açılır menü.
Alan, rasrgele URL ve FORM parametreleri ile önceden doldurulmuştur, ancak kendi parametre adınızı da girebilirsiniz.
Array parametreleri ( URL sorgu bileşenindeki ve x-www-form-urlencoded sorgu gövdesinde) indeksi ile tanımlı. Örneğin, bir isteği içeren için choices[]=ChoiceA&choices[]=ChoiceB ilk parametre belirlendiği gibi tanımlanır ki şekli choices[0] ve ikincisi olarak ise choices[1].

Attack

The payload that was supplied to the Parameter. This is expected to be set only by active scan rules which are the ones that send payloads/attacks.

Evidence

The available information in the message request or response (header or body) that indicates that the issue is present.

CWE ID

The relevant Common Weakness Enumeration (CWE) ID associated with the flagged vulnerability. This field can only be an integer.

WASC ID

The relevant Web Application Security Consortium (WASC) ID associated with the flagged vulnerability. This field can only be an integer.

Description

A general description of the type of issue found.
Bu önceden tanımlı modellerden birini seçtiğinizde doldurulur, fakat gerekli şekilde değiştirebilirsiniz.
Not et ki, bir başka tip seçerseniz yaptığınız seçimler kaybolacaktır.

Other Info

Information specific to the particular issue you have found.
This is not prepopulated.

Solution

Recommendations about how to fix the issue.
This is populated when you select one of the predefined types, but you can also change it as required.
Note that any changes you make will be lost if you select another type.

Reference

One or more URLs pointing to more information on the internet about the selected type of alert.
This is populated when you select one of the predefined types, but you can also change it as required.
Note that any changes you make will be lost if you select another type.

Alert Tags

Alerts can be tagged with arbitrary keys or key=value pairs.
You can add as many tags as you want to each alert. However,
  • You cannot have two tags with the same key.
  • The key must be limited to 1024 characters and the value must be limited to 4000 characters for each tag.
