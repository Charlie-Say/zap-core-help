Geçmiş Filtresi diyaloğu

Bu iletişim kutusu, Geçmiş sekmesinde hangi isteklerin görüntüleneceğini kısıtlamanıza izin verir.

Alanlar

Diyalag aşağıdaki alanlara sahiptir:

Yöntemler

Filtrelenecek HTTP yöntemleri.

Kodlar

Filtrelenecek HTTP yanıt kodları.

Etiketler

O Etiketler filtrelemek için.

Uyarılar

Türleri Uyarılar filtrelemek için.

Notlar

Varlık veya yokluk Notlar üzerinden filtrelenmelidir.

URL inc regex

Evet 'regex kalıplarını dahil et' listesi (her satıra bir tane) - belirtilirse, yalnızca o desenlerle eşleşen Url'ler eklenir.

URL exc regex

'Hariç Tut' regex kalıplarının (her satıra bir tane) bir listesi - belirtilirse, bu kalıplarla eşleşen herhangi bir Url gizli olur.
