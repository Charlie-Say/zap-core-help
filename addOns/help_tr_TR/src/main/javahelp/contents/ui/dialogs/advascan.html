Aktif Tarama iletişim kutusu

Bu iletişim kutusu aşağıdakileri başlatır aktif tarayıcı.

Kapsam

İlk sekme, başlangıç noktasını seçmenize veya değiştirmenize izin verir.
Eğer daha fazlasına sahipseniz tarama ilkeleri o zaman kullanılacak olanı seçebileceksiniz.
Eğer başlangıç noktası bir veya birden fazla ise İçerikler o zaman içlerinden birini seçebilirsiniz.
Eğer bağlam hiçbirini içermiyorsa Kullanıcılar tanımladığınızda bunlardan birini seçebilirsiniz.
Kullanıcılardan birini seçerseniz, o zaman aktif tarama o kullanıcı olarak gerçekleştirilir ve ZAP (tekrar) bu kullanıcı olarak gerektiğinde kimlik doğrulaması yapar.

If you select 'Recurse' then all of the nodes underneath the one selected will also be scanned.
Özel giriş vektörleri yalnızca bu seçenek seçili değilse desteklenir.

If you select 'Show Advanced Options' then the following tabs will be shown which provide fine grain control over the active scanning process.

'Sıfırla' düğmesine tıklamak, tüm seçenekleri varsayılana sıfırlar.

Giriş Vektörleri

Giriş Vektörleri sekmesi, varsayılan giriş vektörlerini geçersiz kılmanıza izin verir Seçenekler Aktif Tarama Giriş Vektörleri ekranı.
'Sıfırla' düğmesini tıklamak, giriş vektörlerini varsayılan seçeneklere sıfırlar.

Özel Vektörler

Özel Vektörler sekmesi saldırı talebine karşı belirli bölgeleri belirtmenize olanak tanır.
Custom Vectors are only available if the 'Recurse' option on the first tab is not selected.
Özel giriş vektörleri eklemek için, isteğinize saldırmak istediğiniz karakterleri vurgulayın ve 'Ekle' düğmesini tıklayın.
İstediğiniz kadar çok sayıda özel giriş vektörü ekleyebilirsiniz.
Özel giriş vektörlerini kaldırmak için seçili karakterlerden herhangi birini vurgulayın ve 'Kaldır' düğmesini tıklayın.
'Özel giriş vektörlerini devre dışı bırak' kutusunu işaretleyerek, bu sekmede manuel olarak tanımladığınız giriş vektörleri hariç tüm giriş vektörleri devre dışı bırakırsınız.

Teknoloji

Teknoloji sekmesi hangi teknolojilerin taranacağını belirlemenizi sağlar.
Hedef uygulamada bulunmayan tanıdığınız teknolojileri seçmemek, taramayı, o teknolojinin bu testleri atlayabileceğini hedefleyen kurallar olarak hızlandırabilir.

İlke

İlke sekmesi seçilen ayarlarda belirtilen ayarların tümünü geçersiz kılmanızı sağlar tarama ilkesi.

Filter

The Filter tab allows you to specify filter criteria and active scan will filter requests based on these criteria's.
There are four filter criteria's available in the filter tab
  • HTTP method
  • Status code
  • Tags
  • URL pattern
HTTP method and Status code filter criteria are only applied to check if a request can be included in active scan or not,
where as Tags and URL pattern filter criteria can be applied to include or exclude a request.
