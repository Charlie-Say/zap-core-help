Spider diyaloğu

Bu iletişim kutusu şunu başlatır spider.

Kapsam

İlk sekme, başlangıç noktasını seçmenize veya değiştirmenize izin verir.
Eğer başlangıç noktası bir veya birden fazla ise Bağlamlar sonra onlardan birini seçebilirsiniz.
Eğer bu bağlam herhangi bir Kullanıcılar tanımladığınızda bunlardan birini seçebilirsiniz.
Eğer kullanıcılardan birini seçerseniz, örümcek bu kullanıcı olarak görülecek ve gerektiğinde ZAP (tekrar) bu kullanıcı olarak kimlik doğrulaması yapacaktır.

If you select 'Recurse' then all of the nodes underneath the one selected will also be used to seed the spider.

'Yalnızca Örümcek Alt Ağacı' nı seçerseniz örümcek, başlangıç noktasının (URI) altındaki kaynaklara erişir. Belirtilen alt ağacın içinde bir kaynak bulunup bulunmadığını değerlendirirken örümcek, URI'nin yalnızca şema, ana makine, port ve yol bileşenlerini dikkate alır.

If you select 'Show Advanced Options' then the following tab will be shown which provide fine grain control over the spider process.

'Sıfırla' düğmesine tıklamak, tüm seçenekleri varsayılan değerlerine sıfırlar.

Gelişmiş

Bu sekmedeki parametreler, aynı sekme üzerindeki aynı parametrelere karşılık gelir Örümcek Ekran Seçenekleri.

Üzerinden erişildi

    Spider sekmesi'Yeni Tarama' butonu
    Siteler sekmesi'Atak / Ağ ...' sağ tık menü öğesi
    Geçmiş sekmesi'Atak / Ağ ...' sağ tık menü öğesi
