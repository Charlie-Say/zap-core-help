Seçenekler HTTP Oturumları Ekranı

This screen allows you to configure the HTTP Sessions Extension's options:

Varsayılan Simgeleri

Bunlar, varsayılan çerezler olarak kabul edilen tanımlama isimleridir. En kısa sürede Sitedeki parametreler algılanırlarsa, kullanıcı komutuyla el ile geçersiz kılınmadıkça otomatik olarak o Site için oturum simgesi olarak eklenir.

Oturum belirteçleri büyük / küçük harfe duyarlı değildir, bu nedenle varsayılan oturum belirteçleri otomatik olarak daha düşük düzeydedir. Ayrıca, varsayılan oturum belirteçlerinde yapılan değişiklikler sadece Site sayfalarının tekrar ziyaret edilmesinden sonra görülebilir.

Sadece Proxy için etkin

Eğer bu seçenek aktifse, uzantı sadece Proxy ile gönderilen ve alınan iletileri işler(ör. tarayıcıdan/tarayıcıya). Diğer uzantılardan gelen mesajlar yalnızca bu seçenek devre dışı kaldığında işlenir.
