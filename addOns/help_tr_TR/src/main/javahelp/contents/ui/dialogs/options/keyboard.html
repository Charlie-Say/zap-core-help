Klavye Seçenekleri Ekranı

Bu ekran size bütün ZAP menüleri için klavye kısayolları yapılandırma imkanı verir.

Tablodaki satırlardan herhangi birine tıklamak klavye kısayollarını değiştirebileceğiniz bir diyalog kutusu ortaya çıkartacaktır.

'Eylem' Kopya Kağıdı

Bu butona tıklamak, tüm 'eyleme' göre sıralanmış klavye kısayollarını gösteren bir HTML sayfası oluşturacaktır.
Bu düğme, yalnızca API etkinleştirildiyse ve işletim sisteminiz URL'leri bir tarayıcıda açmayı destekliyorsa etkin hale gelir.

'Anahtar' Kopya Kağıdı

Bu butona tıklamak, tüm 'anahtara' göre sıralanmış klavye kısayollarını gösteren bir HTML sayfası oluşturacaktır.
Bu düğme, yalnızca API etkinleştirildiyse ve işletim sisteminiz URL'leri bir tarayıcıda açmayı destekliyorsa etkinleştirilir.

Sıfırla

Bu düğmeye tıklamak tüm kısayolları var olan ayarlarına sıfırlayacaktır.
