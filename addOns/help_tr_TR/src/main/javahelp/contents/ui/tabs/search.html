Arama sekmesi

Arama sekmesi, URL'lerin, isteklerin, yanıtların, başlıkların ve eklentiler tarafından sağlanan diğer işlevlerin tümünün içinde düzenli ifadeleri aramanıza olanak verir.

Aramak istediğiniz düzenli ifadeyi arama kutusuna girin ve geri dön'e basın veya ara butonuna tıklayın:

Aşağı doğru çekme, URL'lerde, isteklerde, yanıtlarda veya her şeyde arama yapmayı seçmenize izin verir.

Arama modelini içeren tüm URL'ler, istekler veya yanıtlar sekmesinde görüntülenecektir. <0 /> Arama teriminize. * Eklemeniz yararlı olabilir - bu size daha fazla bağlam verecektir; parola.*

Bulunan arama terimlerini İstekler ve Yanıt sekmeleri'nde görüntülemek içinSonraki ve <1/>Önceki düğmelerini kullanabilirsiniz.

Sonuçlar listesinde uygun satıra tıklayarak doğrudan özel bir örneğe gidebilirsiniz.

Sağ tık menüsü

Bir node'a sağ tıklamak size şunları yapmanıza izin veren bir menü getirecek:

Atak

Saldırı menüsünde aşağıdaki alt menüler bulunur:

Aktif Tarama...

Bu şunu başlatacak Aktif tarama penceresi şunu başlatmanıza imkan sağlar aktif tarama seçtiğiniz sorgulamaya başlangıç noktası ayarlamak ile.

Ağ...

Bu şunu başlatacak Örümcek diyaloğu başlatmanıza olanak sağlar örümcek seçmiş olduğunuz talebin başlangıç noktası ile.

İçeriğe dahil et

Bu menü, seçilen düğümleri ve ona ait belirtilen tüm bağlı bileşenleri dahil etmenize izin verir içerik.
Ayrıca, yeni bir içerik yaratma seçeneğine sahipsiniz.
O Oturum İçerikleri ek değişiklik yapmanıza izin vermek için iletişim kutusu görüntülenecektir.

İçeriğin dışında bırakma

Bu menü, belirtilen düğümleri ve bu düğümlerin tüm astlarını çıkartmak için izin verir içerik.
O Oturum İçerikleri ek değişiklik yapmanıza izin vermek için diyalog görüntülenecektir.

İçerik olarak işaretle

Bu menü takip eden diğer her bir alt menüyü bağlamlar tanımladığınız:

Bağlam ismi Form-temelli Doğrulama Oturum Açma talebi

Belirtilen bağlam için belirtilen düğümü bir oturum açma talebi olarak tanımlar.
Bir ortamda sadece bir düğüm tanımlayabilirsiniz.
The Oturum Bağlam Doğrulama ekranı daha fazla değişiklik yapabilmeniz için gösterilecektir.

Context name JSON-based Auth Login request

This identifies the specified node as a login request for the specified context.
You may only have one node identified as such in any one context.
The Session Context Authentication ekranı daha fazla değişiklik yapabilmeniz için gösterilecektir.

Context name Data driven node

This identifies the specified node as Data driven content for the specified context.
The Session Context Structure screen will be displayed to allow you to make any additional changes.

Şundan hariç tut

This menu has the following submenus:

Proxy

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
This can be used to ignore URLs that you know are not relevant to the system you are currently testing.
The nodes can be included again via the Oturum Özellikleri dialog

Scanner

This will prevent the selected nodes from being actively scanned.
The nodes can be included again via the Oturum Özellikleri dialog

Spider

This will prevent the selected nodes from being spidered.
The nodes can be included again via the Session Properties dialog

Open/Resend with Request Editor...

This will bring up the Manual Request Editor dialog which allows you to resend the request after making any changes to it that you want to.

Yeni Uyarı...

This will bring up the Add Alert dialog which allows you to manually record a new alert against this request.

Geçmiş Sekmesinde Göster

This will show the selected node in the History tab.

Siteler sekmesinde göster

This will show the selected message in the Sites tab.

Tarayıcıda URL aç

This will open the URL of the selected node in your default browser.
