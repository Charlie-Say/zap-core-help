Yanıt sekmesi

Cevap sekmesi, sahip olduğunuz isteğe göre tarayıcınıza gönderilen verileri gösterir Siteler sekmesi veya Geçmiş sekmesi .

Çekme işlemleri farklı seçim yapmanızı sağlar Görüntüleme yanıt başlığı ve gövdesi için.

Note that images are only shown in the Geçmiş sekmesi eğer 'Geçmişte Görüntü Etkinleştir' bayrağı içindeyse Görünüm menüsü seçildi.

Sağ tık menüsü

Ağın üzerine sağ tıklamak sizin şunları yapmanıza izin veren bir menü getirecek:

Bul...

Bu şunu öne çıkaracak Diyalog bul.

Copy

This will copy the selected string to the clipboard.
