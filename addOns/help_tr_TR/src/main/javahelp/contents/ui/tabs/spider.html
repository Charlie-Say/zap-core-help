Örümcek sekmesi

The Spider tab shows you the set of unique URIs found by the Spider during the scans.

'Yeni Tarama' butonu tam olarak taranması gereken şeyi belirtmenizi sağlayan Örümcek diyaloğu'unu açar.
Örümcek paralel olarak birden fazla Site üstünde çalıştırılabilir ve her taramanın sonucu 'İlerleme' çekmeli menüsü aracılığıyla gösterilir.

Araç çubuğu tarama hakkında bilgileri gösterir ve onu kontrol etmenizi sağlar. Bir takım butonlar bulundurur ve şunlara yararlar:

  • seçili örümcek taramasını Durdur (ve devam ettir);
  • Seçili örümcek taramasını durdur;
  • Tamamlanan taramaları temizle;
  • Open the Spider Options screen.
İlerleme göstergesi, seçilen örümcek taramasının ne kadar ilerlediğini gösterir. Ayrıca aktif örümcek taramalarının sayısını ve seçilen taramada bulunan URI sayısı gösterilir.

Bulunan her URL için şunu görebilirsiniz:

  • İşlendi - URL'nin Örümcek tarafından işlendiğini ya da bir kural nedeniyle (örneğin kapsama dışındaydı) getirilmesinin atlandığını
  • Metod - HTTP metodu, örneğin GET ya da POST, erişilmesi gereken kaynağın içinden
  • URI - kaynak bulundu
  • Bayraklar - URI hakkında bilgiler (örn. tohum olması ya da niye işleme alınmadığı)

For each spider message, shown under the Messages tab, you can see details of the request sent and response received. The İşlendi column, indicates whether:

  • Successfully - the response was successfully received and parsed
  • Empty Message - the response was not parsed because it was empty
  • I/O Error - an input/output error occurred while fetching the response
  • Max Children - the response was not parsed because the corresponding parent Sites node already has more child nodes than the maximum allowed
  • Max Depth - the response was not parsed because it passed the maximum depth allowed
  • Max Size - the response was not parsed because its size is not under the maximum allowed
  • Not Text - the response was not parsed because it's not text, for example, an image
  • Spider Stopped - the response was not fetched or parsed because the spider was already stopped
