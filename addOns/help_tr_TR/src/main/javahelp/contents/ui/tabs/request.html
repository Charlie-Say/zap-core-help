Talep sekmesi

Talep sekmesi, vurguladığınız talep için size tarayıcınız tarafından gönderilen veriyi gösterir. Siteler sekmesi ya da Geçmiş sekmesi .
Üst panel istek üstbilgisini, alt panel de gönderilen verileri gösterir, örneğin bir POST isteği sonucunda.

Çekme işlemleri farklı seçim yapmanızı sağlar Görünümler istek başlığı ve gövdesi için.

Sağ tuş menüsü

Bir node üzerinde sağ tıklamak, aşağıdakileri yapmanıza izin verecek bir menü getirir:

Bul...

Bu şunu açar Diyalog bul.

Copy

This will copy the selected string to the clipboard.
