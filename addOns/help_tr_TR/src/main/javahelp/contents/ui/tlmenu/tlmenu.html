Üst Düzey Menü

Varsayılan olarak aşağıdaki üst düzey menü öğeleri bulunur:
     Dosya menüsü Mevcut oturumu yönetir
     Düzenleme menüsü This handles finding strings in specific tabs, searching for strings across all requests and responses, changing ZAP's mode, and managing forced user mode
     Görünüm menüsü Görüntüleme seçeneklerini yönetir
     Analiz menüsü Tarama ilkelerini yönetir
     Rapor menüsü Raporları yönetir
     Araçlar menüsü Araçları ve genel seçenekleri yönetir
     Import menu This gives access to menus that allow to import data into ZAP.
     Online menu This gives access to a set of online resources
     Help menu This gives access to the 'about' dialog and this help file

Not olarak eklentiler fazladan menü ögesi ekleyebilir.
