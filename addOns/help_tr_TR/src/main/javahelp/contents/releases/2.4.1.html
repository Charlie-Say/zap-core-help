Sürüm 2.4.1

Bu sürüm önemli güvenlik düzeltmelerini içeriyor - kullanıcıların en kısa sürede güncellemesi gerekmektedir.

Değişikliklerden biri, varsayılan olarak bir API anahtarı oluşturulduğu anlamına gelir; ZAP API'sini kullanan tüm uygulamaların bu anahtarı uygulayacak şekilde güncelleştirilmediği sürece başarısız olacağı demektir.
API Anahtarı bulunabilir API seçenekleri ekranında
Ayrıca bir seçenek gibi kullanarak komut satırından ayarlayabilirsiniz: 

-config api.key=change-me-9203935709

Daha fazla ayrıntı için https://www.zaproxy.org/faq/why-is-an-api-key-required-by-default/
