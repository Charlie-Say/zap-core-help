Sürüm 2.5.0

Bu sürümde aşağıdaki değişiklikler yapıldı:

ZAP Uygulama Programlama Arayüzü değişiklikleri:

GetAuthorizationDetectionMethod / VIEW yetkilendirme

AuthorizationDetectionMethod adlı gizli dosyaları bir nesneye sarılmış verileri döndür: 
{"authorizationDetectionMethod":{"statusCode":"-1"..."headerRegex":""}}
yerine: 
{"statusCode":"-1"..."headerRegex":""}

VIEW context / context + technologyList

Gizli dosyaları şimdi dönüş verileri "içerik" adı verilen bir nesneye sarılmış olarak dönüştür: 
{"bağlantı":{"id":"1", ..., "inScope":"gerçek","loggedOutPattern":""}}
yerine: 
{"bağlantı":{"id":"1", ..., "inScope":"gerçek","loggedOutPattern":""}}

ACTION spider / scan + scanAsUser

İkisi de bağlantıyı belirtilen alt ağa sınırlayan yeni bir isteğe bağlı 'alt başlık' parametresini destekler. Geçerli bir 'içerik' parametresi sağlandığı sürece 'url' parametresi artık isteğe bağlıdır

Yeni 'istatistik' öğesi

API bileşeni içinde yeni 'istatistikler' istatistikler ZAP tarafından korunuyor.

Mevcut bileşenlere de yeni işlemler yapılacağını unutmayın API Web Kullanıcı Arayüzü daha fazla ayrıntı için.
