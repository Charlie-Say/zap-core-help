Sorun 153: fuzzdb birleştirme

Fuzzer, fuzzdb (https://github.com/fuzzdb-project/fuzzdb) fuzz dosyaları içeriyor.
Bazı fuzzdb dosyalarının virüsten koruma amaçlı ortak anti virüs tarayıcılarına neden oldukları için bırakıldığını unutmayın.
Bunları değiştirebilir ve fuzzdb'nin en son sürümünü indirip 'fuzzers' kitaplığında genişleterek fuzzdb'yi yükseltebilirsiniz.

Çapraz Site Senaryosu aktif tarayıcısı, muhtemel XSS sorunlarını bulmak ve daha az yanlış pozitif rapor vermek için baştan yeniden yazıldı.

Sorun 244: Veri kanalı pasif izleyiciyi kontrol edin

Aşağıdaki kontroller Watcher tarafından aktarıldı (buna bakmamıza olanak sağlayan Chris Weber'e teşekkürler):
    Pasif Kesişen Domain Komut Dizisi Belgesini Kontrol Edin .csalanlar arası javascript dosyalarının dahil edilmesini denetler.
    Pasif Üstbilgi Cache Control' ünü Kontrol Edin .csHTTP önbelleğini kontrol eder-SSL sayfalarındaki başlığı kontrol eder.
    Pasif Üstbilgi Eksik İçerik Türünü Kontrol Edin .cscontent-Type HTTP üstbilgisinin eksik olmadığını kontrol eder.
    Pasif Üstbilgi Çerçeve Seçeneklerini Kontrol Edin .csx-FRAME-OPSİYONLARININ eksik veya yetersiz olduğunu kontrol eder.
    Pasif Üstbilgi ieXss Korumasını Kontrol Edin .csx-XSS-Protection'ın IE'nin XSS korumasını devre dışı bırakacak şekilde ayarlanmadığını denetler.
    Pasif Üstbaşlık Mime Çözümleyicisini Kontrol Edin .csx-CONTENT-TYPE-OPTIONS'ın ayarlandığını kontrol eder.
    Pasif Bilgi Gizliliği Veri Tabanı Hatalarını Kontrol Edin .csveritabanındaki hata mesajlarını denetler.
    Pasif Bilgi Gizliliği Hata Ayıklama Sorununu Kontrol Edin .cshata mesajlarının ayıklanmasını denetler.
    URL' deki Pasif Bilgi Gizliliğini Kontrol Edin .csuRL parametrelerindeki bilginin ortaya çıkarılmasını denetler.
    Check.Pasv.InformationDisclosure.ReferrerLeak.csbilgiyi ortaya çıkarmak için HTTP Yönlendirici başlığını denetler.
