Sürüm 2.4.3

Bu sürümde aşağıda ki değişiklikler yapıldı:

Geliştirmeler:

  • Sorun 1576 : URL yolu öğelerini 'yapısal olmayan' olarak tanımla
  • Sorun 1711 : Özellik isteği: Oluşturulacak belirteç sayısını belirleme seçeneği.
  • Sorun 1768 : Daha yeni bir kullanıcı aracısı kullanmak için güncelle
  • Sorun 1894 : Zap yalnızca "User-Agent", "Referer" ve "Host" başlıklarını tarar
  • Sorun 1911 : AJAX örümcek isteklerini ara
  • Sorun 1913 : Tüm parametreleri zap.sh -debian paketine geçir
  • Sorun 1914 : Çoklu eklenti dizinleri
  • Sorun 1920 : Ana bilgisayarı bildir : ZAP bağlantı noktası hayalet modunda dinliyor, dinleyemiyorsa çıkış yapın
  • Sorun 1927 : Geliştirme: Sadece kapsamdaki URL'ler için kır.
  • Sorun 1944 : Grafik tarama yanıtları saniyede başına ilerliyor
  • Sorun 1953 : ZAP API'de doğrudan örümcek içeriğe izin ver
  • Sorun 1962 : Komut satırındaki eklentileri yüklemek ve güncellemek
  • Sorun 1975 : RC4-SHA SSL şifre paketi desteklenmiyor
  • Sorun 1980 : Docker görüntülerine ZAP CLI ekle
  • Sorun 1985 : Kullanıcı AbstractMultipleOptionsBaseTablePanel'da bir satıra çift tıklarsa düzenle penceresini getir
  • Sorun 2009 : FAQ(Sıkça Sorulan Sorular) ve bülten sayfalarına online bağlantılar ekle
  • Sorun 2084 : Eğer kullanıcılar tarihi geçmiş versiyonları kullanıyorlarsa, onları uyar
  • Sorun 2086 : Eklenti başına rapor isteklerini say
  • Sorun 2088: Bütün güncel eklentiler için 'hepsini güncelle' düğmesi desteği
  • Sorun 2094 : '-addoninstallall' komut satırı seçeneği destekle
  • Sorun 2102 : Ajax ağ seçeneklerinin API aracılığıyla ayarlanmasına izin verme

Düzeltilen hatalar:

  • Sorun 1070: Sorgu parametreleri ile özel URL'yi kırmak işe yaramıyor.
  • Sorun 1555: Rapor oluşturulurken SAXParseException
  • Sorun 1617 : ZAP 2.4.0 Monitörsüz makinelerde hayalet modunda çalışırken Monitörsüz Seçenekleri verir
  • Sorun 1877 : zap.sh içinde .ZAP_JVM.properties yolunu düzelt
  • Sorun 1893 : Regresyon: Belirli tarayıcıları API ile etkinleştirip devre dışı bırakamıyor
  • Sorun 1910 : API hatalarda beklenen şifreleme içeriğini döndürmüyor
  • Sorun1917: İstek ve Yanıt sekmeleri metni ölçeklendirmiyor
  • Sorun 1939 : Komut Dosyaları Kaydet Düğmesini Etkinleştirme Sorunu
  • Sorun 1949: Kayıp türü olan komut dosyası, komut dosyalarının ve şablonların yüklenmesini engelliyor
  • Sorun 1950 : Doğrulama skripti ile hataları bağlantı yanıtsız kapandı
  • Sorun 1951: Kimlik doğrulama komut dosyasını yüklemeden önce kullanıcı eklemeye çalışırken özel durum
  • Sorun 1960: Etkin Tarama diyalogu, güncel olmayan tarama politikasını kullanabilir
  • Sorun 1969 : Tarayıcıların kurulumuyla ilgili sorunlar
  • Sorun 1970 : Yüklü eklenti gereklikleri kullanıcı da eklentileri kurarken dikkate alınmayabilir
  • Sorun 1973 : İade edilmiş HAR/list doğru yönlendirme mesajları içermez
  • Sorun 1981 : Örümcek, URL'nin bulduğu doğru sayıları rapor etmeyebilir
  • Sorun 2005 : Aktif tarama, kardeş düğümlerde hatalı gerçekleştirildi
  • Sorun 2044 : Hirschberg's algoritması uygulamasındaki sorunlar
  • Sorun 2045 : Eğer -dir seçeneği kullanılmışsa eski yapılandırmaları kopyalama
  • Sorun 2052 : Doğrulama değişiklikleri doğrudan kaydedilmemiş API döneminde yapılır
