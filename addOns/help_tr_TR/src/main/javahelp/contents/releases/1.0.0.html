Sürüm 1.0.0

Bu sürümde aşağıdaki değişiklikler yapılmıştır:

Önemli değişiklikler:

Yardım ekranları

Yardım ekranları şimdi ZAP tarafından sağlanan işlevlerin tümü için eklendi.
Böylece F1 tuşuna ardından seçtiğiniz sekme ile ilişkili yardım ekranı bastığınızda görüntülenecek içerik duyarlı yardım de uygulamaya konmuştur.

Breakpoints

Uygulanması Breakpoints tarafından sağlanana daha yakın olacak şekilde değiştirildi. yazılım Entegre Geliştirme Ortamları.
The breakpoint controls have been moved to the new Üst Düzey Araç Çubuğu.
Breakpoints can now also be set by right clicking nodes in the Siteler ve Geçmiş sekmeleri.

Rapor menüsü

, Rapor menüsü önemli ölçüde değişti ve şimdi aşağıdaki menü öğeleri:
    Rapor oluştur...Şu anda talep üzerine raporlar üreten
    İletileri Dosyaya Ver ... Dosya Menüsünden taşındı
    Tepki Dosyaya Ver...Taşındı Dosya menüsü
    TÜM URL'leri Dosyaya Ver...Yeni işlevsellik
    Başka bir oturumla karşılaştırın...Yeni işlevsellik

Geliştirilmiş geçmiş araştırması

Yeni Arama sekmesi düzenli ifadeleri aramanıza izin verir tüm URL'lerde, isteklerde ve yanıtlarda.
Sonraki ve önceki düğmeler, bulunan arama terimlerini Talep ve Yanıt sekmeleri.

Geçmiş filtresi

Geçmiş Filtresi Araç Çubuğu eklemişsiniz ki hangi istekler görüntülenir.
İstekler, HTTP yöntemlerine, HTTP sonuç kodlarına, Ve herhangi biri etiketler, uyarılarveya notlar uygulanmış.

Notlar

Şimdi ilişkilendirebilirsiniz notlar herhangi bir istekle.

Birden Fazla Etiket

Artık birden çok kişiyi ilişkilendirebilirsiniz etiketler herhangi bir istekle.

Uyarılar üzerinde daha fazla kontrol

, Uyarı iletişim Ekle el ile eklemek veya bir uyarı değiştirmek sağlar.

Yeni kodlayıcı/çözücü

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Pasif tarayıcı

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Küçük değişiklikler:

Üst Düzey Araç Çubuğu

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Uyarı altbilgi özetleri

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Yeni görünüm

A new look and feel 'Nimbus' is used.

Proxy parolası işleme

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Siteleri sekmesinden yeniden gönderin

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Oturumlar arasında tutulan kullanıcı dizini

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
