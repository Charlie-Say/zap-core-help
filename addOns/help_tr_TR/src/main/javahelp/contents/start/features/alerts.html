Uyarılar

Uyarı, olası bir güvenlik açığı olup, belirli bir istekle ilişkilidir.
Bir isteğin birden fazla uyarısı olabilir.

Uyarılar UI'de riski belirten bir bayrakla gösterilir:
      Yüksek
      Orta
      Düşük
      Bilgilendirici
      Yalnış pozitif

Uyarılar, bunlarla sınırlı olmamakla beraber, çeşitli ZAP bileşenleri tarafından eklentiler (uzantılar) veya Add Alert dialog (uyarı ayrıntılarını / bilgileri güncellemenize yada değiştirmenize izin veren) aracılığıyla geliştirilebilir: active scanning, passive scanning,.

Uyarılar Geçmiş sekmesinde en yüksek tehlike uyarısını gösteren bir bayrak ile işaretlenir.
Uyarılar sekmesindeki tüm uyarılar, ve tehlikeye göre toplam uyarı sayısı alt bilgide gösterilir.

Uyarı geçersiz kılmaları

ZAP tarafından ortaya atılan uyarılarda, ortaya çıkan uyarılar hakkında genel ve özel bilgiler bulunur. Spesifik bilgiler, URL ve etkilenen parametre gibi doğrudan potansiyel sorunlarla ilgilidir. Genel bilgi, bir açıklama ve ilgili çevrimiçi kaynaklara bağlantılar gibi şeyleri içerir.

Bir 'uyarı geçersiz kılma' yapılandırma dosyası kullanarak jenerik bilgileri değiştirebilir veya ekleyebilirsiniz. Bu, zorunlu politikalar, dahili bağlantılar veya kullandığınız belirli teknolojilere ilişkin tavsiyeler gibi şirketinize özgü bilgileri eklemenize izin verir.

Bir uyarıyı geçersiz kılma yapılandırma dosyası, sadece değiştirmek istediğiniz bilgileri içeren bir UTF-8 özellik dosyasıdır. '#' ile başlayan satırlar yorum olarak kabul edilir ve göz ardı edilirler.

Format şöyledir: 

<alertid>.<property> = [ + | - ] <your information>
Aşağıdaki özellikler desteklenir:
  • isim
  • açıklama
  • çözüm
  • diğer bilgi
  • referans
The format for alert tags is: 
<alertid>.tag.<key> = [ + | - ] [ <value> ]
Note that:
  • Spaces in the tag key must be escaped with a backslash.
  • The tag value is optional.
For example 
# 40012 = Reflected XSS
40012.solution = Follow our company specific guidelines at http://internet.example.com/xss.html
40012.tag.Bug\ Tracker\ URL = https://example.com/issues/1234
If the value starts with a '+' then it is appended to the existing information.
If it starts with a '-' then it is prepended to the existing information.
If it does not start with a '+' or '-' then it replaces the existing information.

Uyarıyı geçersiz olarak ayarlama dosya yolu ile yapılabilir API, Seçenek Uyarı ekranı veya komut satırı seçenek: 

-config alert.overridesFilename=<filename> kullanın
