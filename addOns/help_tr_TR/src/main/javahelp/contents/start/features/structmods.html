Yapısal Değiştiriciler

Yapısal değiştiriciler, Zap uygulamasının yapısınının nasıl temsil edildiğini değiştiren kontrollerdir.
O Sites Tree is ZAP's representation of the application.
Yapının iyi bir temsili yoksa ZAP uygulamaya etkin bir biçimde saldırması mümkün değildir.
Şu anda yapısal 2 tip Yapısal Değiştiriciler vardır:

  • Veriyi temsil eden URL yollarını tanımlayan Veri Tabanlı İçerik
  • Kullanıcı verisi yerine uygulama yapısını temsil eden parametreleri tanımlayan Yapısal Parametreler
Yapısal Değiştiriciler yalnızca şuna uygulanır İçerikler.
