Manipulator-in-the-middle Proxy

ZAP is a Manipulator-in-the-middle Proxy. Bir web uygulamasına yaptığınız tüm istekleri görmenizi sağlar ve ondan aldığınız tüm cevapları.
Diğer şeylerin yanı sıra, aksi halde açık olmayabilecek AJAX çağrılarını görmenizi sağlar.

You can also set breakpoints which allow you to change the requests and responses on the fly.
