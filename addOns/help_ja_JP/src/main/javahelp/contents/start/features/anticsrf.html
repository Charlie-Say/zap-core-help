Anti CSRF トークン

Anti CSRFトークンはクロスサイトリクエストフォージェリ(CSRF)攻撃を防ぐ為に使われている(疑似)ランダムのパラメータです。
しかしながら、これはペネトレーションテスターの仕事を困難なものにしてしまいます。特にトークンがフォームがリクエストされる度に毎回生成される場合です。

　ZAPは、Anti CSRFトークンを属性名で検出します。Anti CSRFトークンと見なされる属性名のリストは、Anti CSRF オプションの画面を使用して構成されます。
　ZAPがこれらのトークンを検出すると、トークン値とトークンを生成したURLを記録します。
　アクティブスキャナのような他のスキャナには、必要に応じてZAPが自動的にトークンを再生成するオプションがあります。
