API

ZAPはApplication Programming Interface (API) を提供しており、ZAPとプログラムで対話する事を可能としています。

The API is available in JSON, HTML and XML formats.
A simple web UI which allows you to explore and use the API is available via the URL http://zap/ when you are proxying via ZAP, or via the host and port ZAP is listening on, eg http://localhost:8080/.

デフォルトでは、ZAPを実行しているマシンのみがAPIにアクセスできます。 ZAPをプロキシとして使用可能な他のマシンに対しても、APIへのアクセスを許可することこができます。 APIはAPI オプション画面 を使用して設定します。.

APIは動的スキャナーやスパイダーなどのZAPのコア機能へのアクセスを提供しています。
ZAPの将来のバージョンでは、APIを通して利用可能な機能を増やす予定です。
