Release 1.3.0

Les modifications suivantes ont été apportées dans cette version :

Significant changes:

Générateur de bruit

Des chaînes dans une réponse peut maintenant être embrouillé pour essayer de trouver des failles.
Anti CRSF jetons peuvent être détectées et automatiquement généré lorsque le fuzzing.
Cette fonctionnalité est basée sur le code de l'OWASP JBroFuzz projet.

Dynamique SSL certificats

Le soutien pour SSL connexions a été amélioré et simplifié.
Les utilisateurs peuvent désormais créer leur propre certificat racine et de distribuer ce dans leur HTTP les clients.
Voir Option: Dynamic SSL Certificates for more details.

Daemon mode and API

Starting ZAP with the "-daemon" command line option will cause it to run in the background in 'headless' mode, meaning that no UI is displayed.
An initial API has been implemented in XML, JSON and HTML.
If ZAP is running as a daemon then the API is automatically enabled, otherwise the API must be enabled via the Options API screen.
The API can be navigated by opening http://zap/ in your browser when proxying via ZAP.

Intégration BeanShell

The BeanShell is an interactive Java shell that can be used to execute BeanShell scripts.
These scripts are a simplified form of Java that use many elements from Java syntax, but in a simpler scripting format.
Tous les code Java est également valable BeanShell code.
BeanShell intégration dans l'OWASP ZAP vous permet d'écrire des scripts à l'aide du ZAP de fonctions et de données.
Cela peut être une fonctionnalité très puissante pour l'analyse des applications web.

Full internationalisation

Toutes chaîne d'affichage sont maintenant entièrement internationalisé.

Localisation

Changements mineurs :

Hex view

The Requête, Réponse tabs now provide a 'Hex View' option which will display the request and response bodies in hex format.

Search results

The Search tab maintenant affiche toutes les occurrences d'une chaîne est trouvée plutôt que de simplement la première instance dans chaque requête ou de la réponse.

Exclude URLs

URLs can be explicitly excluded from the active scanner, spider and from the proxy.

Copy support

La plupart des panneaux fournissent maintenant un 'clic droit' sur "Copier" du menu option, y compris l'Analyse du Port et de la Force Brute des panneaux.

Undo/Redo support

All of the input fields now support Undo/Redo actions using the operating systems default Undo/Redo accelerators:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Port scanner proxy support and timeout option

The Port Scanner can now use the outgoing proxy (if configured) and the timeout in milliseconds can also now be set.

De demande et de réponse pause boutons

Il y a maintenant 2 'Set Casser" des boutons pour permettre des pauses pour être fixé sur toutes les demandes et toutes les réponses de façon indépendante.

Développer des Sites et des Informations boutons de l'onglet

Il y a maintenant 2 boutons qui vous permettent de basculer entre les Sites et les onglets d'Informations élargi.

Break tab icon changes colour when breakpoint hit

While the Break tab is not in use its icon is a grey cross:    .

Adjustable timeout

The Option: Connection l'écran vous permet de définir le délai d'attente en secondes pour le rendre plus facile pour tester des applications lentes.

Library updates

La plupart des bibliothèques utilisées par ZAP ont été mis à jour vers les dernières versions.

A new icon :)

Merci à Simon Egli et tous les autres pour la présentation cool icône suggestions.

Known issues:

Mac OS X: Dynamic SSL and Google Chrome

Actuellement Dynamique SSL ne fonctionne pas lors de l'utilisation de Google Chrome. C'est parce que de résoudre un problème connu avec Google Chrome et Mac OS X un Trousseau de clés. Lors de l'importation de l'OWASP ZAP de l'autorité de certification Racine dans le trousseau d'accès et de demander un SSL site web, un "certificat non Valide" message d'erreur est affiché.
