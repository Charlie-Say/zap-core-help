2.5.0版本

此版本进行了以下更改：

ZAP API 更改：

查看授权/getAuthorizationDetectionMethod

现在返回包装在对象中，名为authorizationDetectionMethod的数据 例如： 
{"authorizationDetectionMethod":{"statusCode":"-1"..."headerRegex":""}}
而不是： 
{"statusCode":"-1"..."headerRegex":""}

查看上下文/上下文+technologyList

现在返回所有包装在对象中，被称为"上下文"的数据 例如： 
{"上下文":{"id":"1", ...,"inScope":"true","loggedOutPattern":""}}
而不是： 
{"id":"1", ...,"inScope":"true","loggedOutPattern":""}

ACTION spider / scan + scanAsUser

两者现在都支持一个新的可选的"subtreeOnly"参数，该参数将spider程序限制在指定的子树中。 "url"参数现在也是可选的只要提供有效的"上下文"参数

新的"统计"组件

新的"统计"API组件提供了访问权限 统计 现在由ZAP维护。

请注意，一些现有的组件也将有新的操作，请参阅 API Web UI 以获取更多详情
