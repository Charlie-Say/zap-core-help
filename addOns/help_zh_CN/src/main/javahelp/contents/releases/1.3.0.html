1.3.0版本

本版本进行了以下更改：

重大变化：

模糊化

现在响应中的字符串可以模糊化以便发现漏洞。
反CSRF令牌可以被检测到，并在模糊化时自动重新生成。
该功能基于OWASP JBroFuzz项目的代码。

动态SSL证书

对SSL连接的支持得到了改进和简化。
用户现在可以创建自己的根证书并将其分发到他们的HTTP客户端。
请参阅 选项：动态SSL证书 以获取更多信息。

守护进程模式和API

使用“-daemon”命令行选项启动ZAP将使其在“无头”模式下在后台运行，这意味着没有UI显示。
最初版本的API已经在XML，JSON和HTML中实现。
如果ZAP作为守护程序运行，则API将自动启用，否则API必须通过选项 API屏幕启用。
在通过ZAP进行代理时，可以通过在浏览器中打开http://zap/ 来浏览API。

Beanshell集成

The BeanShell 是一个交互式的Java shell，可以用来执行BeanShell脚本。
这些脚本是Java的一种简化形式，它使用Java语法中的许多元素，但采用更简单的脚本格式。
所有的Java代码也是有效的BeanShell代码。
OWASP ZAP中的BeanShell集成使您能够使用ZAP函数和数据集编写脚本。
这可以是分析Web应用程序的一个非常强大的功能。

更国际化

所有显示的字符串现在已经完全国际化。

本地化

对下列语言内置支持：
    英文
    巴西 葡萄牙语
    中文
    法语
    德语
    希腊语
    印尼语
    日语
    波兰语
    西班牙语

小更改：

十六进制视图

一个 请求, 响应 tabs now provide a 'Hex View' option which will display the request and response bodies in hex format.

搜索结果

The Search tab now displays all instances of a string found rather than just the first instance in each request or response.

排除地址（URLs）

URLs可以明确地从主动扫描器，Spider和代理中排除。

提供复制功能

大多数面板现在提供“右键单击”“复制”菜单选项，包括端口扫描面板和蛮力组件面板。

提供Undo/Redo功能

所有输入字段区域现在都支持使用操作系统默认的撤销/重做加速器的撤消/重做操作：
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

端口扫描器代理支持和超时选项

端口扫描器现在可以使用传出代理（如果已配置），现在也可以设置超时（以毫秒为单位）。

请求和响应中断按钮

There are now 2 'Set Break' buttons to allow breaks to be set on all requests and all responses independently.

Expand Sites and Information tab buttons

There are now 2 buttons which allow you to switch between having the Sites and Information tabs expanded.

Break tab icon changes colour when breakpoint hit

可调整的超时设置

一个 选项：连接 screen allows you to set the timeout in seconds to make it easier to test slow applications.

库的更新

ZAP使用的大多数库已经更新到最新版本。

一个新的图标

已知问题：

Mac OS X：动态SSL和Google Chrome浏览器

目前动态SSL在使用Google Chrome时无法使用。 这是因为Google Chrome和Mac OS X钥匙串尚未解决的已知问题。 将OWASP ZAP的根CA导入钥匙串并请求SSL网站时，会显示“无效证书”错误消息。
