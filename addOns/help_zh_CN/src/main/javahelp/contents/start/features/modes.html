模式

ZAP的模式可以是：

  • 安全 - 不允许有潜在危险的操作
  • 受保护 - 您只能对范围中的网址执行（可能的）危险操作
  • 标准 - 和以前的版本一样，您可以做任何事情
  • 攻击 - 一旦发现范围中的新节点，就立即进行主动扫描
建议您使用保护模式来确保您只攻击您指定的网站。

The mode can be changed via the toolbar (or the ZAP API) and is persisted between sessions.

在不对范围中的网址进行操作时，在安全模式或保护模式下无法实现的示例：
  • 抓取
  • 主动扫描
  • 模糊化
  • 强制浏览
  • 中断
  • 重新发送请求
您可以定义 扫描策略 to be used for the Attack mode the Options Active Scan screen.
