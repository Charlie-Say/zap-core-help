رہائی 1.0.0

اس ریلیز میں مندرجہ ذیل تبدیلیاں کی گئی تھیں:

اہم تبدیلیاں:

مدد کی اسکرینز

مدد اسکرینوں اب زیپ کی طرف سے فراہم کی فعالیت کے سب کے لئے شامل کیا گیا ہے.
متناسب حساس مدد کو بھی لاگو کیا گیا ہے تاکہ جب آپ F1 کلیدی پر دبائیں تو آپ کے منتخب کردہ ٹیب کے ساتھ منسلک اسکرین کی مدد کی جائے گی.

Breakpoints

کا عمل Breakpoints اس کے ذریعہ فراہم کردہ اس کے قریب ہونے کے لئے تبدیل کر دیا گیا ہے سوفٹ ویئر انٹیگریٹڈ ڈویلپمنٹ ماحول.
The breakpoint controls have been moved to the new اوپر سطح ٹول بار.
Breakpoints can now also be set by right clicking nodes in the سائٹس اور تاریخ ٹیبز.

رپورٹ مینو

اس قدر رپورٹ مینو بہت اہم تبدیل ہوا ہے اور اب بھی شامل ہے مندرجہ ذیل مینو اشیاء:
    رپورٹ بنائیں ...جو اب مطالبہ پر رپورٹ پیدا کرتا ہے
    فائلوں میں فائل برآمد کریں ...سے منتقلفائل مینو
    فائلوں میں فائل برآمد کریں ......سے منتقلفائل مینو
    تمام URL کو فائل میں برآمد کریں ...نئی فعالیت
    دوسرے سیشن کے ساتھ موازنہ کریں ...نئی فعالیت

بہتر تاریخ کی تلاش

نیا تلاش ٹیب آپ کو تمام URLs، درخواستوں اور جوابات میں باقاعدگی سے اظہار کی تلاش کرنے کی اجازت دیتا ہے.
اگلے اور پچھلے بٹن آپ کو تلاش کرنے کی شرائط کو دیکھتے ہیں درخواستیں اور جوابی ٹیبز.

تاریخ فلٹر

تاریخ فلٹر ٹول بار شامل کردی گئی ہے جو آپ کو محدود کرنے کی اجازت دیتا ہے جو درخواستیں پیش کی جاتی ہیں.
درخواستوں کو HTTP طریقوں، HTTP کے نتائج کوڈوں پر مبنی فلٹر کیا جا سکتا ہے، اور کوئی ٹیگز, انتباہاتیا نوٹ وہ لاگو کیا گیا ہے.

نوٹس

آپ اب ایسوسی ایٹ کرسکتے ہیں نوٹ کسی بھی درخواست کے ساتھ.

ایک سے زیادہ ٹیگز

آپ اب ایک سے زیادہ مل سکتے ہیں ٹیگز کسی بھی درخواست کے ساتھ.

انتباہات پر مزید کنٹرول

اس قدر انتباہ ڈائیلاگ شامل کریں آپ کو دستی طور پر شامل کرنے یا تبدیل کرنے کی اجازت دیتا ہے انتباہ.

نیا انکوڈر / ڈوڈیڈر

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

غیر فعال سکیننگ

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

چھوٹی موٹی تبدیلیاں:

اوپر سطح کے ٹول بار

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

انتباہ فوٹر خلاصہ

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

نئی نظر اور محسوس

A new look and feel 'Nimbus' is used.

پراکسی پاس ورڈ ہینڈلنگ

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

سائٹس کے ٹیب سے رجوع کریں

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

صارف ڈائرکٹری سیشن کے درمیان برقرار رکھا گیا ہے

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
