جاری ہے 2.0.0

اس رہائی میں مندرجہ ذیل تبدیلیاں کی گئی تھیں:

اہم تبدیلیاں:

ایک مربوط اضافے کی مارکیٹ

زیپ اضافوں زیپ اندرونی کے تمام کرنے کے لئے مکمل رسائی حاصل ہے کہ کی طرف سے فراہم کیا جا سکتا. کوئی بھی شخص شامل ons لکھیں اور زیپ پر اپ لوڈ کر سکتے ہیں مارکیٹ پر شامل کریں (ٹھیک ہے، تو اس کی ایک گوگل کوڈ منصوبہ جیپ-توسیعات کہا جاتا ہے، لیکن آپ کو اندازہ ہو).
مزید اہم بات اب آپ براؤز کرسکتے ہیں، ڈاؤن لوڈ اور زیپ میں سے ان اضافوں انسٹال کریں. زیادہ اضافی اضافی طور پر انسٹال کیا جا سکتا ہے (اور انسٹال شدہ) لہذا آپ کو دوبارہ شروع کرنے کی ضرورت نہیں ہے.
آپ اپ ڈیٹس کے بارے میں مطلع کرنے کا انتخاب کرسکتے ہیں، اور یہاں تک کہ خود بخود اپ ڈیٹ کیا جائے. اور جیسے ہی اسکین کے قوانین کو اضافی اضافی طور پر نافذ کیا جاسکتا ہے، جیسے ہی آپ شائع کیے جائیں گے جیسے ہی آپ تازہ ترین قواعد حاصل کرسکتے ہیں.

'معیاری' مکڑی کے لئے متبادل

'پرانے' مکڑی اپنی عمر کو دکھا رہا تھا، لہذا یہ مکمل طور پر دوبارہ لکھا گیا ہے، اور پرانے ایک سے زیادہ تیز اور زیادہ جامع ہے. یہ اب بھی ایک 'روایتی' مکڑی ہے جو کسی بھی لنکس کے لۓ HTML کوڈ کا تجزیہ کرتا ہے.

ایک نیا 'ایجیکس' مکڑی

'روایتی' مکڑی کے علاوہ ہم ایک ایجیکس مکڑی شامل کر چکے ہیں جو ایپلی کیشنز کے ساتھ زیادہ موثر ہے جو جاوا اسکرپٹ کا بھاری استعمال کرتی ہے. یہ کرلاجیکس پراجیکٹ کا استعمال کرتا ہے جو براؤزر کو چلاتا ہے (سلیمانیم کا استعمال کرتے ہوئے) اور اس وجہ سے کسی بھی لنکس کو کسی بھی ایپلیکیشن کی تخلیق کرسکتا ہے، یہاں تک کہ پیدا شدہ کلائنٹ کی طرف.

ویب ساکٹ کی حمایت

زیپ اب ویب ساکٹ کو حمایت, تاکہ زیپ اب کرنے کے لئے اور آپ کے براؤزر کی طرف سے بھیجے گئے تمام ویب ساکٹ کی پیغامات کو دیکھ سکتے ہیں. HTTP کی بنیاد پر پیغامات کے ساتھ، زیپ بھی ویب ساکٹ کی پیغامات تقطیع اور آپ مکھی پر ان کو تبدیل کرنے کی اجازت دیتا ہے کر سکتے ہیں.
آپ Fuzz ویب ساکٹ پیغامات کے ساتھ ساتھ ZBroFuzz اور Fuzzdb جیسے منصوبوں میں شامل تمام فجینگ پاؤ لوڈز کا استعمال کرتے ہوئے بھی فجی کرسکتے ہیں.

فوری شروع ٹیب

آپ پہلے دیکھیں گے کہ پہلا اہم ٹیب ایک 'فوری آغاز' ٹیب ہے جس سے آپ کو صرف ایک یو آر ایل میں ٹائپ کرنے اور ایک کلک کے ساتھ اسکین کرنے کی اجازت دیتا ہے.
یہ درخواست کے سیکورٹی میں نئے لوگوں کے لئے ایک مثالی نقطہ نظر ہے، لیکن ماہرین اسے مشغول کرنے کے لۓ آسانی سے دور کرسکتے ہیں.

سیشن بیداری

زیپ تاکہ زیپ کو تسلیم اور ایک سے زیادہ سیشن کے ٹریک رکھ سکتے ہیں، سیشن بیداری اب ہے. یہ آپ کو نئے سیشن بنانے، ان کے درمیان سوئچ بنانے، اور مکڑی اور فعال سکینر کی طرح دیگر اجزاء پر لاگو کرنے کی اجازت دیتا ہے.

صارف کی وضاحت کی شرائط

اب آپ یو آر ایل کے کسی بھی قسم کے 'قواعد و ضوابط' سے متعلق سیٹ کی وضاحت کرسکتے ہیں جو ایک اپلی کیشن بناتے ہیں. پھر آپ سبھی یو آر ایل کو ایک سیاق و سباق میں ھدف کرسکتے ہیں، مثال کے طور پر مکڑی یا فعال سکینر. آپ اساتذہ کو گنجائش میں بھی شامل کرسکتے ہیں، اور دیگر معلومات کو شریک کرسکتے ہیں، جیسے توثیق کی تفصیلات.

اجلاس کا گنجائش

سیشن گنجائش آپ کو یہ بتانے کی اجازت دیتا ہے کہ آپ کسی بھی وقت دلچسپی رکھتے ہیں. آپ مختلف ٹبوں میں صرف گنجائش کے URL پر جو کچھ دیکھتے ہیں اسے روک سکتے ہیں، اور غیر متوقع طور پر حملہ کرنے والے URL کو محفوظ طریقے سے استعمال کرنے کے ذریعہ گنجائش میں نہیں روک سکتے ہیں.

مختلف طریقوں

زیپ اب تین طریقوں کی حمایت:
  • محفوظ، جس میں کوئی ممکنہ خطرناک آپریشن کی اجازت نہیں ہے
  • محفوظ، جس میں آپ گنجائش میں یو آر ایل پر کوئی کارروائی کر سکتے ہیں
  • معیاری، جس میں آپ کسی بھی URL میں کچھ بھی کرسکتے ہیں

توثیقی ہینڈلنگ

آپ اب آپ لاگ آؤٹ کر رہے ہیں اگر اور جب پتہ لگانے اور خود کار طریقے سے ایک بار پھر میں آپ کو دوبارہ لاگ طرح زیپ چیزیں کرنے کی اجازت دیتا ہے جس میں کسی بھی سیاق و سباق کے ساتھ ایسوسی ایٹ تصدیق تفصیلات. یہ خاص طور پر مفید ہے جب حفاظتی رجحان ٹیسٹ میں API کے ذریعے استعمال کیا جاتا ہے.

More API support

The REST API has been significantly extended, giving you much more access to the functionality ZAP provides.

Fine grained scanning controls

The active scan rules can now be tuned to adjust their strength (the number of attacks they perform) and the threshold at which they report potential issues.

New and improved active and passive scanning rules
