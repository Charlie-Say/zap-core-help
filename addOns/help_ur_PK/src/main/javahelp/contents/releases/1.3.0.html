1.3.0 ریلیز

اس رہائی میں مندرجہ ذیل تبدیلیاں کی گئی تھیں:

اہم تبدیلیاں:

فجینگ

ایک جواب میں ہڑتال اب کمزوریاں تلاش کرنے کی کوشش کرنے کے لئے پریشان ہوسکتی ہیں.
فجینگ کرتے وقت اینٹی CSRF ٹکنز کا پتہ چلا اور خود کار طریقے سے دوبارہ تخلیق کیا جاسکتا ہے.
یہ فعالیت OWASP JBroFuzz پروجیکٹ سے کوڈ پر مبنی ہے.

متحرک SSL سرٹیفکیٹ

ایس ایس ایل کنکشن کے لئے تعاون بہتر اور آسان کیا گیا تھا.
یوزر کا اب اپنا اپنا جڑ سرٹیفکیٹ تشکیل دے سکتا ہے اور اسے اپنے HTTP گاہکوں میں تقسیم کر سکتا ہے.
دیکھو اختیاری: متحرک SSL سرٹیفکیٹس مزید تفصیلات کے لیے.

ڈیمون موڈ اور API

"ڈیڈینیم" کمانڈ لائن اختیاری کے ساتھ ZAP شروع کرنا اس کے نتیجے میں 'سرless' موڈ میں پس منظر میں چلاتا ہے، مطلب یہ ہے کہ کوئی UI نہیں دکھایا گیا ہے.
ایک ابتدائی API کو XML، JSON اور HTML میں لاگو کیا گیا ہے.
اگر ZAP ڈیمون کے طور پر چل رہا ہے تو API خود بخود فعال ہوجاتا ہے، دوسری صورت میں API کو API API کے ذریعے فعال ہونا ضروری ہے.
API آپ کے براؤزر میں ZAP کے ذریعے proxying جب http: // زپ / کھولنے کی طرف سے نیویگیشن کیا جا سکتا ہے.

بیشیل انضمام

اس قدر بین شیل ایک انٹرایکٹو جاوا شیل ہے جو BeanShell سکرپٹ کو عمل کرنے کے لئے استعمال کیا جا سکتا ہے.
یہ سکرپٹ جاوا کے ایک سادہ شکل ہیں جو جاوا نحو سے بہت سے عناصر کا استعمال کرتے ہیں، لیکن ایک آسان سکرپٹ کی شکل میں.
تمام جاوا کوڈ بھی درست شین شیل کوڈ ہے.
OWASP ZAP میں بین انضمام آپ کو ZAP افعال اور ڈیٹا سیٹ کا استعمال کرکے سکرپٹ لکھنے کے قابل بناتا ہے.
یہ ویب ایپلی کیشنز کا تجزیہ کرنے کے لئے بہت طاقتور خصوصیت ہوسکتی ہے.

مکمل بین الاقوامی کاری

تمام ڈسپلے سٹرنگ اب مکمل طور پر بین الاقوامی بنائے جاتے ہیں.

Minor changes:

Hex view

The Request, Response tabs now provide a 'Hex View' option which will display the request and response bodies in hex format.

Search results

The Search tab now displays all instances of a string found rather than just the first instance in each request or response.

Exclude URLs

URLs can be explicitly excluded from the active scanner, spider and from the proxy.

Copy support

Most of the panels now provide a 'right click' 'Copy' menu option, including the Port Scan and Brute Force panels.

Undo/Redo support

All of the input fields now support Undo/Redo actions using the operating systems default Undo/Redo accelerators:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Port scanner proxy support and timeout option

The Port Scanner can now use the outgoing proxy (if configured) and the timeout in milliseconds can also now be set.

Request and response break buttons

There are now 2 'Set Break' buttons to allow breaks to be set on all requests and all responses independently.

Expand Sites and Information tab buttons

There are now 2 buttons which allow you to switch between having the Sites and Information tabs expanded.

Break tab icon changes colour when breakpoint hit

Adjustable timeout

The Option: Connection screen allows you to set the timeout in seconds to make it easier to test slow applications.

Library updates

Most of the libraries used by ZAP have been updated to the latest versions.

A new icon :)

Thanks to Simon Egli and all others for submitting cool icon suggestions.

Known issues:

Mac OS X: Dynamic SSL and Google Chrome

Currently Dynamic SSL is not working when using Google Chrome. This is because of an unresolved known issue with Google Chrome and Mac OS X Keychain. When importing OWASP ZAP's Root CA into the keychain and requesting a SSL website, an "Invalid certificate" error message is shown.
