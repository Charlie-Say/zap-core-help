Xəbərdarlıq təsir edə bilər, müxtəlif zap komponentləri, o cümlədən, lakin məhdud deyil: fəal scan, passiv scan, skript istifadə edərək, addons (uzantıları), və ya əl ilə Əlavə et Алерт dialoq (ki, bu da imkan verir yeniləmək və ya dəyişmək xəbərdarlıq məlumat/məlumat).

Xəbərdarlıq qeyd olunur bookmark "Tarix" bayrağı ilə olan göstərir ki, yüksək risk xəbərdarlıq edir.
Bütün xəbərdarlıq verilən siqnallar nişanõ və counter ümumi sayının xəbərdarlıq risk təmsil футер.

Xəbərdarlıq ləğv edir

Siqnallar zap daxildir həm ümumi, həm də konkret məlumatlar haqqında bildiriş. Konkret məlumatlar aiddir birbaşa potensial problemi, məsələn, URL-ünvanı və parametr təsir edir. Ümumi məlumat daxildir, belə şeylər kimi təsviri və links müvafiq internet resursları.

Əvəz edə bilərsiniz və ya əlavə etmək üçün Ümumi məlumatlar istifadə edərək, konfiqurasiya faylı və siqnallar yalnış'. Bu, imkan verir məlumatları daxil nəzərdə tutulmuşdur üçün Sizin şirkət kimi elan siyasət, daxili bağlantılar və ya məsləhətlər üzrə xüsusi texnologiyalar istifadə etdiyiniz.

An alert override configuration file is a UTF-8 property file containing just the information you would like to change. Lines starting with '#' are treated as comments and ignored.

Format: 

<alertid>.<property> = [ + | - ] <your information>
Dəstəklənir aşağıdakı xassələri:
  • name
  • description
  • solution
  • otherInfo
  • reference
For example 
# 40012 = əks xss azn 40012.qərar = saxlamaq üçün bizim konkret rəhbər prinsipləri, şirkətin http://internet.example.com/xss.html
Əgər əhəmiyyəti ilə başlayır'+', onda o əlavə edilir və mövcud məlumat.
Əgər o, başlayır ' - ' o əlavə edilir və mövcud məlumat.
Əgər o, heç bir ilə başlayır '+' və ya ' - ' o əvəz mövcud məlumat.

Konfiqurasiya siqnallar yalnış fayl vasitəsilə göstərilə bilər API, Options Alert screen or using the command line option: 

-config alert.overridesFilename=<filename>
