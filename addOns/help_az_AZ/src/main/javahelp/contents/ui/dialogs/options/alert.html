Seçimlər xəbərdarlığı ekranı

Bu ekran, xəbərdarlıq variantlarını yapılandırmanıza imkan verir:

Merge related alerts in report

Seçilibsə, əlaqədar problemlər yaradılan bütün hesabatlarda birləşdirilir.
Bu, təkrar məlumatlar qaldırılacaq hesabat ölçüsünü əhəmiyyətli ölçüdə azaldır. cted sonra əlaqədar problemlər yaradılan hər hansı bir hesabat birləşdirilir.

Hesabatda ən çox xəbərdarlıq nümunələri

The maximum number of alert instances to include in a report.
A value of zero is treated as unlimited.

Alert overrides file

The full name of a properties file that specifies any alert overrides that you want to apply.
