Probleme cunoscute:

Mac OS X: SSL dinamic și Google Chrome

SSL-ul dinamic curent nu merge cand este folosit Google Chrome. Acest lucru se întamplă din cauza unei probleme nerezolvate cu Google Chrome și Keychain-ul Mac OS X. În momentul importării Ca-ului rădăcină a OWASP ZAP în keychain și cererea unui site SSL, o eroare este afișată ”Certificat Invalid”.
