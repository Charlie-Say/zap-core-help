Versiunea 1.0.0

Următoarele schimbări au fost aduse acestei versiuni:

Schimbări semnificative:

Ecranele de ajutor

Ecrane de ajutor au fost adăugate pentru toate funcțiile oferite de ZAP.
Ajutorul contextual a fost de asemenea implementat, astfel când apeși tasta F1 va fi afișat ecranul de ajutor asociat tab-ului selectat.

Breakpoints

Implementarea Breakpoints a fost schimbată pentru a fi mai apropiată software-ului mediilor de integrare a dezvoltării.
The breakpoint controls have been moved to the new Top Level Toolbar.
Breakpoints can now also be set by right clicking nodes in the tab-ul Site-uri și Istoric.

Meniu pentru rapoarte

Meniul pentru rapoarte a suferit schimbări majore, incluzând următoarele elemente de meniu:
    Generează Raport...Ce poate acum genera rapoarte la cerere
    Exportă Mesaje către Fișier...Mutat din Meniu Fișier
    Exportă Răspuns către Fișier...Mutat din Meniu Fișier
    Exportă toate URL-urile către Fișier...Funcționalitate nouă
    Compară cu alte sesiuni...Funcționalitate nouă

Căutare îmbunătățită a istoricului

Noul tab de Căutări îți permite să cauți exprimări normale în cererile și răspunsurile tuturor URL-urilor.
Butoanele următorul și anteriorul îți permit să vizualizezi termenii căutați în tab-ul Cerere și Răspuns.

Filtrul pentru istoric

Toolbar-ul pentru a filtra istoricul a fost adăugat pentru a îți permite să restricționezi cererile afișate.
Cererile pot fi filtrate în baza metodelor HTTP, codurilor de rezultat HTTP, și orice tag-uri, alertesau notițe ce au fost aplicate.

Notițe

Acum poți asocia notițe oricărei cereri.

Tag-uri multiple

Acum poți asocia multiple tag-uri oricărei cereri.

Mai mult control asupra alertelor

Căsuța de dialog Adaugă Alertă îți permite să adaugi sau să schimbi manual o alertă.

Noul codor/decodor

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Scanare pasivă

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Schimbări minore:

Toolbar-ul Top Level

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Rezumatele alertelor de subsol

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Un nou aspect

A new look and feel 'Nimbus' is used.

Manipularea parolelor proxy

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Retrimite din tab-ul Site-uri

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Directorul utilizatorului este menținut între sesiuni

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
