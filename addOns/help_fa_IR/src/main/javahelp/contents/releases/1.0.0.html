نسخه 1.0.0

تغییرات زیر در این نسخه ایجاد شده اند:

تغییرات مهم:

صفحات راهنما

صفحات راهنما برای تمام قابلیت های فراهم شدهی کنونی توسط ZAP افزوده شده اند.
راهنمای حساس به متن نیز پیاده سازی شده است چنانکه وقتی کلید F1 را فشار دهید سپس صفحه ی راهنمای مربوط به برگه ای که انتخاب کرده اید نمایش داده خواهد شد.

Breakpoints

The implementation of Breakpoints تغییر یافته است تا به آنچه که توسط محیط توسعه ی ادغام شده ی نرم افزار فراهم شده است، نزدیکتر باشد.
The breakpoint controls have been moved to the new نوار ابزار بالایی.
Breakpoints can now also be set by right clicking nodes in the سایت و برگه های سوابق.

منوی گزارش

این منوی گزارش بطور قابل ملاحظه ای تغییر یافته و حالا شامل آیتم های منوی زیر می باشد:
    تولید گزارش...که حالا گزارش ها را بنا به درخواست تولید می کند
    صدور پیام ها به فایل...از منوی فایل تغییر مکان داده شده
    صدور پاسخ به فایل...از منوی فایل تغییر مکان داده شده
    صدور همه ی URL ها به فایل...قابلیت جدید
    مقایسه با جلسه ای دیگر...قابلیت جدید

جستجوی سوابق پیشرفت یافته

این جدید برگه ی جستجو به شما اجازه می دهد که برای عبارات متداول در تمام URL ها جستجو کنید، درخواست ها و پاسخ ها.
دکمه های بعدی و قبلی به شما اجاز می دهند که عبارات جستجوی یافته شده را مشاهده کنید در درخواست و برگه های پاسخ.

فیلتر سوابق

نوار ابزار فیلتر سوابق افزوده شده است که به شما این امکان را می دهد تا اینکه کدام درخواست ها نمایش داده شوند را محدود کنید.
درخواست ها می توانند فیلتر شوند بر اساس متد های HTTP، نتایج کد های HTTP، و هر برچسب ها, هشدار ها، یا یادداشت هایی که اعمال شده باشند.

یادداشت ها

شما اکنون میتوانید پیوند دهید یادداشت ها را با هر نوع درخواستی.

چنیدن برچسب

شما اکنون می توانید پیوند دهید چندین برچسب را با هر درخواستی.

کنترل بیشتر بر روی هشدار ها

این دیالوگ افزودن هشدار به شما اجازه می دهد به صورت دستی یک هشدار را بیافزایید یا تغییر دهید.

رمزگذار/رمزگشا ی جدید

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

پویش منفعل

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

تغییرات جزئی:

نوار ابزار بالایی

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

خلاصه های پاورقی هشدار

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

حس و شمایل جدید

A new look and feel 'Nimbus' is used.

مدیریت رمز عبور پروکسی

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

ارسال دوباره از برگه ی سایت ها

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

دایرکتوری کاربر در بین جلسات حفظ می شود

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
