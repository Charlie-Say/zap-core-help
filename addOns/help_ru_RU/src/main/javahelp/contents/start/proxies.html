Настройка прокси

Вам нужно будет настроить свой браузер для использования ZAP в качестве прокси.
По умолчанию ZAP использует адрес localhost и порт 8080, но их можно изменить с помощью Экран настроек локальных прокси . Инструкции для последних версий наиболее часто используемых браузеров:

Chrome (on Windows)

    нажмите Кнопка "Настройка и управление Google Chrome" (вверху справа)
    Выберите Пункт меню "Параметры .."
    нажмите Кнопка "Изменить настройки прокси"
    нажмите Кнопка "Настройки LAN"
    Следовать Раздел "Настройки локальной сети Windows" ниже
Примечание: Прокси localhost (и связанные адреса) с более новыми версиями Chrome (> = 72) аргумент командной строки --proxy-bypass-list=<-loopback> должно быть предоставлено.

Firefox (on Windows)

    Выберите Меню "Инструменты"
    Выберите Пункт меню "Параметры .."
    Выберите Панель "Общие"
    Прокрутите до Раздел "Сетевые настройки"
    нажмите Кнопка "Настройки ..."
    Выберите Радиокнопка "Ручная настройка прокси"
    Войдите в "HTTP Proxy:" поле "Адрес", который вы настроили в Экран настроек локальных прокси
    Войдите в Поле «Порт» справа от поля «HTTP-прокси» - это «Порт», который вы настроили в Экран настроек локальных прокси . Убедитесь, что «SSL-прокси» также настроен, выбрав «Использовать этот прокси-сервер для всех протоколов» или задав соответствующие значения.
    нажмите Кнопка «ОК» настройки подключения
    нажмите Опции кнопки "ОК"
Примечание: Прокси localhost (и связанные адреса) с более новыми версиями Firefox (> = 67) предпочтение network.proxy.allow_hijacking_localhost (доступно через about:config страницу) должен быть установлен на истина.

Firefox (on Linux)

    Выберите Меню "Правка"
    Select the'Preferences' menu item
    Follow theinstructions above from the 'General' panel

Firefox (on OS X)

    Select the 'Firefox' menu
    Select the 'Preferences...' menu item
    Follow the instructions above from the 'General' panel

Internet Explorer 7 and 8

    Select the 'Tools' menu
    Select the 'Internet Options' menu item
    Select the 'Connections' tab
    Press the 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

Opera

    Select the 'Menu' in the top left hand corner
    Select the 'Settings' menu item
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Select the 'Network' item in the left hand list
    Press the 'Proxy Servers' button
    Select the 'HTTP' checkbox
    Enter in the field to the right of 'HTTP' the 'Address' you configured in the Options Local Proxies screen
    Enter in the HTTP 'Port' field the 'Port' you configured in the Options Local Proxies screen
    Press the Proxy servers 'OK' button
    Press the Preferences 'OK' button

Safari (on OS X)

    Press the 'Safari settings' button in the top right hand corner
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Press the 'Change Settings' button next to the 'Proxies' label
    Follow the 'OS X System Proxy settings' section below

Safari (on Windows)

    Press the 'Display a menu of general Safari settings' button in the top right hand corner
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Press the 'Change Settings' button next to the 'Proxies' label
    Press the 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

OS X System Proxy settings

    Select the 'Web Proxy (HTTP)
    Enter in the 'Address:' field the 'Address' you configured in the Options Local Proxies screen
    Enter in the 'Port' field the 'Port' you configured in the Options Local Proxies screen
    Press the Proxies Setting 'OK' button
    Press the Network Options 'Apply' button

Windows LAN Settings

    Select the 'Use a proxy server for your LAN' checkbox
    Enter in the 'Address:' field the 'Address' you configured in the Options Local Proxies screen
    Enter in the 'Port' field the 'Port' you configured in the Options Local Proxies screen
    Press the Local Area Network (LAN) Setting 'OK' button
    Press the Internet Options 'OK' button
