Глобально исключенные URL

Глобально исключенные URL-адреса - это набор регулярных выражений. (регулярное выражение), которое ZAP полностью игнорирует во всем приложении. Эти URL-адреса не будут отображаться в прокси, сканере или пауке для ZAP. Кроме того, регулярные выражения URL будут сохранены в глобальном конфигурация для ZAP, а не внутри файла сеанса. Эти URL-адреса будут сохраняться между каждым использованием ZAP.

Регулярное выражение для URL настраивается с помощью Global Экран исключения URL в параметрах ZAP.
