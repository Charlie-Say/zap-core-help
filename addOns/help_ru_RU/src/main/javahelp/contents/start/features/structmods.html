Структурные модификаторы

Структурные модификаторы - это элементы управления, которые изменяют то, как ZAP представляет структуру приложения.
дерево сайтов является представлением приложения ZAP.
Если это не очень хорошее представление структуры, ZAP не сможет эффективно атаковать приложение.
В настоящее время существует 2 типа структурных модификаторов:

  • Контент, управляемый данными , который определяет пути URL, которые представляют данные
  • Структурные параметры , которые определяют параметры, которые представляют структуру приложения, а не данные пользователя.
Структурные модификаторы можно применять только к Контексты.
