Область

Область действия - это набор URL-адресов, которые вы тестируете, и определяется контекстами вы указали. <1 /> <1 /> По умолчанию ничего не входит.

Возможные изменения объема:

  • Что вы можете делать, находясь в защищенном режиме
  • Что показано на вкладке История
  • Защищено - пользователь может выполнять (потенциально) опасные действия только с URL-адресами в Области действия.
  • Стандарт - как и в предыдущих выпусках, пользователь может делать что угодно
  • ATTACK - новые узлы, которые находятся в Scope, активно сканируются , как только они обнаруживаются
Рекомендуется определить новый Контекст для каждого веб-приложения которые составляют тестируемую вами систему, и устанавливайте их масштаб по мере того, как вы тестируете каждую из них.
