API

ZAP предоставляет интерфейс прикладного программирования (API), который позволяет вам программно взаимодействовать с ZAP.

API доступен в форматах JSON, HTML и XML. <0 /> Простой веб-интерфейс, который позволяет вам исследовать и использовать API, доступен по URL-адресу http: // zap / когда вы проксируете через ZAP или через хост и порт, который прослушивает ZAP, например http: // localhost:8080/ .

По умолчанию только компьютер, на котором запущен ZAP, может получить доступ к API. Вы можете разрешить другим машинам, которые могут использовать ZAP в качестве прокси доступ к API. API настраивается с помощью экрана API параметров .

API обеспечивает доступ к большинству основных функций ZAP, таких как активный сканер и паук . <2 /> В будущих версиях ZAP будет увеличена функциональность, доступная через APi.
