Анти-CSRF токены

Токены против CSRF - это (псевдо) случайные параметры, используемые для защиты от атак с подделкой межсайтовых запросов (CSRF).
Однако они также усложняют работу тестерам на проникновение, особенно если токены регенерируются каждый раз, когда запрашивается форма.

ZAP detects anti CSRF tokens purely by attribute names - the list of attribute names considered to be anti CSRF tokens is configured using the Options Anti CSRF screen.
When ZAP detects these tokens it records the token value and which URL generated the token.
Other tools, like the active scanner, have options which cause ZAP to automatically regenerate the tokens when required.
