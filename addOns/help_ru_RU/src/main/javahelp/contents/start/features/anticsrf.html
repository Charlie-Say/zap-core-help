Анти-CSRF токены

Токены против CSRF - это (псевдо) случайные параметры, используемые для защиты от атак с подделкой межсайтовых запросов (CSRF).
Однако они также усложняют работу тестерам на проникновение, особенно если токены регенерируются каждый раз, когда запрашивается форма.

ZAP обнаруживает анти-CSRF-токены исключительно по именам атрибутов - списку имен атрибутов, которые считаются анти-CSRF-токенами. настраивается с помощью экрана параметров Anti CSRF . <1 /> Когда ZAP обнаруживает эти токены, он записывает значение токена и URL, сгенерировавший токен. <1 /> Другие сканеры, такие как активный сканер , имеют параметры, которые заставить ZAP автоматически регенерировать жетоны при необходимости.
