Стратегии проверки

ZAP поддерживает несколько стратегий проверки , чтобы определять, соответствуют ли сообщения аутентифицированным запросам.

Проверить каждый ответ

Когда используется эта стратегия, ZAP будет использовать указанные шаблоны регулярных выражений при каждом ответе. Обычно это полезно для традиционных веб-приложений, которые возвращают полные HTML-страницы.

Проверить каждый запрос

Когда используется эта стратегия, ZAP будет использовать указанные шаблоны регулярных выражений при каждом запросе. Это обычно полезно для современных веб-приложений, которые поддерживают некоторую часть состояния сеанса на стороне клиента, например JWT, который отправляется с каждым запросом.

Проверяйте каждый запрос или ответ

Когда используется эта стратегия, ZAP будет использовать указанные шаблоны регулярных выражений как в запросе, так и в ответе на каждое сообщение.

Опрос указанного URL

Когда используется эта стратегия, ZAP будет опрашивать указанный URL с заданной частотой. Сообщения будут обрабатываться как зарегистрированные или вышедшие из системы в зависимости от последнего запроса опроса. Указанные шаблоны регулярных выражений будут использоваться только в ответе на запрос опроса. Эта стратегия обычно полезна для любого приложения, для которого есть хотя бы один URL-адрес. которые можно надежно использовать для определения того, вошел ли пользователь в систему или нет.

Шаблоны регулярных выражений

Все стратегии используют 2 шаблона регулярных выражений, чтобы определить, зарегистрировано ли указанное сообщение или нет. Вы можете указать одно или оба регулярных выражения - если вы не укажете ни одно из них, стратегия аутентификации использоваться не будет.

Чтобы установить один из индикаторов входа / выхода , введите либо регулярное выражение непосредственно в сеансе Экран контекстной аутентификации диалоговое окно -> панель аутентификации -> поле индикатора входа / выхода , или найдите аутентифицированное сообщение в дереве сайтов или на вкладке «История», выберите его, откройте Просмотр отклика и выберите текст, который вы хотите определить как индикатор, с помощью мышью и выберите Флаг как контекст ... Индикатор входа / выхода , вызываемый правой кнопкой мыши.
