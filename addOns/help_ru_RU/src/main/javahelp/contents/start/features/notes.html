Примечания

Заметка - это любой текст, который вы хотите связать с запросом. <0 /> Например, вы можете использовать примечания для записи деталей дополнительных тестов, которые вам необходимо выполнить по запросу. <0 /> Примечания добавляются и изменяются с помощью диалогового окна "Добавить примечание ". .

Заметки отмечены на вкладке "История" значком Значок <1 />.
