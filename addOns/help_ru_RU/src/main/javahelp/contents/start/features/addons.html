Аддоны

Аддоны добавляют дополнительную функциональность в ZAP.
Они имеют полный доступ ко всем зап внутренних органов и поэтому может обеспечить очень мощные новые возможности.
Вы можете динамически установить дополнения из онлайн магазина дополнения в диалоговое окно Управление надстройками.
Обычно можно добавлять и удалять дополнения и из зап пользовательского интерфейса без необходимости перезагружать его.

Надстройки могут предоставлять контекстно-зависимую справку, которая динамически интегрируется с файлом справки. <0 /> Если вы читаете эту страницу через файл справки ZAP (а не читаете ее в Интернете), то любой страницы справки, связанные с установленными надстройками, будут доступны в здесь .

Дополнениям присваивается статус, который может быть одним из:
     Релизчто указывает на их высокое качество и соответствие назначению.
     Бета-версиячто указывает на их приемлемое качество, но может быть неполным или требовать дальнейшего тестирования.
     Альфачто указывает на то, что они находятся на ранней стадии разработки

Расположение дополнений, доступных для ZAP

Чтобы надстройка стала доступной для ZAP, она должна находиться в одном из следующих мест:
  • plugin directory located in the ZAP's installation folder;
  • plugin directory located in the ZAP's user folder.
