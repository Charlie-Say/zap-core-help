Статистика

ZAP ведет статистику, которая может помочь вам понять, что на самом деле происходит при взаимодействии с большими приложениями.

Статистика доступна через API и также может быть отправлена на сервер Statsd, когда настраивается на экране статистики параметров .

Полный набор статистики, поддерживаемой ядром ZAP и надстройками, поддерживается на веб-сайте ZAP: ﻿Общая статистика.
