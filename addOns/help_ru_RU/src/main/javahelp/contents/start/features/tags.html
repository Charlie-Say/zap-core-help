Теги

Тег-это краткий текст, который вы хотите связать с запросом.
Запрос может иметь несколько тегов, связанных с ним.
Теги можно добавлять и изменять с помощью диалогового окна < 1 > Управление тегами< / 1>.
Их можно также автоматически добавить через<2 > пассивное сканирование < / 2> - это позволяет автоматически помечать страницы, соответствующие определенному регулярному выражению.

Tags are shown in the History tab and can be filtered on using the History Filter dialog.
