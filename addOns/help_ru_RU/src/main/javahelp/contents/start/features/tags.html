Теги

Тег - это короткий текст, который вы хотите связать с запросом. <0 /> С запросом может быть связано несколько тегов. <0 /> Теги можно добавлять и изменять с помощью диалогового окна Управление тегами истории . <0 /> Их также можно добавить автоматически с помощью пассивного сканирования - это позволяет автоматически помечать страницы, которые соответствуют определенному регулярному выражению.

Теги показаны на вкладке История и могут быть отфильтрованы с помощью диалоговое окно "Фильтр истории" .
