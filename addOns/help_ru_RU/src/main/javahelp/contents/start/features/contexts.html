Контексты

Контексты - это способ связать набор URL-адресов вместе. <0 /> Вы можете определить любые контексты, которые вам нравятся, но ожидается, что контекст будет соответствовать веб-приложение. <0 /> Рекомендуется определять новые контексты для каждого веб-приложения. которые составляют тестируемую вами систему, и устанавливайте их масштаб по мере того, как вы тестируете каждую из них.

Контексты определяются как набор регулярных выражений (регулярных выражений), которые применяются ко всем URL-адресам в Дерево сайтов . <1 /> Примечание. Регулярные выражения должны соответствовать всему URL-адресу.

Вы можете настроить контексты с помощью:

  • Щелкните элементы меню правой кнопкой мыши на вкладке "Сайты" или "История".
  • Диалоги контекстов сеанса .
