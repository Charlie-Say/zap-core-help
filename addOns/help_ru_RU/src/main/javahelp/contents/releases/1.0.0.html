Версия 1.0.0

В этой версии были сделаны следующие изменения:

Существенные изменения:

Справка

Справка была добавлена для всех функциональных возможностей, предоставляемых в настоящее время в ZAP.
Контекстно-зависимая справка также была выполнена так, что при нажатии клавиши F1, будет отображаться экран справки, связанный с вкладкой, которую вы выбрали.

Прерывания

Использование Контрольные точки было изменено, чтобы быть ближе к предоставляемым в IDE.
Элементы управления точкой останова были перенесены в новый Панель инструментов верхнего уровня.
Точки останова теперь также можно установить, щелкнув правой кнопкой мыши узлы в Сайты и Истории вкладки.

Версия отчета

В Версия отчета значительный изменилось и теперь включает в себя следующие пункты меню:
    Создать отчет в формате XML...Которая теперь генерирует отчеты по требованию
    Экспортировать сообщения в файл...Перемещен в меню файл
    Экспортировать ответ в файл...Перемещен в меню файл
    Экспортировать все URL-адреса в файл...Режим редактора
    Сравнить с другой сессией...Режим редактора

Улучшенный поиск в истории

Новая Искать позволяет вам искать для регулярных выражений во всех URL-адресов, запросов и ответов.
Следующей и предыдущей кнопки позволяют вам просматривать найденные Поиск терминов в Запрос и Вкладки ответов .

Фильтр истории

Панель инструментов фильтрации истории была добавлена которая позволяет ограничить, какие запросы отображаются.
Запросы могут быть отфильтрованы по HTTP методам, результат коды HTTP и любые ярлыки, оповещения, или заметки это было применено.

Заметки

Теперь можно связать заметки с любой просьбой.

В несколько страниц

Теперь можно связать несколько ярлыки с любой просьбой.

Больше контроля над предупреждениями

Диалоговое окно «Добавить предупреждение» позволяет добавить или изменить оповещение вручную.

Новый кодер/декодер

Новое диалоговое окно «Кодирование / декодирование / хеширование» заменило предыдущее. Кодер / декодер.

Пассивный сканер

Пассивный сканер был добавлен.
В этом выпуске ZAP будет использовать только пассивное сканирование для автоматического добавления теги .
В будущих выпусках это может также поднять предупреждения для потенциальных проблем.

Незначительные изменения:

Панель инструментов верхнего уровня

Панель инструментов верхнего уровня была добавлена, которая предоставляет кнопки настройки для часто используемых функций.

Оповещения нижний колонтитул резюме

Количество высоких, средних, низких и информационных предупреждений теперь отображаются в нижний колонтитул .

Новый внешний вид и

Используется новый внешний вид 'Nimbus'.

Обработка паролей прокси

Экран опций подключения есть новая возможность «Запрашивать учетные данные прокси-сервера при запуске».
Если выбран этот вариант, ваш пароль прокси не будет храниться на жестком диске.

Отправить на вкладке сайты

Вкладка Сайты есть новый пункт меню, вызываемый правой кнопкой мыши, который позволяет вам повторно отправить запрос после внесения в него любых изменений.

Каталог пользователей между сессиями

Последний каталог, используемый пользователем, теперь хранится в файле конфигурации, поэтому он поддерживается между сеансами.
