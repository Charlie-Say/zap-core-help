выпуск 2.5.0

В этот выпуск внесены следующие изменения:

Изменения ZAP API:

ПРОСМОТР авторизации / getAuthorizationDetectionMethod

Теперь возвращает данные, заключенные в объект с именем authorizationDetectionMethod. 
{"authorizationDetectionMethod":{"statusCode":"-1"..."headerRegex":""}}
вместо того: 
{"statusCode": "- 1" ... "headerRegex": ""}

ПРОСМОТР context / context + technologyList

Оба теперь возвращают данные, заключенные в объект под названием «контекст». 
{"context":{"id":"1", ..., "inScope":"true","loggedOutPattern":""}}
вместо того: 
{"id":"1", ..., "inScope":"true","loggedOutPattern":""}

Паук ДЕЙСТВИЯ / сканирование + scanAsUser

Оба теперь поддерживают новый необязательный параметр subtreeOnly, который ограничивает паука указанным поддеревом. Параметр url теперь также является необязательным, если указан допустимый параметр context.

Новый компонент "статистика"

Новый компонент API статистики предоставляет доступ к Статистика в настоящее время поддерживается ZAP.

Обратите внимание, что некоторые существующие компоненты также будут иметь новые операции, см. API Web UI (Веб-интерфейс API ) больше подробностей.
