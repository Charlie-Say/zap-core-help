выпуск 2.4.2

В этот выпуск внесены следующие изменения:

Улучшения:

  • Проблема 1306. Флаг командной строки Java PermSize удален в Java 8.
  • Проблема 1593: автоматическая прокрутка на вкладке "Паук".
  • Проблема 1600: не сообщать о предупреждении X-Frame-Options на страницах 403 и 404
  • Проблема 1654: httpSessions / createEmptySession должен инициализировать сайт, который ранее не посещался.
  • Проблема 1702: добавление опции "рекурсивный" в API-интерфейс паука.
  • Проблема 1715: невозможно передать аргументы при запуске ZAP из командной строки в Mac OS X
  • Проблема 1766: удаление контекста через API
  • Проблема 1768: обновление для использования более свежего агента пользователя.
  • Проблема 1778: запросы паука AJAX пассивного сканирования
  • Проблема 1790: перемещение сканера переполнения буфера из бета-версии в выпуск
  • Проблема 1793: разрешить активным сценариям сканирования проверять, было ли сканирование остановлено.
  • Проблема 1795: Разрешить настройку параметров JVM через графический интерфейс
  • Проблема 1799: Незначительный запрос функции: Разрешить вставку URL-адреса в диалоговое окно запуска Spider.
  • Проблема 1802: Незначительное улучшение: изменение активной кнопки паузы на кнопку воспроизведения
  • Проблема 1849: Возможность объединения связанных проблем в отчетах.
  • Проблема 1857: обновленные библиотеки.
  • Проблема 1865: увеличение максимального размера в БД.

Исправление ошибок:

  • Проблема 1760: Невозможно инициализировать домашний каталог! xml / config.xml (такого файла или каталога нет)
  • Проблема 1763: автоматическая проверка обновлений не позволяет сообщать о новых версиях.
  • Проблема 1770: исключения при вызове (некоторых) действий контекстного API в режиме демона
  • Проблема 1771: для OSX файл zap.sh в основной загрузке жестко кодирует относительное расположение java.
  • Проблема 1772: В OS X найденная версия Java лежит
  • Проблема 1777: «Не удается найти источник конфигурации null.policy» после открытия диалогового окна «Активное сканирование».
  • Проблема 1781: ошибки ZAP с «Неподдерживаемым параметром '-psn_x_xxxxxxx'» в OS X
  • Проблема 1784: исключение NullPointerException при активном сканировании через API с целью без схемы
  • Проблема 1785: подключаемый модуль включен, даже если зависимостей нет, активное сканирование "зависает".
  • Проблема 1787: Контекст не используется пауком, даже если он выбран.
  • Проблема 1788: панель хода сканирования требует изменения сортировки
  • Проблема 1789: сообщения принудительного просмотра / AJAX Spider не восстанавливаются на вкладке "Сайты"
  • Проблема 1792: отчет не создается в режиме демона.
  • Проблема 1798: Функция Stop Attack блокирует ZAP?
  • Проблема 1804: отключение обработки внешних сущностей XML по умолчанию
  • Проблема 1805: API ZAP может не возвращать ответ в запрошенном формате при ошибках.
  • Проблема 1858: Паук мог сообщать о неверном прогрессе после завершения.
  • Проблема 1872: доступ к EDT в режиме демона
