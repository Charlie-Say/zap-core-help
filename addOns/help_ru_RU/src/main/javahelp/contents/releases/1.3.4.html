выпуск 1.3.4

В этот выпуск внесены следующие изменения:

Незначительные изменения:

Проблема 146: обратное регулярное выражение при поиске и выделение нечеткого соответствия

Проблема 202: возможность отключить рекурсию грубой силы (brute force)

Проблема 215: Разрешить легкое добавление пользовательских файлов грубой силы

Также добавлена возможность установить файл грубой силы(brute force) по умолчанию.

Проблема 217. Вызов приложений - добавление поддержки файлов cookie и параметров публикации.

Проблема 218. Разрешить пользователям легко добавлять свои собственные файлы фаззера.

Также добавлена возможность добавления вывода к примечанию, относящемуся к соответствующей записи.

Исправление ошибок:

Проблема 56: отключить запросы POST в Spider

Проблема 186: Параметры подключения - запрашивать учетные данные прокси-сервера при запуске / проверка адреса не пуста

Проблема 188: Проблема с обновлением ZAP в Linux и Windows

Проблема 191: исключение, когда URL-адрес содержит экранированные символы

Проблема 196. Несколько диалоговых окон одного и того же параметра, открытых одновременно, не работают должным образом.

Проблема 199: Уязвимости с усеченным текстом

Проблема 204: поиск по заголовкам находит регулярное выражение только в запросах.

Проблема 206. Исключение на вкладке «Предупреждения» при выборе параметра всплывающего окна.

Проблема 214: Нет предупреждающего сообщения при сохранении отчета в месте только для чтения

Проблема 216: исключение, когда URI не имеет компонента пути

Проблема 219. Разрыв и игнорирование URL-адресов по умолчанию включает GET / POST.

Проблема 220: неправильное сообщение: пароль (хранится в виде открытого текста)
