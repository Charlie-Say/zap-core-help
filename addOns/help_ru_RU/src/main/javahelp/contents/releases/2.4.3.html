выпуск 2.4.3

В этот выпуск внесены следующие изменения:

Улучшения:

  • Проблема 1576: Определите элементы пути URL как «неструктурные».
  • Проблема 1711: Запрос функции: возможность указать количество генерируемых токенов.
  • Проблема 1768: обновление для использования более свежего агента пользователя.
  • Проблема 1894: ZAP сканирует только заголовки запросов «User-Agent», «Referer» и «Host».
  • Проблема 1911: поиск запросов паука AJAX
  • Проблема 1913: передать все параметры в zap.sh - пакет debian
  • Проблема 1914: несколько каталогов надстроек.
  • Проблема 1920: сообщить хосту: порт ZAP прослушивает в режиме демона или выйти, если не может
  • Проблема 1927: Улучшение: Прерывание только для URL-адресов в области видимости.
  • Проблема 1944: диаграмма ответов в секунду при выполнении сканирования.
  • Проблема 1953: разрешить сканирование контекста через ZAP API.
  • Проблема 1962: установка и обновление надстроек из командной строки.
  • Проблема 1975: комплект шифров RC4-SHA SSL не поддерживается
  • Проблема 1980: добавление ZAP CLI в образы Docker
  • Проблема 1985: Вызов диалогового окна Modify, если пользователь удваивает строку в AbstractMultipleOptionsBaseTablePanel
  • Выпуск 2009: добавление интерактивных ссылок на страницы часто задаваемых вопросов и информационных бюллетеней.
  • Проблема 2084: предупреждать пользователей, если они, вероятно, используют устаревшие версии.
  • Проблема 2086: количество запросов отчетов на подключаемый модуль
  • Проблема 2088: Поддержка кнопки «обновить все» для установки всех обновленных надстроек.
  • Проблема 2094: Поддержка параметра командной строки -addoninstallall
  • Проблема 2102: разрешить установку параметров паука ajax через API

Исправление ошибок:

  • Проблема 1070: нарушение пользовательского URL-адреса с параметрами запроса не работает.
  • Проблема 1555: исключение SAXParseException при создании отчета.
  • Проблема 1617: ZAP 2.4.0 выдает исключения HeadlessExceptions при работе в режиме демона на автономном компьютере
  • Проблема 1877: исправлен путь к .ZAP_JVM.properties в zap.sh
  • Проблема 1893: регресс: невозможно отключить и включить определенные сканеры через API
  • Проблема 1910: API не возвращает содержимое в ожидаемой кодировке при ошибках.
  • Проблема 1917: на вкладках "Запрос" и "Ответ" текст не масштабируется.
  • Проблема 1939: Сценарии - проблема включения кнопки сохранения
  • Проблема 1949: сценарий с отсутствующим типом предотвращает загрузку сценариев и шаблонов.
  • Проблема 1950: соединение закрыто без ответа, сценарий аутентификации с ошибками
  • Проблема 1951: исключение при попытке добавить пользователей перед загрузкой сценария аутентификации
  • Проблема 1960: в диалоговом окне активного сканирования может использоваться устаревшая политика сканирования.
  • Проблема 1969: Проблемы с установкой сканеров.
  • Проблема 1970: зависимости установленных надстроек могли не учитываться при установке надстроек.
  • Проблема 1973: возвращенный HAR / список не содержит правильных сообщений перенаправления
  • Проблема 1981: Паук мог не сообщать правильное количество найденных URI
  • Проблема 2005: активное сканирование некорректно выполнялось на дочерних узлах
  • Проблема 2044: Проблемы в реализации алгоритма Хиршберга
  • Проблема 2045: не копируйте старые конфигурации, если использовалась опция -dir
  • Проблема 2052: изменения аутентификации, выполненные через API, не сохраняются в сеансе.
