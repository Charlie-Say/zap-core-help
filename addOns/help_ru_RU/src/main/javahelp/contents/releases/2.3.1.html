выпуск 2.3.1

В этот выпуск внесены следующие изменения:

  • Проблема 81: ZAP изменяет данные запроса (при переключении представлений)
  • Проблема 377: невыполненные зависимости приводят к зависанию активного сканирования
  • Проблема 1073: невозможно удалить сценарии, помеченные как «загрузка при запуске».
  • Проблема 1114: core.newSession не очищает сайты
  • Проблема 1155: вкладка "Исторический запрос" не позволяет изменять форматирование.
  • Проблема 1156: прокси-декодер gzip не обновляет длину содержимого в заголовках ответа.
  • Проблема 1163: невозможно задать домашний каталог с пробелом в командной строке
  • Проблема 1166: избыточные индексы в zapdb.script
  • Проблема 1168: добавлена поддержка прокси-сервера для кодирования содержимого "deflate".
  • Проблема 1170: контекстное меню «Паук» / всплывающие меню пользователя больше не отображаются.
  • Проблема 1179: Невозможно выбрать 2 запроса в результатах нечеткости (Ctrl + щелчок)
  • Проблема 1181: активные страницы с уязвимостями сканируются только один раз.
  • Проблема 1182: оповещения одного типа для разных параметров одной и той же уязвимой страницы отображаются только один раз в дереве «Оповещения».
  • Проблема 1183: исключение NullPointerException при выборе узла на вкладке «Предупреждения» после удаления сообщения.
  • Проблема 1191: параметры сеанса командной строки не действуют
  • Проблема 1193: сканировать элементы пути URL - отключено по умолчанию.
  • Проблема 1194: аргументы командной строки не передаются расширениям при запуске ZAP в режиме демона
  • Проблема 1196: AbstractPlugin.bingo неправильно устанавливает доказательства для атаки.
  • Проблема 1202: проблема с загрузкой надстроек, которые не были правильно инициализированы.
  • Проблема 1203: Сценарий аутентификации Wordpress
  • Проблема 1206: вкладка «История» не очищается при создании нового сеанса через API с ZAP в режиме графического интерфейса.
