Версия 1.0.0

В этой версии были сделаны следующие изменения:

Существенные изменения:

OWASP ребрендинг

Зап была принята как OWASP проект.
Its homepage is now: https://owasp.org/www-project-zap/

Грубой силы

Возможность подбора файлов и каталогов на основе кода из проекта OWASP DirBuster.
На новой вкладке грубой силы показывает файлы и каталоги, нашли.

Сканирование портов

Возможность сканирования сайтов порт.
На новой вкладке сканирование портов показывает портов найдено.

Активное сканирование сайта

Новый Активное сканирование сайта показывает запросы и ответы в результате активно сканирование сайта.

Обойти Url пауком

В Обойти Url пауком теперь позволяет продолжить использование зап во время индексации сайта.
Также можно приостановить и возобновить паука.

Поддержка смарт-карт

З Andiparos проекта была добавлена поддержка смарт-карт.
Следующие устройства смарт-карт были протестированы на Windows:
    ГарантииAladdin eToken
    Aladdin eToken
    OmnikeyOmnikey 3121
    CardMan 6121
    GemaltoРефлекс 20 V2
    Швейцарский Stick
Следующие устройства смарт-карт были протестированы на Windows:
    OmnikeyCardMan 4040

Меню атаки

Новый Закладка сайты щелкните правой кнопкой мыши меню «Атака» позволяет начать различных проверок.

Интернационализация

Все основные вкладки и элементы меню теперь интернационализированным.

Локализация

В эту версию встроена поддержка следующих языков:
    АнглийскийЯзык по умолчанию
    Португальский (Бразилия)
    Немецкий
    Польский/Polish
    Испанский

Направление языка

При запуске вам будет предложено выбрать язык для использования.
Новые языки распознаются автоматически на наличие файлов с именами вида Messages_ <locale>.properties в каталоге зап.

Незначительные изменения:

Отключенные плагины с файлами по умолчанию

Плагины, которые обнаружить файлы по умолчанию эффективно увольняют по новой грубой силы сканера.
Поэтому они были отключены.

Сводные данные сканера

Количество количество и типы текущего сканирования теперь отображаются в нижний колонтитул.
