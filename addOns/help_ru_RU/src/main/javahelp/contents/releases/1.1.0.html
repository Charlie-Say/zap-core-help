Версия 1.0.0

В этой версии были сделаны следующие изменения:

Существенные изменения:

OWASP ребрендинг

ZAP был принят как проект OWASP.
Сейчас его домашняя страница: https://owasp.org/www-project-zap/

Brute Force (Грубая Сила)

Возможность подбора файлов и каталогов на основе кода из проекта OWASP DirBuster.
На новой вкладке грубой силы (Brute Force) показаны найденные файлы и каталоги.

Сканирование портов

Возможность сканирования сайтов.
Новая вкладка Port Scan показывает найденные порты.

Вкладка "Активное сканирование"

Новый Активное сканирование сайта показывает запросы и ответы в результате активно сканирование сайта.

Вкладка паук

Вкладка паук теперь позволяет вам продолжать использовать ZAP при просмотре сайта.
Вы также можете приостановить и возобновить работу паука.

Поддержка смарт-карт

Добавлена поддержка смарт-карт через проект Andiparos.
Следующие устройства со смарт-картами были протестированы в Windows:
    Гарантия Aladdin eToken
    Aladdin eToken
    OmnikeyOmnikey 3121
    CardMan 6121
    GemaltoРефлекс 20 V2
    Swiss Stick
Сообщается, что следующие устройства смарт-карт работают:
    OmnikeyCardMan 4040

Меню атаки

Новая Закладка сайты щелкните правой кнопкой мыши меню «Атака» позволяет запускать различные проверки.

Больше интернационализации

Все основные вкладки и пункты меню теперь интернационализированы.

Локализация

В эту версию встроена поддержка следующих языков:
    АнглийскийЯзык по умолчанию
    Португальский (Бразилия)
    Немецкий
    Польский/Polish
    Испанский

Направление языка

При запуске вам будет предложено выбрать язык для использования.
Новые языки автоматически обнаруживаются по наличию файлов с именами вида Messages_ <locale>.свойства в каталоге ZAP.

Незначительные изменения:

Отключены плагины для файлов по умолчанию

Плагины, которые обнаруживают файлы по умолчанию, эффективно заменяются новым brute force сканером.
Поэтому они были отключены.

Резюме сканера

Количество и типы текущих сканирований теперь отображаются в нижний колонтитул.
