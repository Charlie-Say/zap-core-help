выпуск 2.4.1

В этот выпуск включены важные исправления безопасности - пользователям настоятельно рекомендуется выполнить обновление как можно скорее.

Одно из изменений означает, что ключ API создается по умолчанию, а это означает, что любые приложения, использующие ZAP API, не будут работать, если они не будут обновлены для использования этого ключа.
Ключ API можно найти в Экран параметров API
Вы также можете установить его из командной строки, используя такую опцию: 

-config api.key=change-me-9203935709

Подробнее см. https://www.zaproxy.org/faq/why-is-an-api-key-required-by-default/
