Незначительные изменения:

Проблема 744: разрешить перезапись пользовательских файлов фаззинга.

Проблема 795: разрешить настройку сканируемых типов параметров через пользовательский интерфейс.

Ошибка 791: сохраненные сеансы сбрасываются при выходе из ZAP.

Ошибка 792: диалог Diff не работает при вызове из Zest.

Проблема 794: Исключение искаженной байтовой последовательности
