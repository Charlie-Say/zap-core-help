Выпуск 1.3.0

В этом выпуске были внесены следующие изменения:

Важные перемены:

Fuzzing

Строки в ответе теперь могут быть заполнены, чтобы попытаться найти уязвимости.
Анти маркеры CRSF можно обнаружить и автоматически восстанавливаются при фаззинг.
Эта функциональность основана на коде из проекта OWASP JBroFuzz.

Динамические SSL-сертификаты

Поддержка SSL-соединений была улучшена и упрощена.
Теперь пользователь может создать свой собственный корневой сертификат и распространить его на свои HTTP-клиенты.
Прочитайте Вариант: динамические сертификаты SSL более подробный.

Режим демона и API

Начиная зап с "демона" в командной строке, чтобы запустить в фоновом режиме в безголовый режим, это означает, что пользовательский интерфейс не отображается.
Начальный API был реализован в XML, JSON и HTML.
Если зап запущен в качестве демона, то API включена автоматически, в противном случае API должен быть включен через экран АПИ функции.
Для навигации по API можно открыть http://zap / в вашем браузере при проксировании через ZAP.

Интеграция с BeanShell

The BeanShell это интерактивный Шелл java, которая может быть использована для выполнения beanshell скрипты.
Эти сценарии являются упрощенной формой Java, которые используют много элементов из синтаксиса Java, но в более простом формате сценариев.
Все java-код и валидный код состоит.
Интеграция состоит в список зап позволяет писать скрипты с использованием зап функции и набор данных.
Это может быть очень мощным инструментом для анализа веб-приложений.

Minor changes:

Hex view

The Запрос, Ответ вкладки теперь обеспечивают Опция "Hex View", которая будет отображать тела запроса и ответа в шестнадцатеричном формате.

Search results

The Search tab теперь отображаются Все экземпляры найденной строки, а не только первый экземпляр в каждом запросе или ответе.

Exclude URLs

Url-адреса могут быть явно исключены из активного сканера, spider и из прокси-сервера.

Поддержка копирования

Большинство панелей теперь предоставляют опцию меню "правой кнопкой ""копировать", включая панели сканирования портов и грубой силы.

Undo/Redo support

Все поля ввода теперь поддерживают действия отмены / повтора с использованием ускорителей отмены/повтора по умолчанию для операционных систем:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Поддержка прокси-сервера сканера портов и опция тайм-аута

Теперь сканер портов может использовать исходящий прокси (если он настроен), а также можно задать время ожидания в миллисекундах.

Request and response break buttons

Теперь есть 2 "Set Break" кнопки, чтобы позволить перерывы должны быть установлены на все запросы и все ответы независимо друг от друга.

Разверните вкладки сайты и информация

Теперь есть 2 кнопки, которые позволяют переключаться между сайтами и информацией вкладки расширенные.

Break tab icon changes colour when breakpoint hit

While the Break tab is not in use its icon is a grey cross:    .

Adjustable timeout

The Option: Connection экран позволяет установить тайм-аут в секундах чтобы упростить тестирование медленных приложений.

Library updates

Большинство библиотек, используемых ZAP, обновлены до последних версий.

A new icon :)

Благодаря Саймон Эгли и все другие для представления прохладно предложения значок.

Known issues:

Mac OS X: Dynamic SSL and Google Chrome

Currently Dynamic SSL is not working when using Google Chrome. This is because of an unresolved known issue with Google Chrome and Mac OS X Keychain. При импорте корневого ЦС OWASP ZAP в связку ключей и запросе веб-сайта SSL отображается сообщение об ошибке "недопустимый сертификат".
