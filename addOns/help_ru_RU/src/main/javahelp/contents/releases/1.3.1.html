Проблема 110. Выбор языка пользователя по умолчанию при первом запуске

Проблема 121: отчет сеанса сравнения javascript сломан в Firefox 3

Проблема 123: не удается выбрать сайт https из активного окна сканирования, если theres эквивалент http

Проблема 127: функция маркера CSRF не работает в fuzz - Добавлено пояснение справки для частичного исправления

Ошибка SSL, которая не позволяла SSL-подключениям устанавливаться в любом браузере

Known issues:

Mac OS X: динамический SSL и Google Chrome

В настоящее время динамический SSL не работает при использовании Google Chrome. Это из-за нерешенной известной проблемы с Google Chrome и Mac OS X Keychain. При импорте корневого ЦС OWASP ZAP в связку ключей и запросе веб-сайта SSL отображается сообщение об ошибке "недопустимый сертификат".
