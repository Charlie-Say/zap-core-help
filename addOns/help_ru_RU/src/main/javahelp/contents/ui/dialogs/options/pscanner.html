Окно опций пассивного сканера

Этот экран позволяет настроить пассивный сканер .

Параметры конфигурации

Поле Детали По умолчанию Файл конфигурации
Сканировать только сообщения в области действия Устанавливает, должно ли пассивное сканирование выполняться только для сообщений, находятся в области действия. Отменен выбор Ключ: pscans.scanOnlyInScope
Значения: true или false
Включить трафик из фаззера при пассивном сканировании Устанавливает, должно ли выполняться пассивное сканирование сообщений, сгенерированных фаззером. Отменен выбор Ключ: pscans.scanFuzzerMessages
Значения: true или false
Макс предупреждает, что любое правило может поднять Устанавливает максимальное количество предупреждений, которые должно создавать правило пассивного сканирования. Это может быть немного превышено из-за резьбы. Этот параметр обычно полезен только для автоматического сканирования. Правила сканирования, превышающие это значение, будут отключены. и его необходимо будет включить вручную, если будет запущен новый сеанс. 0 (не установлено) Ключ: pscans.maxAlertsPerRule
Значения: 0: не задано или максимальное количество предупреждений
Максимальный размер тела в байтах для сканирования Устанавливает максимальный размер запроса или размера тела ответа в байтах, который будет сканировать пассивный сканер. Это можно использовать, если правила пассивного сканирования слишком долго сканируют очень большие запросы или ответы. Если установлено, количество проигнорированных запросов и ответов фиксируется в статистике с помощью клавиш stats.pscan.reqBodyTooBig и stats.pscan.respBodyTooBig соответственно. 0 (не установлено) Ключ: pscans.maxBodySizeInBytes
Значения: 0: не установлен или максимальный размер тела в байтах
Clear Queue Empties the passive scan queue without passively scanning the messages. Currently running rules will run to completion but new rules will only be run when new messages are added to the queue.
