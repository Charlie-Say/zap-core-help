Экран статистики параметров

На этом экране можно настроить статистику. который поддерживает ZAP.

Статистика в памяти включена

По умолчанию ZAP хранит статистику в памяти. Чтобы отключить статистику в памяти, снимите этот флажок.

Статистический сервер включен

Установите этот флажок, чтобы настроить адрес сервера Statsd, на который ZAP будет отправлять свою статистику.

Имя хоста Statsd

Хост, на котором работает сервер Statsd.

Порт Statsd:

Порт, который прослушивает сервер Statsd, по умолчанию 8125.

Префикс Statsd:

Префикс ZAP будет добавляться к статистике, которую он отправляет на сервер Statsd.
