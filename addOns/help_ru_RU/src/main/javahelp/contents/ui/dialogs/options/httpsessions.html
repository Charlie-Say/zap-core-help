Варианты экрана HTTP Sessions

Этот экран позволяет настроить HTTP Расширения сеансов HTTP опции:

Токены по умолчанию

Это имена файлов cookie, которые по умолчанию считаются токены сеанса. Как только они обнаруживаются в качестве параметров на Сайте, они автоматически добавляются в качестве токенов сеанса для этого Сайта, если только вручную переопределяется командой пользователя.

Токены сеанса используются без учета регистра, поэтому токены сеанса по умолчанию автоматически будут строчными. Кроме того, любой модификации токенов сеанса по умолчанию видны только после повторное посещение страниц Сайта.

Включить только для прокси

Если эта опция включена, расширение будет обрабатывать только сообщения отправлено и получено через прокси (например, из/в браузер). То сообщения от других расширений обрабатываются, только если эта опция включена. инвалид.
