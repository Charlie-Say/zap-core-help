Экран предупреждений о параметрах

На этом экране можно настроить параметры предупреждений:

Объединить связанные оповещения в отчете

Если этот флажок установлен, связанные проблемы будут объединены во все созданные отчеты.
Это значительно уменьшит размер отчета, так как дублирующаяся информация будет удалена.

Максимальное количество экземпляров предупреждений в отчете

Максимальное количество экземпляров предупреждений для включения в отчет.
Нулевое значение считается неограниченным.

Файл переопределения предупреждений

Полное имя файла свойств, в котором указаны любые предупреждения переопределяют что вы хотите подать заявку.
