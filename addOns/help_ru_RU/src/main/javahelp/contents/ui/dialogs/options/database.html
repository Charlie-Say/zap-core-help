Экран базы данных параметров

Этот экран позволяет настроить параметры базы данных:

Компактный (на выходе)

Позволяет сжимать базу данных при закрытии ZAP, сжимая базу данных обеспечивает минимальное использование дискового пространства, но также потребуется больше времени для выхода из ZAP.

Журнал восстановления

Определяет, включен ли журнал восстановления базы данных.
Улучшает производительность базы данных при отключении, но может привести к потере данных при резком выходе из ZAP.
Примечание: текущий сеанс не изменится, изменения вступят в силу для новых и открытых сеансов.

Максимальный размер тела запроса

Максимальный размер тела запроса в байтах, который разрешен ZAP.

Максимальный размер тела отклика

Максимальный размер тела ответа в байтах, допустимый ZAP.

Запрашивать параметры сохранения в новом сеансе

Если отмечено, то Диалоговое окно сохранения сеанса будет отображаться при создании нового сеанса.

Вариант по умолчанию

Параметр по умолчанию для новых сеансов — если указанный выше запрос включен, будет выбран параметр по умолчанию.
Если подсказка не включена, то будет предпринято это действие.
