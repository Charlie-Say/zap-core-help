Экран «Параметры пассивного сканирования»

Этот экран позволяет настроить теги добавленные пассивным сканером .

Вы можете добавлять, изменять и удалять теги с помощью соответствующих кнопок.

Интерполяция тегов

Чтобы обеспечить некоторую гибкость в создании тегов и содержании, пользователь может указать регулярное выражение с группами захвата. который будет использоваться для замены идентификаторов групп (например, $1) в результирующем теге.

Например, тег можно определить следующим образом:
ПолеЗначениеПримечание
Имя:Тест 
Тег:$1-SOMETAG 
Регулярное выражение тела ответа: (\d{3})Найдите строки из 3 чисел, используя группу захвата (круглые скобки).
Результат: Когда 3-значные строки пассивно идентифицируются в содержимом тела ответа (соответствует регулярному выражению), сообщения будут тегами, такими как «123-SOMETAG», «345-SOMETAG» и т. д.
