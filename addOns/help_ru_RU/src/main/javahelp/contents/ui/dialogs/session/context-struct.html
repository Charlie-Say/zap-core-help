Экран структуры контекста сеанса

Это один из Экраны контекста сеанса что позволяет вам управлять пониманием ZAP структуры приложения.

Разделители пар ключ-значение URL-адреса

Символы, разделяющие пары ключей и значений в URL-адресах, по умолчанию - "&".

Разделители значений ключа URL

Символы, разделяющие ключи и значения в URL-адресах, по умолчанию '='.

POST Разделители пар ключ-значение

Символы, разделяющие пары ключей и значений в данных POST, по умолчанию '&'.

POST Разделители значений ключей

Символы, разделяющие ключи и значения в данных POST, по умолчанию '='.

Структурные модификаторы

Структурные модификаторы применимые к этому контексту, которые могут быть:
  • Контент, управляемый данными , который определяет пути URL, которые представляют данные
  • Структурные параметры , которые определяют параметры, которые представляют структуру приложения, а не данные пользователя.
