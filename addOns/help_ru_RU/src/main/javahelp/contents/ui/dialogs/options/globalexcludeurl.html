Опции Global Exclue URL-экран

Этот экран позволяет настроить Global Исключить URL Regexs в ZAP:

Регулярное выражение

Регулярное выражение URL, который ZAP будет игнорировать. Нажмите кнопки Добавлять (Add) , Modify (Модифицировать) , или Удалить (Remove), чтобы редактировать каждое regex (регулярное выражение).

Включено

Если коробка (box) проверяется, то этот URL Regex (регулярное выражение ) будет игнорироваться ZAP.
