Экран «Параметры точек останова»

Этот экран позволяет настроить точка останова ( breakpoint ) опции

Подтвердить отбрасывание захваченного сообщения

Запрашивает подтверждение, когда запрос или ответ отбрасываются.

Прерывать только сообщения в области действия

«Глобальная» точка останова будет запускаться только запросами, которые находятся в область применения.
Пользовательские точки останова не будут затронуты.

ZAP всегда сверху (top), когда срабатывает точка останова

Если выбрано, это будет означать, что ZAP остается поверх всех других окон всякий раз, когда срабатывает точка останова.

Показать кнопки для выбора запросов, которые вы не хотите прерывать в ZAP.

Это определяет, будут ли кнопки для выбора типов файлов, на которые вы не хотите прерывать ZAP, и фильтр области отображаются на панели разрыва.

Регулярное выражение URL-адреса Javascript

Это устанавливает регулярное выражение, используемое для прогнозирования, будет ли содержимое ответа файлом Javascript.

Регулярное выражение URL CSS и шрифтов

Это устанавливает регулярное выражение, используемое для прогнозирования, будет ли содержимое в ответе файлом CSS или файлом шрифта.

Регулярное выражение мультимедийного URL

Это устанавливает регулярное выражение, используемое для прогнозирования, будет ли содержимое в ответе мультимедийным файлом (изображениями, видео или звуками).

Перерыв в режиме кнопок

Это определяет, есть ли одна комбинированная кнопка прерывания на Панель инструментов верхнего уровня или отдельные для запросов и ответов.
