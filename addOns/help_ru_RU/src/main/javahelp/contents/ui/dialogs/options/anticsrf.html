Экран параметров Anti CRSF Tokens

Этот экран позволяет вам настроить параметры анти-токенов CSRF :

Токены

Имена параметров POST формы, которые следует рассматривать как токены против CSRF.

Частичные совпадения

Определяет, следует ли рассматривать имена параметров POST формы как токены против CSRF, если они включают один из токенов (вместо того, чтобы требовать точного совпадения).
