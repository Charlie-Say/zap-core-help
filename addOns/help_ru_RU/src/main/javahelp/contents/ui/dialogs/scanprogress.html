Диалог выполнения сканирования

Это показывает вам статус активное сканирование .

Вкладка "Прогресс"

Это показывает, какие правила сканирования выполняются для каждого сканируемого хоста, а также другие детали, такие как истекшее время их работы и количество запросов, сделанных для каждого правила.
Это также позволяет вам пропустить правило, которое в настоящее время выполняется, нажав на Кнопка "Пропустить текущее активное сканирование" в столбце Статус.

Вкладка "Диаграмма ответов"

Это показывает количество ответов, получаемых ZAP в секунду при активном сканировании сайта.
Ответы заносятся в диаграмму по "группе кодов статуса HTTP-ответа":
  • 1xx Информационный
  • 2xx Успех
  • 3xx перенаправление
  • Ошибка клиента 4xx
  • Ошибка сервера 5xx
Вы можете увеличить масштаб, выделив область диаграммы с помощью мыши.
Вы также можете щелкнуть диаграмму правой кнопкой мыши, чтобы увидеть дополнительные параметры.
Вертикальные полосы указывают приблизительно, когда начинается каждое правило сканирования (очень сложно быть точным).
Диаграмма будет обновляться только при открытии, если вы закроете, а затем снова откроете диаграмму, предыдущие данные будут потеряны.
По умолчанию максимальное время, которое будет отображаться на графике, составляет 10 минут, вы можете изменить это значение с помощью Параметры экрана активного сканирования .
