Окно параметров JVM

Этот экран позволяет настроить сочетания клавиш для всех меню ZAP.

Щелчок по любой из строк таблицы вызовет диалоговое окно, в котором можно изменить сочетания клавиш.

Памятка «Действие»

При нажатии на эту кнопку будет создана HTML-страница, на которой показаны все сочетания клавиш, отсортированные по «действию».
Эта кнопка доступна только в том случае, если включен API и ваша операционная система поддерживает открытие URL-адресов в браузере.

Шпаргалка «Ключ»

При нажатии на эту кнопку будет создана HTML-страница, на которой показаны все сочетания клавиш, отсортированные по «клавише».
Эта кнопка доступна только в том случае, если включен API и ваша операционная система поддерживает открытие URL-адресов в браузере.

Перезагрузить

Нажатие на эту кнопку вернет все ярлыки к значениям по умолчанию.
