Диалоговое окно "Фильтр истории"

Этот диалог позволяет вам ограничить, какие запросы будут отображаться на вкладке История . .

Поля

Диалог имеет следующие поля:

Методы

Методы HTTP для фильтрации.

Коды

Коды ответа HTTP для фильтрации.

Теги

Теги фильтровать.

Оповещения

Типы Оповещения фильтровать.

Примечания

Будь то наличие или отсутствие Примечания следует отфильтровать.

URL с регулярным выражением

Список шаблонов регулярных выражений 'include' (по одному на строку) - если он указан, то будут включены только те URL-адреса, которые соответствуют этим шаблонам.

URL без регулярного выражения

Список шаблонов регулярных выражений «исключить» (по одному на строку) - если он указан, то любые URL-адреса, соответствующие этим шаблонам, будут скрыты.
