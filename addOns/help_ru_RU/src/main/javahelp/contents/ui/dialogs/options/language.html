Экран параметров языка

Языковой экран позволяет настроить:

Выбор языка

Язык, который ZAP использует для отображения.
В ZAP встроен набор языков, однако после последней версии может быть доступно больше языков.
Проверьте страницу загрузки ZAP на наличие последнего доступного языкового пакета.

Использовать локаль системы для форматирования

Если форматирование чисел, дат и времени выполняется с использованием локали системы вместо локали выбранного языка.

Импорт языкового пакета

Помимо выбора языка, используемого для отображения ZAP, инструмент импорта позволяет импортировать новые языковые пакеты или обновления языковых файлов. Пакет языковых файлов всегда будет иметь расширение zaplang. (E.g. language-pack-en_GB.zaplang)
После импорта вы сразу же сможете выбрать новый импортированный язык в раскрывающемся меню выбора языка в верхней части экрана.
Обратите внимание, что вам нужно будет перезапустить ZAP, чтобы окончательно применить новый язык.
