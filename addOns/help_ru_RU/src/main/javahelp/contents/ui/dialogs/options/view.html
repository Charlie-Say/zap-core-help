Экран дисплея параметров

Экран дисплея позволяет настроить:

Изображения

Если ZAP обрабатывает изображения.

Показать (локальные) запросы CONNECT

Если запросы HTTP CONNECT, полученные Локальные прокси должны сохраняться в текущем сеансе и показано в Вкладка История . По умолчанию чтобы не показывать эти запросы, они происходят часто (при подготовке к TLS/SSL, WebSocket... соединений) и, если не проверять поведение клиентского приложения, не представляют (обычно) большого интереса.

Отображение

Компоновка 3-х основных панелей.
Доступны следующие варианты:
  • Развернуть левую вкладку (Сайты) — панель «дерево», содержащая вкладку «Сайты», расширяется на всю длину левой стороны. Это уменьшит количество место, доступное для панели «информация».
  • Разверните нижние вкладки (История и т. д.) — панель «Информация» расширяется для на всю длину дна. Это уменьшит количество места, доступного для панель "дерево".
  • Полный макет — выбранная вкладка занимает весь экран. Это полезно при использовании ZAP на маленьких экранах.

Положение панели реагирования

Позволяет настроить положение вкладки «Ответ» относительно вкладки «Запрос».
Доступны следующие варианты:
  • Вкладки рядом — вкладки «Запрос» и «Ответ» расположены рядом. Этот увеличивает информацию, которая может быть отображена, но означает, что вы не можете видеть как запрос и ответ одновременно.
  • Бок о бок на вкладке — панели «Запрос» и «Ответ» отображаются рядом в та самая вкладка. Это уменьшает информацию, которая может быть отображена, но означает, что вы можете видеть и запрос, и ответ одновременно.
  • Панели рядом — панель «Запрос» отображается слева от панели «Ответ». панель. Это уменьшает информацию, которая может быть отображена, но означает, что вы можете видеть как запрос и ответ одновременно.
  • Запрос отображается над ответом — панель «Запрос» отображается над ответом. панель. Это уменьшает информацию, которая может быть отображена, но означает, что вы можете видеть как запрос и ответ одновременно.
Параметр «Положение панели ответа» не применяется, если для параметра «Отображение» установлено значение «Полный макет».

Показать кнопки перерыва

Расположение кнопок прерывания.

Минимальный размер просмотра большого запроса

Минимальный размер тела запроса в байтах, при котором вместо фактического тела будет отображаться сообщение «большое тело запроса».
Это делается для того, чтобы очень большие тела не замедляли работу пользовательского интерфейса.
Установка этого значения на -1 приведет к тому, что запрос всегда будет отображаться, независимо от его размера.

Минимальный размер большого представления ответа

Минимальный размер тела ответа в байтах, при котором вместо фактического тела будет отображаться сообщение «большое тело ответа».
Это делается для того, чтобы очень большие тела не замедляли работу пользовательского интерфейса.
Установка этого значения на -1 приведет к тому, что ответ всегда будет отображаться независимо от его размера.

Вкладки вывода Параметры штампа времени

После того, как вы включили метки времени на вкладке вывода, вы можете настроить формат, в котором вы хотите, чтобы эти метки времени отображались. Либо выберите предопределенный формат из раскрывающегося списка, либо введите свой собственный. Формат основан на Java Простой формат даты. После выбора или ввода формата метки времени, если вы нажмете ввод, пример справа будет обновлен. чтобы отразить ваш выбор. Если ZAP не может использовать введенный вами формат, пример отображается на основе в формате по умолчанию.
Long & Default yyyy-MM-dd HH:mm:ss
ISO8601 yyyy-MM-dd'T'HH:mm:ssZ
Только время HH:mm:ss

Элементы управления шрифтом

Раздел «Общий шрифт» управляет шрифтом и размером текста по умолчанию, используемым для отображения ZAP, в то время как раздел «Шрифт рабочих панелей» управляет шрифтом, используемым для верхней правой части главного окна ZAP (например, инструмент «Запрос/ответ», «Редактировать и повторно отправить» и т. д.). Если вы установите значение -1, будет использоваться системный размер по умолчанию.
Поле «Пример шрифта» покажет вам, насколько большим будет текст по умолчанию.
Примечание: Этот параметр вступит в силу только после перезапуска ZAP.
