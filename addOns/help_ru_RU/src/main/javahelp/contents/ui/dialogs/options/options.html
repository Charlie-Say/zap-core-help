Сброс до заводских настроек по умолчанию

Кнопка «Восстановить заводские настройки» сбрасывает все параметры до значений, которые были установлены при первом запуске ZAP. Это включает в себя сброс хоста и порта, которые прослушивает ZAP, и ключа API.
Единственный элемент конфигурации, который не изменится, — это сертификат ZAP Root CA. Если вы хотите изменить это, вы можете сделать это через Динамический SSL-сертификат экран.
