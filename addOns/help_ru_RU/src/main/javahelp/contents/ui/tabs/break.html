Вкладка "Разрыв"

Вкладка Break позволяет вам изменить запрос или ответ, когда он был перехвачен ZAP через а точка останова .
Это позволяет вам изменять элементы, которые вы обычно не можете изменить с помощью вашего браузер, в том числе:
     Заголовок
     Скрытые поля
     Отключенные поля
     Поля, использующие javascript для фильтрации недопустимых символов
Эта функциональность является ключом к эффективному пенному тестированию вашего приложения.

Две панели будут содержать что-либо только в том случае, если ZAP перехватил запрос или ответ.
Вы можете изменить что-либо в этих двух панелях, а затем перенаправить запрос или ответ, используя кнопки на панели инструментов верхнего уровня .

Раскрывающиеся списки позволяют выбирать разные представления для заголовка и тела запроса или ответа.

Если вы выбрали «Показать кнопки для выбора запросов, которые вы не хотите прерывать ZAP» в меню опций вы сможете выбрать, если вы просто хотите прерывание для запросов, которые находятся в области действия, и вы сможете выбрать расширения файлов, которые вы не хотите прерывать:
       Javascript-файлы
       CSS и файлы шрифтов
       Мультимедийные файлы
       только в сфере

  Обновить длину содержимого

По умолчанию ZAP автоматически обновляет (или добавляет) заголовок HTTP(S) Content-Length, чтобы он соответствовал размеру отправленных данных. Если этот параметр не выбран, ZAP не будет обновлять заголовок, позволяя вам указать любое значение, которое вам нравится.
Эта кнопка будет отображаться только для сообщений HTTP(S).

Меню правой кнопки мыши

Найти...

Это вызовет Диалог поиска .

Копировать

Это скопирует выбранную строку в буфер обмена.
