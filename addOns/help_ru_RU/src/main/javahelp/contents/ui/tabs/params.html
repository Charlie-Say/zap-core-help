Вкладка "Параметры"

Это показывает сводку параметров и полей заголовка ответа, которые использует сайт.

Сайты можно выбирать с помощью панели инструментов или Вкладка "Сайты" .

Для каждого параметра вы можете увидеть:
     Тип — Cookie, FORM, URL или Заголовок.
     Имя параметра (или заголовок ответа)
     Количество раз это было использовано
     Количество уникальных значений
     Процентное изменение, где 0 означает, что использовалось только одно значение, а 100 означает, что все значения уникальны.
     Флаги, включая флаги cookie, anticsrf и сессию.
     Некоторые значения — полный набор значений может быть не виден.
