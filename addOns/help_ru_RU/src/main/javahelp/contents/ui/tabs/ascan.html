Вкладка «Активное сканирование»

Вкладка «Активное сканирование» позволяет выполнить активное сканирование .

Кнопка «Диспетчер политик сканирования» показывает Диспетчер политики сканирования диалоговое окно, позволяющее настроить политики сканирования .

Кнопка «Новое сканирование» запускает Диалоговое окно активного сканирования что позволяет вам точно указать, что следует сканировать.

Панель инструментов содержит набор кнопок, которые позволяют запускать, останавливать, приостанавливать и возобновлять выбранное сканирование.
Индикатор выполнения показывает, как далеко продвинулось сканирование выбранного сайта.
Значение «Текущие сканы» показывает, сколько сканирований в данный момент активно. при наведении курсора на это значение во всплывающем окне отобразится список сканируемых сайтов.
Кнопка «Показать сведения о ходе сканирования» запускает Диалоговое окно «Ход сканирования» что позволяет вам увидеть подробности о том, какие правила работают, пропустить отдельные правила и увидеть диаграмму ответов.

Меню правой кнопки мыши

Щелкнув правой кнопкой мыши по узлу, вы откроете меню, которое позволит вам:

Исключить из

Это меню имеет следующие подменю:

Прокси-сервер

Это исключит выбранные узлы из прокси. Они по-прежнему будут проксироваться через ZAP, но не будут отображаться в любой из вкладок.
Это можно использовать для игнорирования URL-адресов, которые, как вы знаете, не имеют отношения к тестируемой системе.
Узлы могут быть снова включены через Свойства сеанса диалог

Сканер

Это предотвратит активное сканирование выбранных узлов.
Узлы могут быть снова включены через Свойства сессии диалог

Паук

Это предотвратит сканирование выбранных узлов.
Узлы могут быть снова включены через Свойства сессии диалог

Открыть/повторно отправить с помощью редактора запросов...

Это вызовет Диалоговое окно редактора запросов вручную что позволяет вам повторно отправьте запрос после внесения в него любых изменений, которые вы хотите.

Новое оповещение...

Это вызовет Диалоговое окно «Добавить предупреждение» который позволяет вручную записывать новый оповещение против этого запроса.
