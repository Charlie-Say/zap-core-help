Вкладка «Оповещения»

На вкладке "Оповещения" отображаются Оповещения. которые были подняты на этой сессии. Оповещения отображаются в виде дерева в порядке риска на левой панели, и каждое узел дерева показывает общее количество предупреждений под ним.

Если выбрать оповещение одним щелчком мыши, оно отобразится на правой панели. Двойной щелчок по оповещению отобразит Добавить оповещение. диалоговое окно, которое позволит вам изменить детали предупреждения.

Фильтрация оповещений

Кнопки фильтра, показанные непосредственно над деревом, позволяют ограничить отображаемые оповещения:

 /    Показать только URL-адреса в области видимости/Показать все URL-адреса

Позволяет отображать только оповещения об URL-адресах, которые находятся в сфера действия.

 /    Связать / отменить связь с выбором сайтов

Позволяет отображать только оповещения Вкладка Сайты выбранный узел дерева.

Примечание. Выбор одного фильтра отключит другой, поскольку они взаимно эксклюзив.

  Изменить оповещение

Отобразит Добавить оповещение диалог, который позволит вам чтобы изменить сведения о предупреждении для выбранного предупреждения.

Примечание. Если в дереве выбрано несколько оповещений, последнее выбранное элемент редактируется.

  Удалить все оповещения

Позволяет удалить все оповещения.

Меню правой кнопки мыши

Щелкнув правой кнопкой мыши по узлу, вы откроете меню, которое позволит вам:

Открыть/повторно отправить с помощью редактора запросов...

Это вызовет Диалоговое окно редактора запросов вручную что позволяет вам повторно отправьте запрос, связанный с предупреждением, после внесения в него любых необходимых изменений.
