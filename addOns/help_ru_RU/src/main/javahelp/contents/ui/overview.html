Обзор пользовательского интерфейса рабочего стола

Пользовательский интерфейс рабочего стола состоит из:
    Меню верхнего уровня
    Панель инструментов верхнего уровня
    Три окна, которые содержат несколько вкладок, подробно описанных ниже.
    Нижний колонтитул
Каждое из трех окон имеет набор из одной или нескольких вкладок.
По умолчанию при запуске ZAP теперь отображаются только основные вкладки.
Остальные вкладки открываются, когда они используются (например, для паука и активного сканера) или когда вы открываете их через специальную вкладку в дальнем правом углу каждого окна с зеленым значком «+». Эта специальная вкладка исчезает, если нет скрытых вкладок.
Вкладки можно закрыть с помощью небольшого значка «x», который отображается, когда вкладка выбрана.
Вкладки также можно «закрепить» с помощью небольшого значка «булавки», который также отображается, когда вкладка выбрана - будут отображаться закрепленные вкладки. при следующем запуске ZAP.

'дерево' окно

Он отображается слева и позволяет перемещаться по посещенным URL-адресам.
     Вкладка "Сайты" Это показывает все посещенные URL-адреса в древовидной структуре.

Окно "рабочее пространство"

Он отображается в верхнем правом углу и здесь отображаются запросы и ответы. Вы также можете внести изменения здесь, когда точка останова ( breakpoint ) был поражена.
     Вкладка запроса Это показывает данные, которые ваш браузер отправляет в приложение.
     Вкладка "Ответ" Это показывает данные, которые приложение отправляет обратно в ваш браузер.
     Разрыв вкладки Это позволяет вам манипулировать данными

Информационное окно

Он отображается внизу, и здесь отображается полезная информация.
     Вкладка История Это показывает запросы в том порядке, в котором они были сделаны.
     Вкладка поиска Это позволяет вам искать все запросы и ответы.
     Вкладка "Точки останова" Это показывает набор точек останова
     Вкладка "Оповещения" Это показывает предупреждения, созданные в приложении.
     Вкладка "Активное сканирование" Это показывает активные сканирования
     Вкладка паук Это показывает URL-адреса, которые еще не посещались
     Вкладка Params Здесь отображается сводка параметров, которые использует сайт.
     Вкладка вывода Это показывает различные информационные сообщения

Все вкладки, кроме самых важных, можно скрыть, щелкнув серый значок «x» в заголовке вкладки.
Вы можете отображать скрытые вкладки с помощью Просмотр меню .
Скрытые вкладки также могут появиться при «активации» другим действием.
