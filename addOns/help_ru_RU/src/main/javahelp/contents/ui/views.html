Просмотры

ZAP предоставляет набор подключаемых представлений, которые позволяют отображать запросы и ответы по-разному.
По умолчанию включены следующие представления:
    ТекстДанные в текстовом формате
    Шестадцатиричный (Hex)Таблица, показывающая шестнадцатеричное представление всех символов в заголовке и теле.
Для редактируемых экранов это позволяет добавлять управляющие символы через их шестнадцатеричные коды.
Вы не можете добавлять или удалять символы в этом представлении — для этого переключитесь на одно из других представлений, а затем вернитесь к шестнадцатеричному представлению.
    ИзображениеИзображение - доступно только для тел, содержащих изображения
    ТаблицаТаблица с одной строкой на каждый атрибут и значение.
Для редактируемых экранов все значения будут автоматически закодированы в URL-адресе при отправке.
    Таблица (adv) Таблица с одной строкой на каждый атрибут и значение плюс дополнительные функции.
Для редактируемых экранов значения не будут автоматически кодироваться URL-адресом при отправке, но это можно сделать «вручную» с помощью функции «URLEncode».
    Большой запрос Представление-заполнитель, используемое для предотвращения попыток пользовательского интерфейса загрузить очень большое тело запроса.
Вам нужно будет переключиться на другой вид, чтобы отобразить фактическое содержимое.
Порог для представления можно изменить на экране Параметры отображения.
    Большой ответПредставление-заполнитель, используемое для предотвращения попытки пользовательского интерфейса загрузить очень большое тело ответа.
Вам нужно будет переключиться на другой вид, чтобы отобразить фактическое содержимое.
Порог для представления можно изменить на экране Параметры отображения.

Обратите внимание, что надстройки могут добавлять дополнительные представления.

Используется в

    Вкладка "Разрыв"
    Вкладка запроса
    Вкладка "Ответ"
    диалоговое окно ручного запроса
