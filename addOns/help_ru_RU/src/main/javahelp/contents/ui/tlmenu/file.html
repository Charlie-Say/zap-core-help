Меню «Файл»

Это меню обрабатывает текущий сеанс. По умолчанию будут присутствовать следующие пункты меню:

Новая сессия

Это создает новый сеанс.
Если вы не сохранили текущий сеанс, появится предупреждение.
Запуск нового сеанса без сохранения текущего сеанса приведет к потере всех данных текущего сеанса.

Открыть сессию...

This opens a session that has been previously saved.
Opening a session without saving the current session will loose all of the data in the current session.

Сохранить Сессию...

Это сохраняет текущий сеанс.
Хотя сеанс всегда хранится в базе данных на диске, он будет потерян при остановке ZAP, если только он не был сохранен.
Вам нужно сохранить его только один раз - после этого все изменения будут сохранены.

Снимок сеанса как...

Это сохраняет моментальный снимок сеанса, который уже был сохранен.
Он предложит то же имя файла, что и сохраненный сеанс, с добавленной к нему строкой даты и времени, и позволит пользователю установить любое имя по своему выбору.

Свойства сеанса...

Это отображает Свойства сессии диалог
Это позволяет вам установить имя и описание сеанса.

Импортировать Контекст...

Позволяет импортировать Контекст.

Экспортировать Контекст

Позволяет экспортировать контекст.

Загрузить файл дополнения...

Загрузите локальный дополнения файл.
Дополнения обычно устанавливаются через Управление дополнениями диалог, но это Опция может быть полезна, если вы загрузили надстройку вручную или тестируете надстройку, которую разработали сами.
Надстройки остаются установленными до тех пор, пока вы не удалите их вручную.

Выйти и удалить сеанс...

Это приведет к выходу из ZAP и удалению сеанса, даже если вы ранее сохраняли его.
Сеанс больше не будет доступен после перезапуска ZAP, хотя все снимки, которые вы сделали, останутся доступными.
Появится диалоговое окно с предупреждением, чтобы убедиться, что вы действительно хотели выбрать эту опцию.

Выход

Это приведет к выходу из ZAP.
Если вы не сохранили текущую сессию, вам будет предоставлена возможность сделать это.

Обратите внимание, что надстройки могут добавлять дополнительные пункты меню.
