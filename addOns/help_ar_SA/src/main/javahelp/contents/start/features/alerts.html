تنبيهات

التنبيه هو ثغرة أمنية محتملة ويرتبط بطلب معين.
يمكن أن يحتوي الطلب على أكثر من تنبيه.

تنبيهات تظهر في واجهة المستخدم مع علامة تشير إلى خطر:
      عالي
      متوسط
      منخفض
      معلومات
      إيجابية كاذبة

تنبيهات يمكن أن تثيرها مختلف انطلق العناصر, بما في ذلك ولكن لا تقتصر على: active scanning, المسح السلبي, البرامج النصية ، addons (امتداد) أو يدويا باستخدام إضافة تنبيه الحوار (والذي يسمح لك أيضا إلى تحديث أو تغيير تفاصيل التنبيه/المعلومات).

التنبيهات توضع في التبويب التاريخ مع العلم مما يدل على أعلى المخاطر في حالة تأهب.
جميع التنبيهات المذكورة في تنبيهات التبويب و عد من إجمالي عدد التنبيهات من المخاطر هو مبين في تذييل.

تنبيه يتجاوز

التنبيهات التي أثارها انطلق تشمل كل عام و معلومات محددة حول التنبيهات التي أثيرت. معلومات محددة تتعلق مباشرة إلى مشكلة محتملة وجدت مثل URL المعلمة المتضررة. معلومات عامة تشمل أشياء مثل وصف وصلات إلى الموارد على الانترنت.

يمكنك استبدال أو إضافة إلى معلومات عامة باستخدام تنبيه تجاوز' ملف التكوين. هذا يسمح لك أن تشمل المعلومات التي تخص الشركة مثل تكليف السياسات الروابط الداخلية أو نصيحة محددة التقنيات التي تستخدمها.

تنبيه تجاوز ملف التكوين هو UTF-8 المنشأة ملف يتضمن فقط المعلومات التي تريد تغييرها. الخطوط بدءا من '#' يتم التعامل مع التعليقات تجاهلها.

الشكل: 



<alertid>.<property> = [ + | - ] <your information>
الخصائص التالية معتمدة:
  • الإسم
  • الوصف
  • الحل
  • معلومات أخرى
  • إشارة
The format for alert tags is: 
<alertid>.tag.<key> = [ + | - ] [ <value> ]
Note that:
  • Spaces in the tag key must be escaped with a backslash.
  • The tag value is optional.
For example 
# 40012 = Reflected XSS
40012.solution = Follow our company specific guidelines at http://internet.example.com/xss.html
40012.tag.Bug\ Tracker\ URL = https://example.com/issues/1234
If the value starts with a '+' then it is appended to the existing information.
If it starts with a '-' then it is prepended to the existing information.
If it does not start with a '+' or '-' then it replaces the existing information.

تنبيه تجاوز ملف التكوين يمكن تحديد طريق API, خيارات التنبيه الشاشة أو باستخدام سطر الأوامر خيار صندوق قائمة 

-التكوين في حالة تأهب.overridesFilename=<filename>
