Ang ika 1.1.0 na labas

Ang mga sumusunod na pagbabago ay ginawa sa paglabas na ito:

Mga makabuluhang pagbabago:

Rebranding ng OWASP

Ang ZAp ay tinanggap bilang isang proyekto ng OWASP.
Its homepage is now: https://owasp.org/www-project-zap/

Brute Force

Ang kakayahang sang-ayon sa mga brute force file at mga direktaryo batay sa code mula sa proyektong OWASP DirBuster.
Ang mga bagong Brute Force tab na nagpapakita sa mga file at mga direktaryo na natagpuan.

Ang port ng pag-scan

Ang kakayahang mag-port ng mga site ng pag-scan.
Ang pag-papakita ng mga bagong tab ng Port Scan.

Aktibong Scan tab

Ang Bagong Aktibong tab sa Pag-scan ay nagpapakita ng mga kahilingan at mga pagtugon bilang isang resulta ng aktibong pag-scan sa isang site.

Tab ng ispayder

Ang Tab ng ispayder ngayon ay nagpapahintulot sa iyo na magpatuloy sa paggamit ng ZAP habang nag-ispaydering sa isang site.
Maaari mo din patigilin ng sandali at ipagpatuloy ang ispayder.

Suporta para sa smartcard

Ang suporta ng smart card ay idinagdag c/o ang proyektong Andiparos.
Ang mga sumusunod na mga aparato ng smartcard ay sinubukan na sa Windows:
    PangalagaanAladdin eToken
    Aladdin eToken Pro
    OmnikeyOmnikey 3121
    CardMan 6121
    GemaltoReflex 20 V2
    Swins stick
Ang mga sumusunod na mga aparato smartcard ay iniulat sa:
    OmnikeyAng CardMan 4040

Attack menu

Ang bagong Mga tab para sa site ang pag right click sa 'Attack' menu ay nagpapahintulot sa iyo na mag simula ang iba't ibang klase ng pag-scan.

Higit pang internationalisation

Ang lahat ng pangunahing mga tab at mga item menu ay na-internasyonal na.

Pagpili ng Wika

Sa pagsisimula ikaw ay didiktahan ng pumili ng wika na gagamitin.
Ang bagong wika ay automatikong napapasin sa pamamagita ng paghahanap sa mga file namay pangalan ng mga forn ng mga mensahe_<locale>.pag-aaring nasa direktaryo ng ZAP.

Maliit na mga pagbabago:

Hindi pinaganang mga plugin ng 'default file'

Ang mga plugin na napapasin ng mga default file ay epektibong ginagawang kalabisan ang mga bagong burte force scanner.
Ang mga ito ay hindi pinagana.

Mga buod ng scanner

Ang bilang ng mga numero at mga uri ng kasalukuyang pag-scan ay ipinapakita na ngayon sa footer.
