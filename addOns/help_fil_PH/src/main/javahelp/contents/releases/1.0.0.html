Inilabas 1.0.0

Ang mga sumusunod na pagbabago ay ginawa sa inilabas na ito:

Ang mga makabuluhang pagbabago:

Ang mga tulong ng iskrin

Ang mga tulong ng iskrin ay idinagdag para sa lahat ng mga pagpapatakbo na ibinibigay ng ZAP.
Ang mga sensitibong konteksto na tulong ay naipatupad din kaya kapag ikaw ay pumindot sa F1 key pagkatapos pagkatapos ang tulong ng iskrin na nauugnay sa na iyong pinili ay ipinapakita.

Breakpoints

Ang pagsasagawa ng Breakpoints ay binago upang maging malapit sa inilaan ng sofware Intergrated ng pagpapaunlad sa mga kapaligiran.
The breakpoint controls have been moved to the new Mataas na antas ng toolbar.
Breakpoints can now also be set by right clicking nodes in the Sayt at Mga kasaysayan ng tab.

Ang pag-ulat ng menu

Ang Pag-ulat ng menu ay naging makabuluhang pagbabago at ngayon akbilang ang mga sumusunod na mga aytem ng menu:
    Bumuo ng report...Alin ngayon ang mga binubuong mga ulat na kailangan
    Ang pag-export ng mga mensahe sa payl...Alisin mula sa payl menu
    Ang pag-export ng tugon sa payl...Alisin mula sa payl Menu
    Ang pag-ex[port sa lahat ng mga URL sa payl...Ang bagong pamamahala
    Ihambing sa iba pang sesyon...Ang bagong pamamahala

Ang pagpapabuti sa kasaysayan ng paghahanap

Ang bagong Tab ng paghahanap ay pinahihintulutan ka na maghanap sa regular na mga ekspresyon sa lahat ng mga URL, ng mga kahilingan at mga tugon.
Ang susunod at nakaraang mga buton ay pinahihintulutan ka na makita ang mga termino na hinahanap sa mga Kahilingan at Mga tugon ng tab.

Ang kasaysayan ng piltro

Ang kasaysakayan ng piltro ng toolbar ay naidagdag upang pahintulutan ka na higpitan kung ano man ang ipinapakita na mga kahilingan.
Ang mga kahilingan ay maaaring ma-piltro base sa mga HTTP na mga paraan, ang resulta ng HTTP na mga kowds at anumang mag tag, na mga alerto, o mga tala na nailapat.

Ang mga tala

Maaari mo ngayong i-ugnay ang mga tala sa kahit anumang kahilingan.

Ang mga maramihang tag

Maaari mo ngayong i-ugnay ng maramihan na mga tag sa kahit alinmang kahilingan.

Higit pang pamamahala sa mga alerto

Ang Pagdadagdag ng alerto ng diyalogo na pinahihintulutan ka na magdagdag ng mano mano o palitan ang isang alerto.

Ang bagong encoder /decoder

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

Ang pasibong pag-iskan

Passive scanning has been added.
In this release ZAP will only use passive scanning for automatically adding tags.
In future releases it may also raise alerts for potential issues.

Ang mga maliliit na pagbabago:

Ang mataas na antas ng toolbar

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

Ang mga buod ng alerto ng footer

Counts of the High, Medium, Low and Informational alerts are now displayed in the footer.

Ang bagong hitsura at pakiramdam

A new look and feel 'Nimbus' is used.

Ang pag hawak sa pasword ng proxy

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Ang pagpapadalang muli mula sa mga tab ng sayt

The Sites tab has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

Ang pagpapanatili sa pag gamit ng direktoryo sa pagitan ng mga sesyon

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
