Inilabas na 2.4.3

Ang sumusunod na mga pagbabago ay ginawa dito sa inilabas na ito:

Mga pagpapahusay:

  • Isyu 1576 : Tukuyin ang URL path ng mga elemento bilang 'hindi struktura'
  • Isyu 1711 : Kahilingan ng itinatampok: Pagpipilian upang matukoy ang bilang ng token na nabuo.
  • Isyu 1768 : Baguhin upang magamit ng higit pa sa mas bagong gumagamit na agent
  • Isyu 1894 : Sinusuri lamang ng ZAP ang "Gumagamit na Agent","Nagsangguni", at "Host" ang kahilingan ng mga header
  • Isyu 1911 : Hanapin ang mga kahilingan ng AJAX Ispayder
  • Isyu 1913 : Ipasa lahat ng mga param sa kabuuan ng zap.sh - debian package
  • Isyu 1914 : Maramihang mga kategorya ng add-on
  • Isyu 1920: Iulat ang host:port ZAP ay nakikinig sa paraan ng daemon, o ilabas kung hindi kaya
  • Isyu 1927 : Pagpapahusay: Mag-break lamang sa saklaw ng mga URl.
  • Isyu 1944 : Mga pagtugon sa Chart kada segundo sa progreso ng ascan
  • Isyu 1953 : Payagang ang ispayder sa isang konteksto sa pamamagitan ng ZAP API
  • Isyu 1962 : Magsalin at baguhin ang mga add-on mula sa linya ng nag-uutos
  • Isyu 1975 : RC4-SHA SSL cipher na sumusunod ay hindi suportado
  • Magdagdag ng ZAP CLI sa mga imahe ng Docker
  • Isyu 1985 : Dalahin ang binagong diyalogo kung ang gumagamit ay dumoble ang isang hilera sa AbstractMultipleOptionsBaseTablePanel
  • Isyu 2009 : Magdagdag ng mga link ng Online sa FAQ at mga pahina ng Newsletter
  • Balaan ang mga gumagamit kung sila marahil ay gumagamit ng mga bersyon na wala na sa panahon
  • Iulat ang mga bilang ng kahilingan kada plugin
  • Isyu 2088 : Suportahan ang isang 'baguhin ang lahat' na buton para sa pagsasalin ng lahat ng mga add-on na binago
  • Isyu 2094 : Suportahan ang isang '-addoninstallall' sa linya ng pagpipilian ng taga-utos
  • Isyu 2102 : Payagan ang ajax spider sa mga pagpipilian na maaaring itakda sa pamamagitan ng API
