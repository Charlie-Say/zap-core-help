Ang ika 1.3.0 na paglabas

Ang mga sumusunod na pagbabago ay ginawa sa paglabas na ito:

Mga Makabuluhang pagbabago:

Fuzzing

Ang mga string sa isang pagtugon ay maaring maging fuzzed kapag sinusubukang hanapin ang mga kahinaan.
Maaaring makita ang mga token ng Anti CRSF at awtomatikong magre-regenerat kapag fuzzing.
Ang pag-andar na ito ay batay sa code mula sa proyekto ng OWASP JBroFuzz.

Mga Sertipiko ng Dynamic SSL

Ang suporta para sa mga koneksyon sa SSL ay pinabuti at pinasimple.
Ang mga gumagamit ay maaarin nang gumawa ng sarili nilang root certificate at ipamahagi ito sa mga kliyente nila sa HTTP.
Tignan sa Opsyon: ng mga Sertipiko ng Dynamic SSL para sa iba pang mga detalye.

Mode ng Daemon at API

Ang pagsisimula ng ZAP gamit ang command line opsyon ng "-daemon" ay nagdudulot ng pagtakbo nito sa background ng 'headless' mode, nangangahulugan ito na walang UI na ipinapakita.
Ang unang API ay naipatupad sa XML, JSON at HTML.
Kung ang ZAP ay gumagana bilang isang daemon pagkatapos ang API ay awtomatikong pinagana, kung hindi man ang API ay dapat paganahin sa pamamagitan ng mga opsyon ng API screen.
Ang API ay maaaring ma-navigate sa pamamagitan ng pagbubukas sa http://zap/ sa iyong browser kapag proxying sa pamamagitan ng ZAP.

Pagbubuo ng Beanshell

Ang BeanShell ay isang interactive na Java shell na maaaring gamitin upang magsagawa ng mga BeanShell iskrip.
Ang mga iskrip na ito ay pinasimpleng anyo ng Java na gumagamit ng mga elemento mula sa Java syntax, ngunit sa isang simpleng scripting format.
Ang lahat ng Java code ay mabisa din na BeanShell code.
Ang pagsasama-sama ng BeanShell sa OWASP ZAP ay nagbibigay-daan sa iyo na sumulat ng mga iskrip gamit ang mga ZAP function at data set.
Ito ay maaaring maging isang napaka lakas na tampok para sa pag-aaral ng mga web application.

Buong internationalisation

Lahat ng ipinapakitang string ay ganap ng na-internationalised ngayon.

Mga maliit na pagbabago:

Tignan ang Hex

Ang Paghiling, Pagtugon mga tab na nagbibigay ngayon sa isang 'Hex View' opsyon na nagpapakita ng mga kahilinga at pagtugon na mga katawan sa hex format.

Mga resulta ng paghahanap

Ang Tab ng paghahanap ngayon sa lahat ng mga pagkakataon na nagpapakita ng isang string na natagpuan sa halip na ang unang pagkakataon lamang sa bawat kahilingan o pagtugon.

Ibukod ang mga URL

Ang mga URL ay maaaring ibukod mula sa aktibong scanner, ispayder at mula sa proxy.

Suporta sa pagkopya

Karamihan sa mga panel ngayon ay nagbibigay isang 'right click' 'Copy' menu opsyon kasama ang Port Scan at Brute Force na mga panel.

Suporta sa Undo/Redo

Ang lahat ng inilagay ngayon na mga field support para sa Undo/Redo na mga aksyon gamit ang mga operating system kawalang ng mga Undo/Redo accelerator:
  • Mga Window/Linux: Ctrl+Z/ Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Suporta para sa Port scanner proxy at timeout opsyon

Ang Port Scanner ay maaari na ngayon gamitin ang papalabas ng proxy (kung isinaayos) at ang timeout sa mga milliseconds ay maaari na rin set ngayon.

Mga buton para pag-break ng kahilingan at pagtugon

Meron na ngayon 2 mga buton ng 'Set Break' upang pahintulutan ang mga beak na ma-set sa lahat ng mga kahilingan at lahat ng mga pagtugon nang malaya.

Palawakin ang mga buton sa mga Site at impormasyon sa tab

Meron na ngayon 2 mga buton na nagpapahintulot sa iyo lumipat sa pagitang ng mga Site at mga para sa mga impormasyon na lumalawak.

Madaling i-adjust na timeout

Ang Koneksyon: ng opsyon pinapayagan ka ng screen para itakda ang timeout sa mga segundo upang gawing mas madali ang pagsubok sa mga mabagal na aplikasyon.

Mga update sa library

Karamihan sa mga libraring ginamit ng ZAP ay na-update na sa mga bagong bersyon.

Mga kilalang isyu:

Mac OS X: Dynamic SSL at Google Chrome

Kasalukuyang hindi gumagana ang Dynamic SSl kapag ginagamit ang Google Chrome. Ito ay dahil ang hindi na lutas ng isang kilalang isyu ng Google Chrome at Mac OS X Keychain. Kapag nag-import ang OWASP ZAP's Root CA sa Keychain at humiling ang isang SSL website, isang "Invalid certificate" error na mensahe ang ipapakita.
