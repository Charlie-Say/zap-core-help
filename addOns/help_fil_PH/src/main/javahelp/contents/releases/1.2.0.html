Ang ika 1.2.0 paglabas

Ang mga sumusunod na pagbabago ay ginawa sa paglabas na ito:

Mga makabuluhang pagbabago:

Mga Memory Leak

Ang mga memory leak ay naayos na sa aktibong scanner at ispayder.

Pagtawag ng mga aplikasyon

Ang mga External application ay maaari nang ngayon tumawag mula sa mga tab ng mga Site at Kasaysayan.

Walang pasubali na Iskaner

Ang walang pasubali na Iskaner ay tumitingin na ngayon sa mga kahinaan, tulad ng:
    Ang mga Autocomplete form na may mga patlang ng password
    Mga Cookie na walang hudyat ng 'HttpOnly'
    Mga SSL Cookie na walang hudyat ng 'secure'
    Mahinang pagpapatunay

Maliit na mga pagbabago:

Mga ulat ng XML

Ang isang bagong 'Nilikhang ulat ng XML...' na mga aytem ng pagpipilian ay kasama na sa nangungunang antas ng Mga Ulat ng pagpipilian.

Mga dayalogo ng Editor ng Manu-manong Kahilingan

Mga kahilingan na ibinigay sa pamamagitan ng Manual Request Editor and Resend dialogs are now shown in the Sites and History tabs.
A new 'Method' pull down allows you to switch between the HTTP methods, this automatically moves parameters between the URL and the body when a POST method is selected or deselected.

Ihanda ang UI

The Sites tab now shows any alerts as flags to the right of any node names.
The alert counts in the footer now show the number of different types of alerts rather than the total number of instances.

Naantalang opsyon ng aktibong iskaner

The delay in milliseconds between each active scanner request can now be set via the Options Active Scan screen. This will increase the time an active scan takes but will reduce the load on the target.

Mga pagbabago sa UI

The Sites tab now takes up all of the left hand side - this can be changed back via the Options Display screen if required.

The 'toolbar' on the Request, Response and Break tabs and the Manual Request Editor and Resend dialogs is now at the top rather than the bottom.

The scanner counts in the footer are now displayed on the right hand side.

Tingnan din

     Panimulaang panimula ng ZAP
     Mga Releaseang kabuuang pangkat ng mga release
     Mga Kreditoang tao at mga grupo na ginawang posible ang release na ito
