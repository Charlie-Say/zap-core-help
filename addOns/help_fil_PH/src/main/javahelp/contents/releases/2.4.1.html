Ang Pagpakawala 2.4.1

Itong Pagpapakawala ay mayroong mga mahalagang seguridad na inayos - ang mga gumagamit ay mahigpit na inirekomenda upang ma-upgrade agad.

Isa din sa mga pagbabago na nangangahulugan na ang susi dito sa API ay nilikha sa pamamagitan ng default, ang kahulugan nito ay ang kahit anong aplikasyon na gumagamit ng ZAP API ay nagkamali maliban na lamang kung sila ay mag-update sa pagamit ng ganyang susi.
Ang Susi ng API ay maaaring mahanap sa Ang mga Opsyon na iskrin ng API
Maaari mo din na itakda ito mula pa sa ipinaguutos na linya sa pamamagitan ng opsyon tulad ng: 

-config api. susi=baguhin-ako-9203935709

Para sa marami pang mga detalye ay tingnan lamang ang https://www.zaproxy.org/faq/why-is-an-api-key-required-by-default/
