Isyu 133: Magdagdag ng Syntax na maghighlight sa Panel ng Pagtugon

Isyu 153: pagsasama ng fuzzdb

Ang fuzzer kasama ang mga fuzzdb (https://github.com/fuzzdb-project/fuzzdb) fuzzing file.
Tandaan na ang ibang mga fuzzdb file ay naiwan bilang ito ay naging sanhi ng pangkaraniwang anti virus ng mga iskaner upang hudyatan sila kung sila ay mayroong mga virus.
Maaari mong palitan ito (at i-upgrade ang fuzzdb) sa pamamagitan ng pagdownload ng pinakabagong bersyon ng fuzzdb at palawakin ito sa librerya ng mga 'fuzzer'.

Isyu 212: Pagsusuri sa Parameter

Isyu 228: Pinahusay na XSS iskaner

Ang Cross Site Scripting na muling isinulat ng aktibong iskaner mula sa scratch upang makahanap pa ng mas malaking potensyal sa mga isyu ng XSS at ulat ng mas kaunting positibong mali.

Isyu 244: I-Port ang walang pasubali na pagsisiyasat ng mga tagamasid

Ang mga sumusunod na pagsisiyasat ay nai-port mula sa tagamasid (salamat kay Chris Weber sa pag-oking dito):
    Check.Pasv.CrossDomain.ScriptReference.csnagsusuri para sa nakalakip na cross-domain javascript file.
    Check.Pasv.Header.CacheControl.csnagsusuri ng HTTP cache-control header sa mga pahina ng SSL.
    Check.Pasv.Header.ContentTypeMissing.csnagsusuri na ang Content-Type-HTTP header ay hindi nawawala.
    Check.Pasv.Header.FrameOptions.csnagsusuri na ang X-FRAME-OPTIONS ay hindi nawawala o walang kasiguraduhang nai-set.
    Check.Pasv.Header.IeXssProtection.csnagsusuri na ang X-XSS-Protection ay hindi pa nai-set upang hindi paganahin ang IE's XSS protection.
    Check.Pasv.Header.MimeSniff.csnagsusuri na ang X-CONTENT-TYPE-OPTIONS ay nai-set.
    Check.Pasv.InformationDisclosure.DatabaseErrors.csnagsusuri para sa mga maling mensahe ng database.
    Check.Pasv.InformationDisclosure.DebugErrors.csnagsusuri para sa maling mensahe ng debugging.
    Check.Pasv.InformationDisclosure.InUrl.csnagsusuri para sa pagsisiwalat ng impormasyon sa mga parameter ng URL.
    Check.Pasv.InformationDisclosure.ReferrerLeak.csnagsusuri ng HTTP Referer header para sa pagsisiwalat ng impormasyon.

Isyu 253: Mga ekstensyong Plugable
