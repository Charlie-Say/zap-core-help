Inilibas 2.3.1

Ang mga sumusunod na pagbabago ay ginawa sa inilabas na ito:

  • Isyu 81 : Ang mga pagbabago ng ZAP sa kahilingan ng datos (habang nagpapalit ng mga ipinapakita)
  • Isyu 377 : Ang hindi natapos na mga dependensya na nagha-hang sa aktibong pagsusuri
  • Isyu 1073 : Hindi kayang tanggalin ng mga iskrip na nagmarka bilang 'mag-load sa simula'
  • Isyu 1114 : ang core.newSession ay hindi maliwanag na mga sayt
  • Isyu 1155 : ang Kasaysayan ng kahilingan na Tab ay hindi pinapayagan ang pag-format ng mga pagbabago
  • Isyu 1156 : ang proxy ng gzip na decoder ay hindi maaaring mag-update ng haba ng nilalaman sa pagtugon ng mga header
  • Isyu 1163 : Hindi maaaring mag-takda sa isang home directory kasama ang isang espasyo sa linya ng kautusan
  • Isyu 1166 : Ang kalabisan ng mga index sa zapdb.iskrip
  • Isyu 1168 : ang pagdadagdag ng suporta ng proxy para sa "deflate" na nilalaman ng pag-encode
  • Isyu 1170 : Ang ispayder ng konteksto/gumagamit ng pop up ng mga menu na hindi na makikita pa
  • Isyu 1179 : Hindi maaring piliin ang dalawang mga kahilingan sa mga resulta sa fuzz ng (Ctrl + click)
  • Isyu 1181 : Ang mga pahina ng kahinaan ng aktibong pag-iskan ng isang beses lamang
  • Isyu 1182 : ang mga alerto ng parehong klase para sa magkaibang mga parametro sa parehong kahinaan ng pahina na nagpapakita lamang ng isang beses sa puno ng mga "alerto"
  • Isyu 1183 : ang NullPointerException habang pinipili ang isang node sa mga "Alerto" na tab pagkatapos tanggalin ang isang mensahe
  • Isyu 1191 : ang Cmdline na sesyon ng mga param ay walang epekto
  • Isyu 1193 : ang pag-iskan ng URL na daanan ng mga elemento - isara sa pamamagitan ng default
  • Isyu 1194 : ang linya ng pagpapasunod ng mga argumento ay hindi nakapasa sa mga ekstensyon habang nagsisimula ang ZAP sa moda ng daemon
  • Isyu 1196 : ang AbstractPlugin.bingo na hindi tamang mga takda ng ebidensya sa pag atake
  • Isyu 1202 : Ang isyu na kasama ang pag-loading ng mga add-on na hindi tama ang pagsisimula
  • Isyu 1203 : ang awtentikasyon ng Wordpress ng iskrip
  • Isyu 1206 : ang "Kasaysayan" ng tab ay hindi malinaw kapag ang isang bagong sesyon ay binuo sa pamamagitan ng API kasama ang ZAP sa moda ng GUI

Tingnan din sa

     Ang panimulaang panimula sa ZAP
     Ang mga inilabasang buong ayos ng mga inilabas
     Ang mga kreditoang mga tao at mga grupo na ginawang imposible ang inilabas na ito
