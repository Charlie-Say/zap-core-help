Istatistika

Ang ZAP ay nagpapanatili ng mga istatistika na kung saan ay makakatulong sa iyo na maunawaan kung ano talaga ang nangyayari kapag nakikipagsalamuha sa mga malalaking aplikasyon.

Ang estadistika ay magagamit sa pamamagitan ng API at maari itong ipadala sa isang Statsd server kapag naka-ayos gamit ang screen ng opsyon ng Istatistika.

Mga Istatistika na nakabatay sa Site

Ang mga istatistika na pinapanatili sa bawat batayan ng site ay kinabibilangan ng:
  • mga code ng tugon, halimbawa:
    • stats.cod.200
    • stats.code.302
  • oras ng tugon sa ms (gamit ang logarithmic na sukatan), halimbawa:
    • stats.responseTime.1
    • stats.responseTime.2
    • stats.responseTime.4
    • stat.responseTime.8
    • stats.responseTime.16
  • mga uri ng nilalaman, halimbawa:
    • stats.contentType.text/css
    • stats.contentType.text/html;charset=utf-8
  • mga tag, halimbawa:
    • stats.tag.Password
    • stats.tag.Hidden
  • nabuong mga token ng anticsrf:
    • stats.acsrf.anticsrf
  • impormasyon ng pagpapatunay:
    • stats.auth.success (bilang ng mga tagumpay ng pagpapatunay)
    • stats.auth.failure (bilang ng mga bigong pagpapatunay)
    • stats.auth.state.assumedin (number of messages between successful polls that are assumed to be logged in)
    • stats.auth.state.loggedin (number of messages that appear to be logged in)
    • stats.auth.state.loggedout (number of messages that appear to be logged out)
    • stats.auth.state.noindicator (number of messages where no logged in or out indicators have been set)
    • stats.auth.state.unknown (number of messages which don't contain either logged in or out indicators)
