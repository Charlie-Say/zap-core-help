Ang API

Ang ZAp ay nagbibigay ng isang aplikasyon ng programang interface ng (API) nakung saan ay pinahihintulutan ka na makipag-ugnayan sa programatikong ZAP.

Ang API ay maggamit sa JSON, ang HTML at XML na mga format.
ang isang simpleng web UI kung saan ay pinahihintulutan ka na magsiyasat at gumamit ng API na magagamit sa pamamagitan ng mga URL http://zap/ habang ikaw ay nagpro-proxy sa pamamagitan ng ZAP, o sa pamamagitan ng host at port ng ZAP na nakikinig, halimbawa http://localhost:8080/.

Sa pamamagitan ng deafult lamang ang makina ng ZAP ay tumatakbo ay magagamit upang mapatakbo ang API. Maaari kang pahintulutan ng ibang mga makina, na maaaring gamitin ng ZAP bilang isang proxy, ng pagpapatakbo sa mga API. Ang API ay nakumpigura gamit ang opsyon ng API ng iskrin.

Ang API na nagbibigay daan sa pinaka mga buod ng ZAP na tinampok gaya ng aktibong iskaner at ispayder.
ang panghinaharap na bersyon ng ZAP ay maaaring tumaas ang pagpapatakbo na magagamit sa pamamagitan ng API.
