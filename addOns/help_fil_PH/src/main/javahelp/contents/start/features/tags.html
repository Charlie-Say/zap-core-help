Mga Tag

Ang tag ay isang maikling piraso ng teksto na nais mong iugnay sa isang kahilingan.
Ang isang kahilingan ay maaaring magkaroon ng maraming tag na kugnay nito.
Ang mga tag ay maaari idagdag at baguhin gamit ang Diyalogo ng Pamamahala ng mga Tag.
Ito ay maaari ring awtomatikong idagdag sa pamamagitan ng Passive na Pag-scan - ito ay nagpapahintulot sa iyo na i-tag ang mga pahina na tumugma sa isang partikular na regular expression.

Ang mga tag ay ipinapakita sa Tab ng Kasaysayan at maaaring i-filter gamit ang Diyalogo ng Filter ng Kasaysayan.
