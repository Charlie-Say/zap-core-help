Ang Anti CSRF na mga Token

Ang Anti CSRF na mga token ay (pseudo) random na mga parameter na nagamit sa pagprotekta laban sa Cross Site Request Forgery (CSRF) na mga atake.
Gayunman sila din ay gumawa ng isang pagtagos na mga tagasubok na trabahong mahirap, lalo na kung ang mga token ay nabuo sa bawat oras na ang form ay hiniling.

Ang Zap ay tumutukoy sa anti CSRF token panay sa pamamagitan ng mga pangalan ng attribute - nakakumpigura ang listahan ng mga pangalan ng katangian ng itinuturing na anti CSRF token gamit ang screen ng mga opsyon Anti CSRF .
Noong tumukoy ng Zap ang mga token ito ay nagtala ng mga token na halaga at kung aling mga URL ay nakabuo ng mga tanda. Iba pang scanner tulan ng iskaner ng aktibo, ay may opsyon na dahilan ng mga ZAP sa awtomatikong pagkuha ng token kapag kinakailangan.
