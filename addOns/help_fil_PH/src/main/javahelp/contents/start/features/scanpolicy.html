Ang patakaran ng pag-iskan

Ang isang patakaran ng pag-iskan na tumutukoy eksato kung alin sa mga patakaran ang tumatakbo bilang parte ng isang aktibong pag-iskan.
timutukoy din nito kung paano itong mga patakaran ay maimpluwensyang tumakbo kung paano ang karamihang mga kahilingan ay nagawa at kung paano ang ginustong mga postensyal na isyu ay nai-flag.
maaari mong tukuyin ang kahit ilang mga patakaran ng pag-iskan hanggat gusto mo at piliin ang pinaka isang angkop habang sinisimulan mo ang pag-iskan sa pamamagitan ng mga aktibong pag-iskan ng diyalogo.
maaari mong tukuyin ang default na patakaran ng pag-iskan upang magamit para sa mga aktibong pag-iskan at para sa mga pag-atake ng moda sa pamamgitan ng mga opsyon ng aktibong pag-iskan ng iskrin.

Ang aktibong pag-iskan ay isang pag-atake sa pinuntiryang iyon.

Ang aktibong pag-iskan ay nakumpigura gamit ang mga opsyon ng aktibong pag-iskan ng iskrin.
