Mga Paalala

Ang paalala ay anumang teksto na nais mong makasalamuha sa isang kahilingan
Halimbawa, maaaring mong gamitin ang mga paalala upang maitala ang mga detalye ng ekstrang pagsusuri na kailangan mo ipang isagawa ang isang kahilingan.
Ang mga paaalala ay idinagdag at binago gamit ang Diyalog ng Pagdaragdag ng Paalala.

Ang mga paalala ay naka-flag sa Tab ng Kasaysayan na may na icon.
