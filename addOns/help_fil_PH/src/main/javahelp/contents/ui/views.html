Mga View

Ang ZAP ay nagbibigay ng isang hanay ng mga plugable na view na nagpapahintulot sa iyo na ipakita ang mga kahilingan at mga tugon sa iba't-ibang paraan.
Ang mga sumusunod na mga view ay kabilang bilang default:
    TekstoAng mga datos sa format na teksto
    HexIsang talahanayan na nagpapakita ng hex na representasyon ng lahat ng mga karakter sa header at nilalaman.
Para sa mga maaari ma-edit na mga screen, ito ay nagpapahintulot sa iyo na magdagdag ng mga karakter na kontrol sa pamamagitan ng mga hext code.
Hindi ka maaaring magdagdag o magtanggal ng mga karakter sa view na ito - upang magawa ito lumipat sa isang mga ibang mga view at pagkatapos bumalik para ipagpatuloy sa hex view.
    LarawanAng larawan - na magagamit lamang sa nilalaman na naglalaman ng mga larawan
    TableAng table na may isang hilera bawat katangian at halaga.
Para sa mga maaaring ma-edit na mga screen lahat ng mga halaga ay awtomatikong maging URL na naka-encode kapag ang mga ito ay isinumite.
    Table (adv)Ang table na may isang hilera bawat katangian at halaga gayon din ang mga karagdagang paggamit.
Para sa maaaring ma-edit na mga screen ay hindi awtomatikong maging URL na naka-encode kapag ang mga ito ay isinumite, ngunit ito ay maaaring magawa ng 'mano-mano' sa pamamagitan ng pamamaraan na 'URLEncode'.
    Malaking KahilinganIsang placeholder view na ginamit upang pigilan ang UI sa pagsusubok na i-load ang isang napakalaking nilalaman ng kahilingan.
Kakailanganin mong magpalit sa isang naiibang view upang ipakita ang mga aktwal na nilalaman.
Ang threshold para sa view ay maaari baguhin sa pamamagitan ng Screen ng mga Opsyon ng Display
    Malaking TugonIsang placeholder view na ginamit upang pigilan ang pagsusubok ng UI na i-load ang isang napakalaking nilalaman ng tugon.
Kakailanganin mong magpalit sa isang naiibang view upang ipakita ang mga aktwal na nilalaman.
Ang threshold para sa view ay maaari bahugin sa pamamagitaan ng Screen ng mga Opsyon ng Display

Note that add-ons can add additional views.
