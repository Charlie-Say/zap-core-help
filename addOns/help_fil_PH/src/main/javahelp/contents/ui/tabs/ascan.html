Ang Aktibong Pagscan sa tab

Ang Aktibong Pagscan sa tab ay magpapahintulot na magawa mo ang ang aktibong pagscan.

Ang 'Tagapamahala sa Patakaran ng Pagscan' na button ipakita ang Ang Tagapamahala sa Patakaran ng Pagscan dayalogo na nagpapahintulot sa kompigurasyon ng ang mga patakaran ng pag scan.

Ang button ng 'Bagong Pagscan' ay maglulunsad ng Ang dayalogo ng Aktibong Pagscan na nagpapahintulot sa iyo na tiyaking eksakto kung ano ang dapat ma-scan.

Ang toolbar na magbibigay ng set ng mga button na magpapahintulot sa iyo na umpisahan, itigil, i-pause at ipagpatuloy ang napiling i-scan.
Ang pag-usad ng bar ay nagpapakita ng progreso ng pagscan sa napiling site.
Ang halaga ng 'Kasalukuyang mga pagscan' ay nagpapakita ito kung gaano kadaming pagscan ang kasalukuyang aktibo - ang pag-hover sa ibabaw ng halagang ito ay magpapakita ng listahan ng mga site na na-scan sa loob ng pop-up.
Ang 'Pagpapakita ng mga detalye sa pag-unlad ng pagscan' na button ang paglulunsad ng Pag-unlad ng dayalogo ng pag scan kung saan ikaw ay pinapahintulotan na makita ang mga detalye tungkol sa kung anong mga patakaran ang tumatakbo, laktawin ang mga indibiduwal na mga patakaran at tingnan ang tsart ng mga tugon.

Kanang pag-pindot sa menu

Ang kanang pagpindot ng node ay magpapalabas ng menu na magpapahintulot sa iyo na:

Ang pagbukod mula sa

Ang menu na ito ay may mga kasalukuyang mga submenu:

Ang Proxy

Binubukod nito ang mga napiling mga node na mula sa proxy. Sila ay nag proxy parin gamit ang ZAP pero hindi makikita sa kahit anong mga tab.
Maari din itong magamit upang hindi mapansin ang mga URL na alam mo nang walang kaugnayan sa sistema na kasalukuyan mong sinusuri.
Ang mga node na maaring kabilang ulit sa Mga Pagaari ng Sesyon ang dayalogo

Ang taga scan

Pinipigilan nito ang mga napiling mga node na hindi aktibong mag-scan.
Ang mga node ay maari nang muling ibilang sa pamamagiitan ng Ang mga Katangian sa Sesyon ang dayalogo

Ang ispayder

Papahintulutan nito ang mga napiling mga node na mag-spayder.
Ang mga node ay maari din na masama muli sa pamamagitan ng Ang Mga Katangian sa Sesyon ang dayalogo

New Alert...

Ito ay magpapalabas ng Add Alert dialog which allows you to manually record a new alert against this request.

Show in Sites tab

This will show the selected message in the Sites tab.

Open URL in Browser

This will open the URL of the selected node in your default browser.
