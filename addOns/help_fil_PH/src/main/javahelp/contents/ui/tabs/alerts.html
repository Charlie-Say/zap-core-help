Mga Pang-alerto na tab

The Alerts tab show the Alerts that have been raised in this session. Ang mga alerto na nakikita sa kahoy na may panganib na ayos sa kaliwang kamay na pane, at ang bawat noda ng kahoy ay nagpapakita ng kabuuang bilang ng mga alerto sa loob nito.

Ang pagpili ng alerto gamit ang isang pag-click ay magpapakita nito sa kanang kamay ng pane. Ang dobleng pag-click ng alerto ay magpapakita ng Magdagdag ng Alerto na dialog na magpapahintulot sa iyong baguhin ang mga detalye ng alerto.

Pagsala ng mga alerto

Ang mga buttons na pangsala, nakikita agad sa itaas ng kahoy, ay magpapahintulot sa iyo na higpitan ang mga alerto na makikita:

 /    Ipakita lamang ang mga URL sa Saklaw / Ipakita ang lahat ng mga URL

Nagpapahintulot na mapakita lamang ang mga alerto ng mga URL na nasa saklaw.

 /    I-ugnay / Putulin ang ugnay sa seleksyon ng Mga Site

Nagpapahintulot na mapakita lamang ang mga alerto ng Tab ng mga site napiling noda ng kahoy.

Tandaan:Ang pagpili ng isang pagsasala ay hindi magpapagana ng iba dahil sila ay ekslusibong sa isa't isa.

  I-edit ang Alerto

Magpapakita ng Magdagdag ng Alerto dialogo na magpapahintulot sa iyo na baguhin ang mga detalye ng alerto, para sa napiling alerto.

Tandaan: Kung maramihan ang mga alerto na napili sa kahoy, ang pinakabagong napiling aytem ang ma-edit.

  Delete All Alerts

Allows to delete all alerts.
