Pangbreak na tab

Ang Pangbreak na tab ay magpapahintulot sa iyo na baguhin ang hiling o tugon kapag ito ay mahuli ng ZAP gamit ang breakpoint.
Ito ay magpapahintulot sa iyo na baguhin ang mga elemento na hindi mo mababago sa normal na paraan gamit ang iyong browser, kasali ang:
     Ang header
     Natatagong mga field
     Hindi mapaganang mga field
     Ang mga field na gumagamit ng javascript upang masala palabas ang ilegal na mga karakter
Ang pamamaraang ito ay susi upang epektibong ma-pen test ang iyong aplikasyon.

Ang dalawang mga panel ay mananatili lamang na maglalaman ng kahit ano kapag ang ZAP ay makakuha ng hiling o tugon.
Maari mong baguhin ang kahit ano sa dalawang mga panel at i-forward ang hiling o tugon gamit ang mga button sa Pinakamataas na Lebel ng Toolbar.

Ang paghila pababa ay magpapahintulot sa iyo na pumili ng ibang Mga View para sa hiling o tugon na header at katawan.

       CSS and Fonts Files
       Multimedia Files

  Update Content Length

By default ZAP will automatically update (or add) the HTTP(S) Content-Length header to match the size of the data submitted. If this option is unselected then ZAP will not update the header allowing you to specify any value you like.
This button will only be shown for HTTP(S) messages.
