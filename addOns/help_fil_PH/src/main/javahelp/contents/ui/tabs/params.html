Tab ng Parameter

Nagpapakita ito ng buod ng mga parameter at mga patlang ng header ng tugon na ginagamit ng isang site.

Maaaring mapili ang mga site sa pamamagitan ng toolbar o sa Mga site tab.

Para sa bawat parameter na maaari mong makita:
     Ang uri - Cookie, FORM, URL, o Header
     Ang pangalan ng parameter (or respones header)
     Ang bilang ng beses kung ilang beses ginamit ito
     Ang bilang ng mga natatanging unique values
     Ang kabuuang pagbabago ng porsyento, kung saan ang 0 ay nangunguhulugan ng isang value lang ang nagamit at ang 100 naman ay nangangahulugang ang lahat ng value na nakalaan ay kakaiba
     Ang flags- kabilang ang mga cookie flags at anticsrf at sesyon
     Ilan sa mga values - ang buong hanay ng values ay hindi lahat nakikita

Kanang click na menu

Ang pagright-click sa isang node ay magdadala sayo sa isang menu na magbibigay-daan sayo sa:

Paghahanap

Ito ay magpapakita sa lahat ng mga halimbawa ng parameter na napili sa tab ng Paghahanap.

Flag bilang Anti CSRF token

Ito ay mag-flag ng parameter bilang isang Anti CSRF token.

Unflag bilang Anti CSRF token

Ito ay magtatanggal sa flag ng Anti CSRF token mula sa parameter.

Flag bilang token ng Sesyon

Ito ay magmamarka sa parameter bilang isang token ng Sesyon para sa kasalukuyang Site at mag-aabiso sa mga Sesyon ng HTTP na kasangkapan ayon dito.

Unflag bilang token ng Sesyon

Tatanggalin nito ang marka sa parameter bilang isang token ng Sesyon para sa kasalukuyang site at mag-aabiso sa mga Sesyon ng HTTP na kasangkapan ayon dito.
