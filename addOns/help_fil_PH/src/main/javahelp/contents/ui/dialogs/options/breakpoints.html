Mga opsyon ng mga Breakpoint screen

Ang screen na ito ay pinapayagan ka para i-configure ang breakpoint na mga opsyon:

Ikomperma ang mga drop trapped na mensahe

Magtanong sa kompermasyon kung ang hiling o pagtugon ay bumaba na.

Tanging ang pagsira sa mga mensahe sa scope

The 'global' breakpoint will only be triggered by requests that are in scope.
Custom breakpoints will not be affected.

Ang ZAP ay laging nasa taas sa tuwing ang breakpoint ay tumatama

Kung itong napili na ibig sabihin na ang ZAP ay mananatili sa taas ng lahat ng iba pang mga window sa tuwing ang isang breakpoint ay pumalo.

Show buttons to select the requests you don't want ZAP to break on

This controls whether the buttons to select the file types you don't want ZAP to break on and the scope filter appear in the break panel.

Javascript URL regex

This sets the regex used to predict if the content in the response will be a Javascript file.

CSS and Fonts URL regex

This sets the regex used to predict if the content in the response will be a CSS or a Font file.

Multimedia URL regex

This sets the regex used to predict if the content in the response will be a multimedia file (images, videos or sounds).

Break buttons mode

This controls whether there is a single combined break button on the Taas ng Level ng Toolbar or separate ones for Requests and Responses.
