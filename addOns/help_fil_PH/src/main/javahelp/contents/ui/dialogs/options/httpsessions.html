Opsyon ng Screen ng mga Sesyon ng HTTP

This screen allows you to configure the HTTP Sessions Extension's options:

Default Tokens

Ang mga ito ay mga pangalan ng Cookies na isinasaalang-alang nang default na maging token ng sesyon. Sa lalong madaling panahon sila ay nakita bilang mga parameter ng isang Site, sila ay awtomatikong naidagdag bilang mga token ng sesyon para sa Site na iyon, maliban kung manu-manong sinapawan sa pamamagitan ng command ng gumagamit.

Ang mga token ng sesyon ay case-insensitive na ginamit, samakatuwid ang default na mga token ng sesyon ay awtomatikong maging maliit na titik. Gayundin, anumang mga pagbabago sa default na mga token ng sesyon ay makikita lamang pagkatapos ng pagbisitang muli sa mga pahina ng Site.

Paganahin para lamang sa Proxy

Kung ang opsyon ay pinagana, ang Extension ay magproseso lamang ng mga mensaheng naipada at natanggap sa pamamagitan ng Proxy (halimbawa mula sa/sa browser). Ang mga mensahe mula sa ibang mga Extension ay pinoproseso lamang kung ang opsyon ay hindi pinagana.
