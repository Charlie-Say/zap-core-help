Screen na Options Active Scan

This screen allows you to configure the passive scan rules.

Limitasyon sa

Ito mga kontrol kung paano malamang ZAP ay ang report potensyal kahinaan.
  • If you select Off then the scan rule won't run.
  • Kung pipiliin ninyo ang mababang pagkatapos pang potensyal na isyu ay itinaas na kung saan maaaring tumaas ang bilang sa maling positibo.
  • Kung piliin ninyo mataas pagkatapos ay mas kaunting mga potensyal na isyu ay itinaas na maaaring nangangahulugan na ang ilang mga tunay na isyu ay hindi nakuha (huwad na mga negatibo).
