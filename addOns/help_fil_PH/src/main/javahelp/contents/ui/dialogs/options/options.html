Reset sa Factory default

Ang mga pindutan ng 'Reset sa Factory default' ay resets sa lahat ng mga pagpipilian sa mga pinahahalagahan na itinakda nang una mong tumakbo ZAP. Kabilang dito ang reeseting ang host at port na ZAP ay pakikinig sa at ang mga susi ng API.
Ang kumpigurasyon ng isang bagay na hindi magbabago ang ZAP ugat CA Certificate. Kung gusto mong baguhin ang mga ito at pagkatapos ay magagawa ninyo iyan sa pamamagitan ng Dynamic SSL sertipiko screen.
