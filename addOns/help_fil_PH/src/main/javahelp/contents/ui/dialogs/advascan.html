Ang aktibong pag-iskan ng diyalogo

Ang diyalogong ito ay inilunsad ang aktibong iskaner.

Ang saklaw

Ang unang tab ay pinahihintulutan ka na piliin o baguhin ang mga panimulang punto.
Kung mayroon ka ng higit sa isa na mga patakaran ng pag-iskan kapag ikaw ay mayroong kakayahan na pumili ng isa na gagamitin.
Kung ang panimulang punto ay nasa isa o higit pa Na mga konteksto pagkatapos ay magagawa mo na piliin ang is sa kanila.
Kung ang kontekstong yun ay mayroong anuman Na mga gumagamit na tinukoy pagkatapos ay magagawa mo na piliin ang isa sa kanila.
Kung iyong napili ang isa sa mga gumagamit pagkatapos ang aktibong pag-iskan ay gaganap bilang gumagamit na iyon, kasama ang ZAP ng (re)pagpapataunay bilang iyon na gumagamit kapag kinakailangan.

If you select 'Recurse' then all of the nodes underneath the one selected will also be scanned.
Ang kostumbre ng input ng mga vector ay suportado lamang kung itong opsyon ay hindi napili.

If you select 'Show Advanced Options' then the following tabs will be shown which provide fine grain control over the active scanning process.

Ang pagpindot sa mga 'reset' na buton ay ang pag-reset sa lahat ng mga opsyon sa kanilang default na mga halaga.

Ang mga Input Vector

Ang input ng mga vector ng tab ay nagpapahintulot sayo na i-overrride ang ingput ng mga vector kung saan ay natukoy sa mga Aktibong pag-iskan ng mga opsyon sa input ng mga vector ng iskrin.
Ang pagpindot sa mga 'reset'na buton ay ang ang mag rereset sa input ng mga vector sa default ng mga opsyon.

Ang kustom na mga vector

Ang gawi ng mga vector ng tab na napagpapahintulot sa iyo na matukoy ang tiyak na mga lokasyon sa mga kahilingan upang umatake.
Custom Vectors are only available if the 'Recurse' option on the first tab is not selected.
Upang madagdagan ang gawi ng input ng mga vector i-highlight ang mga katangian na nais mong salakayin sa mga kahilingan at pagpindot ng 'karagdagang'buton.
Maaari mong dagdagan hanggang ilan ang gawi ng input ng mga vector hanggat gusto mo.
Para tanggalin ang gawi ng input ng mga vector i-highlight ang ilan sa mga napiling mga katangian at pindutin ang 'natanggal' na buton.
Ang pagsisiyasat sa 'hindi pinagananag di-gawi ng input ng mga vector' ng kahon na mga hindi pinagana ng lahat ng mga input ng vector maliban sa mga iyon na manu-mano mong natukoy sa tab na ito.

Ang teknolohiya

Ang teknolohiya ng tab na nagpapahintulot sa iyo na tukuyin kung aling mga klase ng mga teknolohiya na mai-iskan.
Ang mga di-napiling mga teknolohiya na alam mo na wala sa mga pinupuntirayang aplikasyon na maaaring pabilisin ang pag-iskan bilang mga patakaran ang alinman na pinupuntirya ng teknolohiya na maaaring laktawan ang mga pagsubok na iyon.

Ang patakaran

Ang patakaran ng tab na nagpapahintulot sa iyo na i-override ang alinman sa mga setting na natukoy sa mga napili ng patakaran ng pag-iskan.

Filter

The Filter tab allows you to specify filter criteria and active scan will filter requests based on these criteria's.
There are four filter criteria's available in the filter tab
  • HTTP method
  • Status code
  • Tags
  • URL pattern
HTTP method and Status code filter criteria are only applied to check if a request can be included in active scan or not,
where as Tags and URL pattern filter criteria can be applied to include or exclude a request.
