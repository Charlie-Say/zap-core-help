Mga Opsyon ng Screen ng mga Extension

This screen allows you to enable/disable extensions that are available to ZAP, either core or added by add-ons.

Paganahin/Hindi Paganahin ang mga Extension

Ang mga gumaganang extension ay nai-load ng ZAP, kaya naman nagdaragdag ng mga andar na ibinigay sa pamamagitan ng mga extension na iyon. Ang mga hindi pinagana na extension ay hindi mai-load.

Mga Core Extension

Ang mga core extension ay mga extension na kapag hindi pinagana ay makapaghina sa napakaliit na pag-andar ng ZAP. Ang mga extension na minarkahan bilang core ay hindi maaaring hindi paganahin.

Paalala: Ang mga core extension ay maaaring hindi paganahin sa pamamagitan ng file ng kumpigurasyon, kahit na ito ay hindi inirerekomenda.

Extensions' Dependencies

Ang mga extension ay maaaring nakasalalay sa iba pang mga extension para gumana. Kapag ang isa sa mga extension na iyon (dependencies) is hindi pinagana, gagawin nito awtomatikong hindi paganahin ng ZAP ang lahat ng mga extension na nakasalalay sa mga ito. Kaaya maaaring mangyari na kapag hindi pinapagana ang isang extension, ang ibang extension (na nakasalalay nito) ay hindi din paganahin. Awtomatikong hindi pinagana na mga extension ay hindi papahintulutan na paganahin hangga't ang lahat nga dependencies ay pinagana na.

Paalala: Kailangan mong i-restart ang ZAP para magkabisa ang mga pagbabago.
