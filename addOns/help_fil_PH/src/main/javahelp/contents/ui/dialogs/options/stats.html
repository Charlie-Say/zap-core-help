This screen allows you to configure the statistics that ZAP maintains.

Pinagana ang In Memory na Istatistika

Bilang default, pinapanatili ng ZAP ang istastika sa memory. Upang hindi paganahin ang istatistika ng memory, alisin ang tsek sa kahong ito.

Pinagana ang Statsd Server

I-tsek ang kahon na ito upang masaayos ang address ng Statsd server na papadalhin ng istatistika ng ZAP.

Hostname ng Statsd

Ang host kung saan tumatakbo ang Statsd server.

Port ng Statsd

Ang port na pinapakinggan ng Statsd server, ang default ay 8125.

Prefix ng Statsd

Ang prefix na dadagdagan ng ZAP sa una para sa istatistika kung saan ito ay nagpapadala sa Statsd server.
