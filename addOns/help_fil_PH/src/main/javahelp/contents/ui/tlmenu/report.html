Ang Menu ng Ulat

Ang menu na ito ay nagdadala ng mga ulat.

Lumikha ng Ulat ng HTML...

Ito ay lumilikha ng bagong ulat ng HTML na naglalaman ng mga iniaakyat na mga alerto.

Lumikha ng Ulat ng XML...

Ito ay lumilikha ng bagong ulat ng XML na naglalaman ng mga iniaakyat na mga alerto.

Lumikha ng Ulat ng Markdown...

Ito ay lumilikha ng bagong ulat ng Markdown na naglalaman ng mga iniaakyat na mga alerto.

Mag-export ng mga Mensahe sa File...

Ito ang nagpapahintulot sa iyo upang i-save ang mga kahilingan at kasagutan sa isang tekstong file.
Piliin ang mga mensahe upang i-save sa History tab - gamitin ang Shift key upang pumili ng higit sa isang mensahe.

Mag-export ng Kasagutan sa File...

Ito ang nagpapahintulot sa iyo upang i-save ang espesipikong kasagutan sa isang file.
Piliin ang kaugnay na mensahe sa History tab - tandaan na ang mga kasagutan ng binary (tulad ng litrato) ay maaring i-save kasabay ng pagsuri sa mga kasagutan.

I-export ang lahat ng mga URL sa File...

Ito ang nagpapahintulot sa iyo upang i-save ang lahat ng mga pwedeng URL to isang teksto o HTML na file.
Ito ay maaaring gamitin, sa kahit anumang bagay, upang ikumpara ang mga URL na meron para sa mga taga-gamit magkaibang alituntunin at permisyon sa parehong sistema.

I-export ang mga Napiling URL sa File...

Ayon sa pagpipili (kasali na rin ang maramihang-pagpili) sa mga puno ng Sites lahat ng URL at mga pambatang URL ng napiling nodes are na-export na.

I-export ang mga URL sa Kontesksto

Lahat ng mga URL sa puno ng Sites na bumagsak papunta sa napiling konteksto ay na-export na. Ang functionality na ito ay meron rin mula sa Kanang-click na talaan kapag ginamit sa isang konteksto node sa puno nga mga Sites na panel.

Ikumpara sa iba pang Sesyon...

Ito ang nag-uudyok sa iyo papunta sa isang ZAP sesyon na minsan mo nang nai-save.
Ito rin ay nag-uudyok sa iyo papunta sa isang kinalabasang file kung saan ay nakasulat lahat ng mga URL na meron sa kasalukuyang sesyon at ang sesyon na iyong napili ay ikukumpara dito.
Ang file na ito ay maglalaman ng isang talaan ng listahan ng mga URL at ang HTTP na mga kasagutan para sa mga URL na nasa 2 sesyons.
Ang JavaScript buttons ay nagpapahintulot sa iyo na ipakita ang lahat ng mga URL, yun lamang mga pwede sa unang sesyon, sa ikalawang sesyon at mga sesyon na pwede sa parehong mga sesyon.
Ito ay partikular na kapaki-pakinabang sa pagkukumpara ng 2 sesyon kung saan may akses sa parehong aplikasyon gamit ang magkaibang taga-gamit. Maaari mong makita kung ano ang mga URL na makikita ng mga taga-gamit at maaari ring masubukan na ma-access lahat ng mga URL kung nakalog-in bilang alinman sa mga taga-gamit.

Note that add-ons can add additional menu items.
