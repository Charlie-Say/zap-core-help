Ang tanaw na menu

Itong menu ay nagdadala ng display na mga opsyon.

Paganahin ang Imahe sa kasaysayan

Kung nakapili ng mga imahe ay ma-display sa Ang kasaysayan na tab.

Ipakita ang tab

Ito ay nagpapahintulot sa iyo na mgapakita ng anumang mga tab na magagamit, kasali na yung hindi makikita.

Ipakita ang lahat na mga tab

Ito ay nagpapahintulot sa iyo upang ipakita ang lahat ng mga tab.

Itago ang naka-unpinned na mga tab

Ito ay nagpapahintulot sa iyo na itago ang lahat na mga tab na hindi naka-pinned.

I-pin ang lahat ng nakikitang mga tab

Ito ay nagpapahintulot sa iyo upang i-pin ang lahat ng nakikitang mga tab.

I-unpin ang lahat ng mga tab

Ito ay nagpapahintuloy sa iyo na i-unpin ang lahat ng mga tab.

Note that add-ons can add additional menu items.
