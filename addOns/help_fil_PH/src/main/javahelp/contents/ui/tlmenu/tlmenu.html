Ang ibabaw na antas na menu

Sa pamamagitan ng default sa sumusunod sa itaas na antas na menu na mga item ay nakaharap:
     Ang File menu Ito ay nagdadala sa kasalukuyang sesyon
     Ang Edit menu This handles finding strings in specific tabs, searching for strings across all requests and responses, changing ZAP's mode, and managing forced user mode
     Ang Tanaw na menu Ito ay nagdadala sa display na mga opsyon
     Ang Analyse na menu Ito ay nagdadala ng Scan na mga policy
     Ang ulat na menu Ito ay nagdadala ng mga ulat
     Ang mga kagamitan na menu Ito ay nagdadala ng mga kagamitan at general na mga opsyon
     Import menu This gives access to menus that allow to import data into ZAP.
     Online menu This gives access to a set of online resources
     Help menu This gives access to the 'about' dialog and this help file

Tandaan na ang add-ons maari magdagdag ng karagdagang mga menu item.
